Wijmo Glyphs

wijmo.css 文件包含了若干以纯 CSS 方式定义的 glyph。 这些 glyph 在 Wijmo 控件以及其扩展中使用，您的应用程序也可以使用它们。

使用 CSS 定义 glyph 免去了部署额外字体或者图片文件的困扰，同时可以确保各种图形按照 当前主题指定的前景色和字体呈现。

为了在您的应用程序中使用 glyph，需要向您的标记语言中添加一个 span 元素，同时设置其 class 为 glyph 的名称。例如： 

<span class="wj-glyph-diamond"></span>

下表展示了在 wijmo.css 文件中定义的 glyph。

名称 Glyph 标记语法
{{g}} <span class="wj-glyph-{{g}}"></span>
