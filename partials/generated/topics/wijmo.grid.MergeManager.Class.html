定义 FlexGrid 的单元格合并行为。

该类的一个实例将自动地被创建并添加到表单的 mergeManager 属性，以实现表单的默认合并行为。

如果您希望自定义默认的合并行为，可以创建一个类，继承自 MergeManager 类，并覆盖 getMergedRange 方法。

方法

名称 参数 描述
getMergedRange​(p, r, c, clip?): CellRange

p (GridPanel): 包含该范围的 GridPanel 对象。
r (number): 包含该单元格的行的索引。
c (number): 包含该单元格的列的索引。
clip? (boolean): 是否将合并的范围和表单的当前可视范围做 clip。
(默认值: true)

 获取一个 CellRange，指定在 GridPanel 中的单元格合并扩展。

返回值 指定合并范围的一个 CellRange 对象，或者如果单元格没有合并，则返回 null。
