PdfGradientStop 类

模块
wijmo.pdf
文件
wijmo.pdf.js

表示一个对象，该对象决定一个渐变的过渡点。

构造函数

名称 参数 描述
constructor​(offset?, color?, opacity?): PdfGradientStop

offset? (number): 渐变轴上渐变停止的位置。
color? (any): 渐变停止的颜色。一个 Color 对象或者可被 fromString 方法接受的任何字符串。
opacity? (number): 渐变停止的透明度。

 初始化 PdfGradientStop类的一个新实例。

属性

名称 类型 描述
color Color 获取或设置渐变色标的颜色。 默认的颜色为黑色。
offset number 获取或设置画刷渐变轴上渐变色标的位置。 值必须位于范围 [0, 1] 之间，这里 0 表示渐变色标位于渐变轴的起始位置，而 1 表示渐变色标位于渐变轴的结束为止。 默认值为 0。
opacity number 获取或设置渐变色标的不透明度。 值必须位于范围 [0, 1] 之间，这里 0 表示渐变色标完全透明，而 1 表示渐变色标完全不透明。默认值为 1。

方法

名称 参数 描述
clone​(): PdfGradientStop 创建该 PdfGradientStop 的一份拷贝。

返回值 渐变停止的一个拷贝。
equals​(value): boolean

value (PdfGradientStop): 待比较的 PdfGradientStop 对象。

 确定是否指定的 PdfGradientStop 实例和当前的实例相等。

返回值 如果指定的对象与当前相等，则返回true，否则为false。
