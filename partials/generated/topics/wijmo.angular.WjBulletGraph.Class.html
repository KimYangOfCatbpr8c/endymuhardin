AngularJS 指令应用于 BulletGraph 控件。

使用 wj-bullet-graph 指令添加 bullet graphs 到 AngularJS 应用中。 注意指令和参数名称必须使用小写字母+短线语法替代驼峰命名。 示例：

<wj-bullet-graph 
  value="ctx.gauge.value" 
  min="0" max="10"
  target="{​{item.target}}"
  bad="{​{item.target * .75}}"
  good="{​{item.target * 1.25}}">
</wj-bullet-graph>

wj-bullet-graph 指令支持以下属性:

control
= 该指令创建的指向 BulletGraph control created by this 指令。
direction
@ GaugeDirection 值，测量值增长的方向。
initialized
& 该事件在绑定完成控件特性值的初始化后被触发。
is-initialized
= 绑定是否已经完成控件特性值的初始化。
target
@ 测量的目标值。
good
@ 测量时判断为正确的值。
bad
@ 测量时判断为错误的值。
value
= 测量的实际值。

wj-bullet-graph 指令可以包含一个或多个 WjRange 指令。
