PdfFont 类

模块
wijmo.pdf
文件
wijmo.pdf.js

表示一个字体。

构造函数

名称 参数 描述
constructor​(family?, size?, style?, weight?): PdfFont

family? (string): 字体名称。
(默认值: 'times')
size? (number): 字体大小。
(默认值: 10)
style? (string): 字体样式。
(默认值: 'normal')
weight? (string): 字体粗细
(默认值: 'normal')

 初始化 PdfFont 类的一个新实例。

属性

名称 类型 描述
family string

获取或设置字体的字体集名称。

字体集名称的列表按照优先级顺序， 由逗号分隔。每一个字体集名称可以为之前由 registerFont 方法注册的或者 PDF 标准字体集：'courier'， 'helvetica'，'symbol'，'times'，'zapfdingbats' 或者一些超级字体集名称：'cursive'，'fantasy'，'monospace'，'serif'，'sans-serif'。
size number 获取或设置字体的尺寸。
style string

获取或设置字体的样式。

支持以下值：'normal'，'italic'，'oblique'。
weight string

获取或设置字体的粗细。

支持以下值：'normal'，'bold'，'100'，'200'，'300'，'400'，'500'，'600'，'700'，'800'，'900'。

方法

名称 参数 描述
clone​(): PdfFont 创建该 PdfFont 的一份拷贝。

返回值 这个字体的一个拷贝 。
equals​(value): boolean

value (PdfFont): 待比较的 PdfFont 对象。

 确定是否指定的 PdfFont 实例和当前的实例相等。

返回值 如果指定对象与当前对象相等，则返回真，否则返回假。
