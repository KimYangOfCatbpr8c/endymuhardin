FlexGridXlsxConverter 类

模块
wijmo.grid.xlsx
文件
wijmo.grid.xlsx.js

该类型提供了静态的 import 以及 export 方法，以便从 Excel xlsx 格式的文件导入和导出 FlexGrid 控件。

方法

名称 参数 描述
S load​(grid, workbook, options?)

grid (FlexGrid): 加载该 workBook 对象的 FlexGrid。
workbook (any): 一个 WorkBook 实例或者一个 Blob 实例，再或者一个 base 64 编码字符串或一个 ArrayBuffer，包含 xlsx 文件内容。
options? (IFlexGridXlsxOptions): 加载该 workBook 对象至 FlexGrid 实例时所使用的选项（包括是否保存行头和列头，表单名称，等等）。

加载一个包含 xlsx 文件内容的 Workbook 实例或者一个 Blob 对象至 FlexGrid 实例。

例如： 

// 该示例通过打开文件对话框打开一个 xlsx 文件
// 并使用第一个表单的数据填充
// FlexGrid。
 
// HTML
<input type="file" 
    id="importFile" 
    accept="application/vnd.openxmlformats-officedocument.spreadsheetml.sheet" 
/>
<div id="flexHost"></>
 
// JavaScript
var flexGrid = new wijmo.grid.FlexGrid("#flexHost"),
    importFile = document.getElementById('importFile');
 
importFile.addEventListener('change', function () {
    loadWorkbook();
});
 
function loadWorkbook() {
    var reader,
        file = importFile.files[0];
    if (file) {
        reader = new FileReader();
        reader.onload = function (e) {
            wijmo.grid.xlsx.FlexGridXlsxConverter.load(flexGrid, reader.result,
                { includeColumnHeaders: true });
        };
        reader.readAsArrayBuffer(file);
    }
}
S save​(grid, options?, fileName?): Workbook

grid (FlexGrid): 待保存的 FlexGrid。
options? (IFlexGridXlsxOptions): 当保存表单时，使用的选项（包括是否保存行头和列头，表单名称，等等）。
fileName? (): 待生成的文件的文件名。

保存 FlexGrid 实例至 Workbook 实例。

例如： 

// 该示例导出 FlexGrid 的内容至一个 xlsx 文件 
// 单击。
 
// HTML
<button 
    onclick="saveXlsx('FlexGrid.xlsx')">
    Save
</button>
 
// JavaScript
function saveXlsx(fileName) {
    // 保存 flexGrid 至 xlsx 文件。
    wijmo.grid.xlsx.FlexGridXlsxConverter.save(flexGrid,
            { includeColumnHeaders: true }, fileName);
}



返回值 一个 Workbook 对象，可以在保存之前对其进行自定义（通过 Workbook.save 方法进行保存）。
