IPdfTextSettings 类

模块
wijmo.pdf
文件
wijmo.pdf.js

表示 drawText 以及 measureText 方法使用的文本设置。

属性

名称 类型 描述
F align? PdfTextHorizontalAlign 决定文本如何在绘制区域内对齐显示。 默认值为 Left。
F characterSpacing? number 决定文本中两个字符之间的间隔。 默认值为 0。
F columnGap? number 决定各个列之间的间隔，单位是点。 默认值为 18。
F columns? number 决定文本浮动绘制的总列数。 默认值为 1。
F continued? boolean

表示是否随后的文本应当紧随其后还是另起一个新的段落。如果设置为 true，则文本设置在不同的 drawText 方法调用之间将被保持。 这意味着选项参数会和之前一次的 drawText 调用合并。

默认值为 false。
F ellipsis? any 决定当文本超出指定区域时，显示在文本末尾的字符。默认值为 undefined，表示不显示省略号。 设置为 true 表示使用默认字符。
F fill? boolean 表示是否文本是否应当被填充。 默认值为 true。
F height? number 决定绘制区域的高度，单位是点，文本到达此位置需要被截断。 默认值为 undefined，表示文本区域受到到底部 body 区域边距的限制。 使用 Infinity 表示该文本区域具有无限制的高度。
F indent? number 决定在每一个段落起始位置文本的缩进，单位是点。 默认值为 0。
F lineBreak? boolean 表示是否考虑文本折行。 如果定义了 width，则该属性被忽略。 默认值为 true。
F lineGap? number 决定两行文本之间的间隔。 默认值为 0。
F link? string 决定一个用作创建一个链接标注的 URL（URI 动作）。
F paragraphGap? number 决定在两个段落的文本之间的间隔，单位是点。 默认值为 0。
F strike? boolean 表示是否文本应当具有删除线。 默认值为 false。
F stroke? boolean 表示是否文本是否应当描边。 默认值为 false。
F underline? boolean 表示是否文本是否应当具有下划线。 默认值为 false。
F width? number 决定文本区域的宽度，单位是点，文本到达此位置需要折行。 默认值为 undefined，表示文本区域受到右侧页面边距的限制。 使用 Infinity 表示该文本区域具有无限制的宽度。 如果定义了值，则强制地使得 lineBreak 属性启用。
F wordSpacing? number 决定文本中两个单词之间的间隔。 默认值为 0。
