GroupDescription 类

派生类
PropertyGroupDescription
模块
wijmo.collections
文件
wijmo.js

表示类型定义分组条件的基类。

普遍用作这个PropertyGroupDescription目的的具体类。

方法

名称 参数 描述
groupNameFromItem​(item, level): any

item (any): 为该项目获取组名。
level (number): 零基组水平索引。

 返回给定项目的组名。

返回值 该项目属于该组的名称。
namesMatch​(groupName, itemName): boolean

groupName (any): 组名。
itemName (any): 项目名称。

 返回一个值，该值指示组名称和项目名称匹配 （意味着该项目属于该组 ）。

返回值 如果名字匹配为真，否则为假。
