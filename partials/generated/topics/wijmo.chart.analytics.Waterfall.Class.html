Waterfall 类

基类
SeriesBase
派生类
WjFlexChartWaterfall
模块
wijmo.chart.analytics
文件
wijmo.chart.analytics.js
表示

代表FlexChart 的一个瀑布系列。 Waterfall 系列主要用于展示开始的位置的增长或下降情况。

构造函数

名称 参数 描述
constructor​(options?): Waterfall

options? (): 包含初始化数据的Javascript对象。

 初始化类 Waterfall 的一个新实例。

属性

名称 类型 描述
altStyle any

获取或设置系列的交替样式。来自于该属性的值将用作 Bar，Column，以及 Scatter 图表上的负值；以及金融图表， 比如说 Candlestick，LineBreak，EquiVolume 等等，的上升值。

如果没有提供该值，则使用默认的样式。


(继承自 SeriesBase).
axisX Axis 获取或设置系列的 X 坐标轴。
(继承自 SeriesBase).
axisY Axis 获取或设置系列的 Y 坐标轴。
(继承自 SeriesBase).
binding string 获取或设置包含系列 Y 值的属性名称。
(继承自 SeriesBase).
bindingX string 获取或设置包含系列 X 值的属性名称。
(继承自 SeriesBase).
chart FlexChartCore 获取拥有该系列的 FlexChart 对象。
(继承自 SeriesBase).
collectionView ICollectionView 获取包含系列数据的 ICollectionView的对象。
(继承自 SeriesBase).
connectorLines boolean 获取或设置一个值决定是否显示连接线。
cssClass string 获取或设置系列的 CSS 类。
(继承自 SeriesBase).
hostElement SVGGElement 获取对象容器元素。
(继承自 SeriesBase).
intermediateTotalLabels any 获取或设置一个属性值，该属性包含中间总计栏的标签。它应该是一个数组或者字符串。 这个属性应该和showIntermediateTotal 和 intermediateTotalPositions 工作。
intermediateTotalPositions number[] 获取或设置一个属性的值，该属性包含中间总计栏的位置的索引。 这个属性应该和showIntermediateTotal 和 intermediateTotalLabels 一起工作。
itemsSource any 获取或设置包含系列数据的数组或者 ICollectionView 对象。
(继承自 SeriesBase).
legendElement SVGGElement 获取图例区对应的该系列的元素。
(继承自 SeriesBase).
name string

获取或设置系列名称。

系列名称显示在图表图例区。任何不具有名称的系列将不会显示在图例区域。


(继承自 SeriesBase).
relativeData boolean 获取或设置一个值决定给定的数据是相对的
showIntermediateTotal boolean 获取或设置一个值，决定是否展示中间总计栏。 这个属性应该和 intermediateTotalPositions and intermediateTotalLabels 一起工作。
showTotal boolean 获取或设置一个值决定是否显示总计栏
start number 获取或决定开始栏的值，如果开始为空，那么开始栏就不会显示，
startLabel string 获取或设置开始栏的标签
style any 获取或设置系列样式。
(继承自 SeriesBase).
styles any

获取或设置瀑布图的样式，支持下面的样式

start: 制定开始栏的样式

total: 制定总计栏的样式

intermediateTotal: 制定中间总计列的样式。

falling: 制定下落列的样式。

rising: 制定上升列的样式。

connectorLines: 制定连接线的样式。

waterfall.styles = {
  start: {
     fill: 'blue',
     stroke: 'blue'
  },
  total: {
     fill: 'yellow',
     stroke: 'yellow'
  },
  falling: {
     fill: 'red',
     stroke: 'red'
  },
  rising: {
     fill: 'green',
     stroke: 'green'
  },
  connectorLines: {
     stroke: 'blue',
     'stroke-dasharray': '10, 10'
  }
}
symbolMarker Marker 获取或设置系列中每一个数据点的标记形状。 应用到 Scatter，LineSymbols，以及 SplineSymbols 图表类型。
(继承自 SeriesBase).
symbolSize number 获取或设置用来呈现该 Series 的符号的尺寸（单位是像素）。 应用到 Scatter，LineSymbols，以及 SplineSymbols 图表类型。
(继承自 SeriesBase).
symbolStyle any 获取或设置系列符号样式。 应用到 Scatter，LineSymbols，以及 SplineSymbols 图表类型。
(继承自 SeriesBase).
totalLabel string 获取或设置总计栏标签。
visibility SeriesVisibility 获取或设置一个枚举值，表示是否显示该系列以及在何处显示该系列。
(继承自 SeriesBase).
