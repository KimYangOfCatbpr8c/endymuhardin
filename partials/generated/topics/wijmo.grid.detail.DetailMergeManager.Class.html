该 FlexGridDetailProvider 类使用的合并管理器类。

DetailMergeManager 将明细单元格（DetailRow 中的单元格）合并为单一的 明细单元格，该单元格横跨全部的表单列。

构造函数

名称 参数 描述
constructor​(grid): DetailMergeManager

grid (FlexGrid): 拥有该 DetailMergeManager 的 FlexGrid 对象。

 初始化类 DetailMergeManager 的一个新实例

方法

名称 参数 描述
getMergedRange​(p, r, c, clip?): CellRange

p (GridPanel): 包含该范围的 GridPanel 对象。
r (number): 包含该单元格的行的索引。
c (number): 包含该单元格的列的索引。
clip? (boolean): 是否将合并的范围和表单的当前可视范围做 clip。
(默认值: true)

 获取一个 CellRange，指定在 GridPanel 中合并为一个单元格的范围。

返回值 表示合并范围的一个 CellRange 对象，或者当单元格没有发生合并时返回 null。
