CellRange 类

模块
wijmo.grid
文件
wijmo.grid.js

表示由两个行索引以及两个列索引定义的一个方形的单元格分组。

构造函数

名称 参数 描述
constructor​(r?, c?, r2?, c2?): CellRange

r? (number): 范围中第一行的索引（默认值为 -1）。
(默认值: -1)
c? (number): 范围中第一列的索引（默认值为 -1）。
(默认值: -1)
r2? (number): 范围中最后一行的索引（默认值为 r）。
(默认值: r)
c2? (number): 范围中最后一列的索引（默认值为 r）。
(默认值: c)

 初始化类 CellRange 的一个新实例

属性

名称 类型 描述
bottomRow number 获取范围内最后一行的索引。
col number 获取或设置范围中的第一列的索引。
col2 number 获取或设置范围中的第二列的索引。
columnSpan number 获取该范围内列的个数。
isSingleCell boolean 检查是否该范围关联到一个单个的单元格（起始行和结束行具有相同的索引，且起始列和结束列也具有相同的索引）。
isValid boolean 检查是否该范围包含合法的行和列索引 （行和列的索引值大于等于零）。
leftCol number 获取范围内最左侧一列的索引。
rightCol number 获取范围内最右侧一列的索引。
row number 获取或设置范围中的第一行的索引。
row2 number 获取或设置范围中的第二行的索引。
rowSpan number 获取该范围内行的个数。
topRow number 获取范围内第一行的索引。

clone​(): CellRange 创建一个该范围的拷贝。
contains​(r, c?): boolean

r (any): 待查找的 CellRange 对象或者行索引。
c? (number): 列索引（如果 r 参数不是一个 CellRange 对象，则此参数为必须）。

 检查是否该范围包含另一个范围或者一个特定的单元格。
containsColumn​(c): boolean

c (number): 待查找的列索引。

 检查是否该范围包含一个指定的列。
containsRow​(r): boolean

r (number): 待查找的行的索引。

 检查是否该范围包含一个指定的行。
equals​(rng): boolean

rng (CellRange): 和该范围进行比较的 CellRange 对象。

 检查是否该范围和另一个范围相等。
getRenderSize​(p): Size

p (GridPanel): 包含该范围的 GridPanel 对象。

 获取该范围呈现的尺寸。

返回值 一个表示该范围内所有行高的总和以及所有列宽的综合的 Size 对象。
intersects​(rng): boolean

rng (CellRange): 待检查的 CellRange 对象。

 检查是否该范围和另一个范围有交集。
intersectsColumn​(rng): boolean

rng (CellRange): 待检查的 CellRange 对象。

 检查是否该范围和另一个范围中的列有交集。
intersectsRow​(rng): boolean

rng (CellRange): 待检查的 CellRange 对象。

 检查是否该范围和另一个范围的行有交集。
setRange​(r?, c?, r2?, c2?)

r? (number): 范围中第一行的索引（默认值为 -1）。
(默认值: -1)
c? (number): 范围中第一列的索引（默认值为 -1）。
(默认值: -1)
r2? (number): 范围中最后一行的索引（默认值为 r）。
(默认值: r)
c2? (number): 范围中最后一列的索引（默认值为 r）。
(默认值: c)
