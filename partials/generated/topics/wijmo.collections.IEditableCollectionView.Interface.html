F canCancelEdit boolean 获取一个值，该值指示集合视图是否可以放弃更改和恢复编辑对象的原始值。
F canRemove boolean 获取一个值，该值指示是否可从集合中删除项目。
F currentAddItem any 获取在当前添加事务处理期间添加的项目。
F currentEditItem any 获取在当前编辑事务中编辑的项目。
F isAddingNew boolean 获取一个值，指示是否添加事务正在进行中。
F isEditingItem boolean 获取一个值，这个值指示编辑事务是否正在进行中。
