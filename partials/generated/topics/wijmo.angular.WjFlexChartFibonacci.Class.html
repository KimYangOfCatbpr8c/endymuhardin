AngularJS 指令应用于 FinancialChart Fibonacci 对象。

wj-flex-chart-fibonacci 指令必须包含于 WjFinancialChart 指令。 支持以下属性:

binding
@ 包含Y轴数据的属性名。该值可被任意重写以绑定图表数据。
binding-x
@ 包含X轴数据的属性名。该值可被任意重写以绑定图表数据。
css-class
@ 渲染序列的 CSS 样式类。
items-source
= 一个数组或者 ICollectionView 对象，其中包含序列数据。
high
@ Fibonacci 的高度。
labelPosition
@ Fibonacci 的标签级别位置。
levels
@ Fibonacci 的 Level 值。
low
@ Fibonacci 的低值。
minX
@ Fibonacci 的最小值。
maxX
@ Fibonacci 的最大值。
name
@ 序列显示的图例名称。
style
= 渲染序列的样式，使用 ng-attr-style 指定序列对象。参见 ngAttr AngularJS Creating Custom Directives 以及 FlexChart 101 Styling Series 上的实例以获取更多信息。
altStyle
= 序列交替样式。
visibility
= SeriesVisibility 值，用于指定是否显示序列。
uptrend
@ 是否创建增加趋势的 Fibonacci 。
