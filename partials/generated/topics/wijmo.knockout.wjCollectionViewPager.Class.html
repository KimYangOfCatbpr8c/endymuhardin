使用 KnockoutJS 绑定 ICollectionView 分页元素.

使用 wjCollectionViewPager 指令，可以实现 ICollectionView 的分页导航，示例如下：

Here is a CollectionViewPager:</p>
<div 
  data-bind="wjCollectionViewPager: { cv: myCollectionView }">
</div>

wjCollectionViewPager 指令只有一个属性：

cv
引用分页 ICollectionView 对象用于导航。
