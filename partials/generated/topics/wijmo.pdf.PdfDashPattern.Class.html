PdfDashPattern 类

模块
wijmo.pdf
文件
wijmo.pdf.js

表示用做绘制路径的虚线样式。

构造函数

名称 参数 描述
constructor​(dash?, gap?, phase?): PdfDashPattern

dash? (number): 交替短线的长度，以像素为单位。
(默认值: null)
gap? (number): 交替间隔的长度，以像素为单位。
(默认值: dash)
phase? (number): 短线模式下短线开始的地方，单位是像素。
(默认值: 0)

 初始化 PdfDashPattern 类的一个新实例。

属性

名称 类型 描述
dash number 获取或设置相间的短线的长度，单位是点。 默认值为 null，表示没有虚线样式，显示为一条实线的直线。
gap number 获取或设置相间的空隙的长度，单位是点。 默认值和 dash 相同，表示短线和间隔具有相同的长度。
phase number 获取或设置开始虚线的起始位置的距离，单位是点。 默认值为 0。
