模块 wijmo/wijmo.angular2.chart.hierarchical

文件
wijmo.angular2.js

包含用于 wijmo.chart.hierarchical 模块的 Angular 2 组件。

wijmo.angular2.chart.hierarchical 是一个外部的Typescript模块，可以使用它的模块名导入到您的代码中：

import * as wjHierarchical from 'wijmo/wijmo.angular2.chart.hierarchical';
 
@Component({
    directives: [wjHierarchical.WjSunburst],
    template: `
      <wj-sunburst [itemsSource]="data" [binding]="'y'" [bindingX]="'x'">
      </wj-sunburst>`,
    selector: 'my-cmp',
})
export class MyCmp {
    data: any[];
}

类

名称 描述
WjSunburst

Angular 2 组件用于 Sunburst 控件 。

Use the wj-sunburst component to add Sunburst controls to your Angular 2 applications. 关于跟多 Angular 2标记语言，请参见Angular 2 Markup。

WjSunburst 组件继承自 Sunburst 控件，继承了全部的属性，事件和方法 。

The wj-sunburst component may contain WjFlexChartLegend and WjFlexPieDataLabel child components.
