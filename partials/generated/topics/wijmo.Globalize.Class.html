wijmo

wijmo.js

类实现数字和日期的格式化和解析。

默认地, Globalize 使用美国英语文化。 为了切换多种文化，包含wijmo文件之后的合适的 wijmo.culture.*.js 文件.

方法

名称 参数 描述
S format​(value, format, trim?, truncate?): string

value (any): 数字或者日期进行格式化（所有其他类型转换为字符串 ）。
format (string): 格式化数字或日期时使用的格式字符串。
trim? (boolean): 是否从数值结果中移除尾随零。
truncate? (boolean): 是否缩短数字值而不是四折五入。

格式化一个数字或者日期。

使用format 函数格式化的字符串与使用Globalize.js 和.NET 全局库处理的字符串相似。下面的表格包含了描述格式的可用的链接：

  • 标准数值格式化的字符串
  • 标准日期和时间格式的字符串
  • 自定义日期和时间格式化的字符串



返回值 给定值的字符串表示形式。
S formatDate​(value, format): string

value (Date): 数字或者日期格式化。
format (string): .NET风格的日期格式字符串.

使用当前的文化格式化日期。

format 参数包含.NET风格的 日期格式的字符串 补充如下:

  • Q, q 日历季度。
    • U 会计季度 (政府)。 u 会计季度 (私营部门)。 EEEE, EEE, EE, E 财政年度 (政府). eeee, eee, ee, e 财政年度 (私营部门)。

例如: var d = new Date(2015, 9, 1); // Oct 1, 2015 console.log(wijmo.Globalize.format(d, '"FY"EEEE"Q"U') + ' (US culture)'); > FY2016Q1 (US culture)



返回值 给定日期的字符串表示形式。
S formatNumber​(value, format, trim?, truncate?): string

value (number): 格式化数字。
format (string): .NET风格的标准数字格式的字符串(例如 'n2', 'c4', 'p0', 'g2', 'd2')。
trim? (boolean): 是否从结果中移除尾随零。
truncate? (boolean): 是否缩短值而不是四舍五入。

使用当前的文化格式化一个数字。

formatNumber 方法接受更多的.NET风格的 标准数值格式化字符串，除过不支持'e'和'x'格式（科学计数法和十六进制）的字符串。

数字格式的字符串采取这种表单 Axxccss, 如下这里:

  • A 是一个唯一的不区分大小写被称为格式说明符的字母文字。
  • xx 是一个被称为格式说明符的可选的整型。 该精确的说明符影响着结果中数字的位数。
  • cc 是一个可选的字符串，当格式化货币值时被用作重载货币符号。当格式化与当前默认的文化不同的货币值时，该字符串很有用 (例如，在应用中使用英国文化格式化欧元或者日元值时)。
  • ss 是一个可选的字符串，用来缩放数量。如果提供了该参数，必须以逗号分隔。数字以每1000为单位使用指定的逗号分隔。

下面的表格描述了标准的数字格式说明符，并且通过每一个格式说明符为默认的文化显示示例输出。

n 数字: formatNumber(1234.5, 'n2') => '1,234.50'
f 定点: formatNumber(1234.5, 'f2') => '1234.50'
g 常规 (没有尾随零): formatNumber(1234.5, 'g2') => '1,234.5'
d 十进制 (整型): formatNumber(-1234, 'd6') => '-001234'
x 十六进制 (整型): formatNumber(1234, 'x6') => '0004d2'
c 货币: formatNumber(1234, 'c') => '$ 1,234.00'
p 百分比: formatNumber(0.1234, 'p2') => '12.34 %'

当绘制大数据时，缩放说明符尤其有用。例如，下面的标记创建了一个地表人口与国内生产总值对比的图表。该原始数据表达了单位人口的百万国内生产总值。 在坐标轴格式下制定的缩放引起了图表显示以百万为单位的人口和以百亿为单位的国内生产总值：

<wj-flex-chart
  items-source="countriesGDP" binding-x="pop" chart-type="Scatter">
  <wj-flex-chart-series
    name="GDP" binding="gdp"></wj-flex-chart-series>
  <wj-flex-chart-axis
    wj-property="axisX" title="Population (millions)" 
    format="n0,,">
  </wj-flex-chart-axis>
  <wj-flex-chart-axis
    wj-property="axisY" title="GDP (US$ trillions)"
    format="c0,,">
  </wj-flex-chart-axis>
</wj-flex-chart>

value (string): 将一个字符串转换为整数。

 将字符串转换为整数。
