AngularJS 指令应用于 annotations.

wj-flex-chart-annotation 指令必须包含于 WjFlexChartAnnotationLayer 指令。

wj-flex-chart-annotation 指令用于所有类型的注释区域，如 Circle, Rectangle, Polygon 等等。 指令的注释区域的类型由 type 属性指定。

指令支持以下属性:

type
@ 指令注释的类型，可能的值有 Circle, Ellipse, Image, Line, Polygon, Rectangle, Square, Text.
attachment
@ AnnotationAttachment 值，定义注释的附件。
content
@ Circle, Ellipse, Image, Line, Polygon, Rectangle 或者 Square 注释文本。
end
@ Line 注释的截止位置。
height
@ Ellipse, Image 或者 Rectangle 注释的高度。
href
@ Image 注释的图片源。
is-visible
@ 注释的可见性。
length
@ Square 注释的长度。
name
@ 注释的名称。
offset
@ 注释的偏移量。
point
@ 注释的位置，间隔控件依赖 attachment 属性值。 该属性在以下注释类型有效 Circle, Ellipse, Image, Rectangle, Square 和 Text 。
point-index
@ 注释附加的序列的索引。
position
@ AnnotationPosition 值，注释的位置对应的 point.
radius
@ Circle 注释的半径。
series-index
@ 注释附件的数据序列的索引。
start
@ Line 注释的起始位置。
style
@ 注释的样式。
text
@ Text 注释的文本。
tooltip
@ 注释的工具提示。
width
@ Ellipse, Image or Rectangle 注释的宽度。
