模块 wijmo/wijmo.angular2.grid.detail

文件
wijmo.angular2.js

包含 wijmo.grid.detail 模块的 Angular 2 组件。

wijmo.angular2.grid.detail 是一个外部的 TypeScript 模块，可以通过其环境模块名称导入到您的代码中。例如：

import * as wjDetail from 'wijmo/wijmo.angular2.grid.detail';
import * as wjGrid from 'wijmo/wijmo.angular2.grid';
 
@Component({
    directives: [wjGrid.WjFlexGrid, wjDetail.WjFlexGridDetail],
    template: `
      <wj-flex-grid [itemsSource]="data">
          <template wjFlexGridDetail>
              Detail row content here...
          </template>
      </wj-flex-grid>`,
    selector: 'my-cmp',
})
export class MyCmp {
    data: any[];
}

类

名称 描述
WjFlexGridDetail

FlexGrid DetailRow 模板的Angular 2 Directive 指令。

wj-flex-grid-detail 指令必须在一个包含在 wj-flex-grid 组件中包含的一个 <template> 模板元素。

wj-flex-grid-detail 指令继承自 FlexGridDetailProvider 类，可以管理详细行的可见性，详细行的内容可以在 directive 指令标签中定义为任意的 HTML 片段。 该片段可以包含 Angular 2 绑定，组件以及指令。 row 和 item 模板变量可以在 Angular 2 绑定中使用，分别表示详细行的父 Row 以及 Row.dataItem 对象。
