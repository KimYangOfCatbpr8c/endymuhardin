DateTime 类

模块
wijmo
文件
wijmo.js

提供日期和时间的工具包。

方法

名称 参数 描述
S addDays​(value, days): Date

value (Date): 原始日期。
days (number): 为给定的日期添加天数。

 通过添加指定的天数到给定的日期上来获取新的日期。
S addHours​(value, hours): Date

value (Date): 原始日期。
hours (number): 向给定的日期添加数小时。

 获取一个新的日期，该日期将在给定日期中添加指定数量的小时。
S addMinutes​(value, minutes): Date

value (Date): 原始日期。
minutes (number): 向给定的日期添加数分钟。

 获取一个新的日期，该日期将在给定日期中添加指定数量的分钟时间。
S addMonths​(value, months): Date

value (Date): 原始日期。
months (number): 向给定的日期添加数月。

 获取一个新的日期，该日期将在给定日期中添加指定数量的月份。
S addSeconds​(value, seconds): Date

value (Date): 原始日期。
seconds (number): 向给定的日期添加数秒。

 获取一个新的日期，该日期将在给定日期中添加指定数量的秒数。
S addYears​(value, years): Date

value (Date): 原始日期。
years (number): 向给定的日期添加数年。

 获取一个新的日期，该日期将在给定日期中添加指定数量的年。
S clone​(date): Date

date (Date): 日期对象进行复制。

 为给定的日期对象创建一个副本。
S equals​(d1, d2): boolean

d1 (Date): 第一个日期
d2 (Date): 第二个日期。

 如果两个日期对象引用相同的日期和时间，则返回真。
S fromDateTime​(date, time): Date

date (Date): 日期对象包含这样的日期（天/月/年）。
time (Date): 日期对象包含这样的时间（时:分:秒）。

 获取日期对象，日期和时间日期设置在两个日期对象上。
S fromFiscal​(date, govt)

date (Date): 财年日期。
govt (boolean): 是否使用政府或企业财年。

 使用当前文化，将财年日期转换为日历日期。
S newDate​(): Date Gets a new Date object set to today's date and no time portion.
S sameDate​(d1, d2): boolean

d1 (Date): 第一个日期。
d2 (Date): 第二个日期。

 如果两个日期对象引用相同的日期（忽略时间），则返回真。
S sameTime​(d1, d2): boolean

d1 (Date): 第一个日期。
d2 (Date): 第二个日期。

 如果两个日期对象引用相同的时间（忽略日期），则返回真。
S toFiscal​(date, govt)

date (Date): 日历日期。
govt (boolean): 是否使用政府或企业财年。

 使用当前文化，将日历日期转换为财政日期。
