使用 KnockoutJS 绑定 InputColor 控件。

使用 wjInputColor 绑定添加 InputColor 控件，示例如下：

<p>Here is a InputColor control:</p>
<div 
  data-bind="wjInputColor: { value: theColor }">
</div>

wjInputColor 支持 InputColor 控件的所有可读写属性及事件。以下属性支持双向绑定：

  • isDroppedDown
  • text
  • value
