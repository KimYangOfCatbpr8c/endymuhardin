IWorkbook 类

模块
wijmo.xlsx
文件
wijmo.xlsx.js

表示 Excel 工作簿。该接口是 Excel Workbook 对象模型（WOM）的一个根定义， 提供了定义存储在 xlsx 文件中数据和属性的简单方式。

为了创建一个 xlsx 文件，您需要创建一个符合 IWorkbook 接口定义的 JavaScript 对象，同时创建子对象定义工作簿表单以及其中的列，行以及单元格。 接着将该对象传递给 XlsxConverter 类的 export 方法，该方法会返回给您一个相关的 xlsx 文件的内容，该内容可以被保存在磁盘 或上传到服务器。

为了读取一个 xlsx 文件数据，您需要将 xlsx 文件的内容传递给 XlsxConverter 类的 import 方法， 该方法将返回一个符合 IWorkbook 接口的 JavaScript 对象，该对象允许您容易地检查工作簿的数据和属性。您可以以各种 方式进一步地使用该对象，例如，将其内容绘制到 HTML 页面或者将其保存为一个数据数组，以便进一步处理或用于数据绑定。

属性

名称 类型 描述
F activeWorksheet? number 该 xlsx 文件中，活动表单的索引。
F application? string 出现在文件属性的生成该文件的应用程序的名称。
F company? string 出现在文件属性的生成该文件的公司名称。
F created? Date 该 xlsx 文件的创建时间。
F creator? string 该 xlsx 文件的创建者。
F lastModifiedBy? string 该 xlsx 文件的最后一次修改人。
F modified? Date 该 xlsx 文件的最后一次修改时间。
F reservedContent? any 该工作簿的保留内容。
F sheets IWorkSheet[] 定义一个 Excel Workbook 表单的数组。
F styles? IWorkbookStyle[] 该工作簿的样式表。
