AngularJS 指令用于 FlexRadar 控件。 使用 wj-flex-radar 指令想你的 AngularJs 应用添加雷达图。注意，指令和参数名必须使用小写加短线格式。 wj-flex-radar 指令支持下面的属性：

binding
@包含图表Y轴值的属性名。你可以在系列级别上覆盖它。
binding-x
@ 包含图表X轴值的属性名。你可以在系列级别上覆盖它。
chart-type
@ 选择这个系列对象的默认图表类型。你可以在系列级别上覆盖它。 The default chart type to use in rendering series objects. 你可以在系列级别上覆盖它。 查看 RadarChartType.
control
=指向这里指定创建的FlexRadar控件的引用。
footer
@ 在图表底部展示的图表内容（纯文本）。
footer-style
= 应用到图表底部的样式。
header
@ 在图表标题展示的图表内容（纯文本）。
header-style
= 应用到图表标题的样式。
initialized
& 在绑定完成对控件的初始化后触发。
is-initialized
= 一个值指示绑定是否完成对控件的初始化。
interpolate-nulls
@ 获取一个值，决定当数据为null是，插入值还是留空隙。

item-formatter
= 自定义数据点外观的格式化函数。
items-source
= 包含用于创建图表数据的数组或者ICollectionView 对象。
legend-toggle
@ 获取一个值，决定在点击图例时是否切换图表可见性。
options
= 只能应用与确定图表类型的 options 。
palette
= 包含用于展示每个系列的颜色的数组。
plot-margin
= 在控件边缘留空隙的像素数，或是 CSS-style 的margin。
stacking
@ Stacking 值指定系列堆积还是独立绘制。
reversed
@ reversed 决定角度是否为逆时针。
startAngle
@ 指定控件的其实角度，单位是度。
totalAngle
@ 指定控件的总角度，单位是度。
symbol-size
@ 用于在Scatter,LineSymbols, 和 SplineSymbols 图表中绘制数据点的符号尺寸，单位是像素，你可以在系列级别覆盖。
tooltip-content
@ 展示在ChartTooltip 属性中的文本。
rendering
& rendering 事件处理程序。
rendered
&rendered 事件处理程序 。
series-visibility-changed
& seriesVisibilityChanged 事件处理程序。
wj-flex-radar 指令可能包含下面的子指令： WjFlexChartAxis, WjFlexRadarSeries, WjFlexChartLegend 和 WjFlexChartDataLabel.
