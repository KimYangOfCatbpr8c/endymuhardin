模块 wijmo/wijmo.angular2.grid.multirow

文件
wijmo.angular2.js

包含用于 wijmo.grid.multirow 模块的 Angular 2组件。

wijmo.angular2.grid.multirow 是一个外部的 Typescript 模块，可以使用模块名导入到你的代码。比如：

import * as wjMultiRow from 'wijmo/wijmo.angular2.grid.multirow';
 
@Component({
    directives: [wjMultiRow.WjMultiRow],
    template: `<wj-multi-row></wj-multi-row>`,
    selector: 'my-cmp',
})
export class MyCmp {
}

类

名称 描述
WjMultiRow

Angular 2 组件用于 MultiRow 控件 。

Use the wj-multi-row component to add MultiRow controls to your Angular 2 applications. 关于跟多 Angular 2标记语言，请参见Angular 2 Markup。

WjMultiRow 组件继承自 MultiRow 控件，继承了全部的属性，事件和方法 。
