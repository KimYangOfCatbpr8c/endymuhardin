AngularJS 指令应用于 RadialGauge control.

使用 wj-radial-gauge 指令添加 radial gauges 到 AngularJS 应用中。 注意指令和参数名称必须使用小写字母+短线语法替代驼峰命名。 示例：

Here is a <b>RadialGauge</b> control:</p>
<wj-radial-gauge
  style="height:300px" 
  value="count" 
  min="0" max="10"
  is-read-only="false">
</wj-radial-gauge>

wj-radial-gauge 指令支持以下属性:

ng-model
@ 使用Angular的ng-model指令绑定控件的 value 属性。使用ng-model指令绑定属性能够获得一些共通的特性，例如验证，自动添加控件状态到form实例，等等。 使用 wj-model-property 重定义被 ng-model指定绑定的控件属性。
wj-model-property
@ 使用 ng-model 指令指定控件绑定的作用域。
control
= 指向该指令创建的 RadialGauge 的引用
auto-scale
@ 自动缩放。
format
@ 以文本显示测量值时的格式化字符串。
has-shadow
@ 是否显示阴影。
initialized
& 该事件在绑定完成控件特性值的初始化后被触发。
is-initialized
= 绑定是否已经完成控件特性值的初始化。
is-animated
@ 是否使用动画显示测量值的变化。
is-read-only
@ 是否只读。
min
@ 可显示的最小测量值。
max
@ 可显示的最大测量值。
show-text
@ ShowText 值，测量值显示的文本
step
@ 用户按下增加或减小按钮时的步进值。
start-angle
@ 起始位置的角度。
sweep-angle
@ 扫描的角度（可正可负） 。
thickness
@ 测量值的厚度，从0到1的值。
value
= 显示的测试值。
got-focus
& gotFocus 事件。
lost-focus
& lostFocus 事件。

wj-radial-gauge 指令可以包含一个或多个 WjRange 指令。
