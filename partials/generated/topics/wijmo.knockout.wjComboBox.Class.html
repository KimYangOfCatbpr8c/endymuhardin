使用 KnockoutJS 绑定 ComboBox 控件。

使用 wjComboBox 绑定添加 ComboBox 控件，例如:

<p>Here is a ComboBox control:</p>
<div data-bind="wjComboBox: {
  itemsSource: countries,
  text: theCountry,
  isEditable: false,
  placeholder: 'country' }">
</div>

wjComboBox 支持 ComboBox 控件上的所有可读写属性和事件。以下属性提供了双向绑定：

  • isDroppedDown
  • text
  • selectedIndex
  • selectedItem
  • selectedValue
