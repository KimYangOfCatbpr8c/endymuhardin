属性

名称 类型 描述
content any

获取或设置工具提示内容。

工具提示的内容可以指示为一个字符串，或者一个以一个 HitTestInfo 对象作为参数的函数。

工具提示内容通过一个模板生成，其中可以包含以下任意参数：

  • propertyName: 由该点所表示的数据对象的任意属性。
  • seriesName: 包含该数据点的系列名称（仅 FlexChart）。
  • pointIndex: 数据点的索引。
  • value: 数据点的值（对于 FlexChart 而言是 y-值，而对于 FlexPie 而言是项目值）。
  • x: 数据点的 x-值（仅 FlexChart）。
  • y: 数据点的 y-值（仅 FlexChart）。
  • name: 数据点的名称（对于 FlexChart 而言是 x-值，而对于 FlexPie 而言是图例项）。

参数必须包含在一对花括号中。例如：

  // 'country' 以及 'sales' 是数据对象上的属性。
  chart.tooltip.content = '{country}, sales:{sales}';

下一个示例展示了如何通过一个函数设置工具提示的内容。

  // 设置工具提示的内容。
  chart.tooltip.content = function (ht) {
    return ht.name + ":" + ht.value.toFixed();
  }

指定工具提示内容到一个特定的页面上的元素。

同样的提示可以显示任意数量的元素在页面中的信息。为了从元素上移除提示信息， 调用setTooltip 并且制定内容为null。
