模块 wijmo/wijmo.angular2.grid.sheet

文件
wijmo.angular2.js

包含 wijmo.grid.sheet 模块的 Angular 2 组件。

wijmo.angular2.grid.sheet 是一个外部的 TypeScript 模块，可以通过其环境模块名称导入到您的代码中。例如：

import * as wjSheet from 'wijmo/wijmo.angular2.grid.sheet';
 
@Component({
    directives: [wjSheet.WjFlexSheet],
    template: `<wj-flex-sheet></wj-flex-sheet>`,
    selector: 'my-cmp',
})
export class MyCmp {
}

类

名称 描述
WjFlexSheet

Angular 2 组件用于 FlexSheet 控件 。

使用 wj-flex-sheet 组件向你的 Angular2 应用添加 FlexSheet 控件。关于跟多 Angular 2标记语言，请参见Angular 2 Markup。

WjFlexSheet 组件继承自 FlexSheet 控件，继承了全部的属性，事件和方法 。

The wj-flex-sheet component may contain WjSheet child components.
WjSheet

WjFlexSheet Sheet 对象对应的 Angular 2 组件。

wj-sheet 组件必须包含在一个 WjFlexSheet 组件内部。

使用 wj-sheet 组件添加 Sheet 对象至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjSheet 组件继承自 Sheet 类，因此继承了其全部的属性，事件和方法。
