IWorkbookStyle 类

模块
wijmo.xlsx
文件
wijmo.xlsx.js

表示用做给 Excel 单元格，列和行添加样式的 Workbook 对象模型样式定义。

属性

名称 类型 描述
F basedOn? IWorkbookStyle

定义该样式继承自的基础的样式。

该属性仅适用于导出操作。该样式获取定义在基础样式中的全部属性， 同时可以通过设置其自有属性覆盖或添加新的属性。
F borders? IWorkbookBorder 单元格外边框设置。
F fill? IWorkbookFill 获取或设置单元格背景。
F font? IWorkbookFont 获取或设置字体属性。
F format? string

单元格值格式，使用 Excel 格式语法定义。

Excel 格式语法的描述可以在 这里 查看。

您可以使用 XlsxConverter 类上的xlsxNumberFormat 和 xlsxDateFormat 静态方法 以转换 .Net (Globalize) 格式至 Excel 格式。
F hAlign? HAlign 获取或设置文本的水平对齐方式。
F indent? number 文本缩进。 这是一个整数值，增量 1 表示三个空格。
F vAlign? VAlign 获取或设置文本的垂直对齐方式。
