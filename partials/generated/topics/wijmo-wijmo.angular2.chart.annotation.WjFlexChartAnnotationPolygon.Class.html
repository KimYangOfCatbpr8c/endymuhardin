表示

WjFlexChart Polygon 标注对象对应的 Angular 2 组件。

wj-flex-chart-annotation-polygon 组件必须包含在一个 WjFlexChartAnnotationLayer 组件内部。

通过 wj-flex-chart-annotation-polygon 组件添加 Polygon 标注对象至您的 Angular 2 应用程序。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjFlexChartAnnotationPolygon 组件继承自 Polygon 类，因此继承了其全部的属性，事件和方法。
