模块

Row 以及 Column 类的一个抽象基类。

属性

名称 类型 描述
allowDragging boolean 获取或设置一个值，该值表示是否用户可以通过鼠标将行或者列移动到一个新位置。
allowMerging boolean 获取或设置一个值，该值表示是否位于该行或该列的单元格可以被合并。
allowResizing boolean 获取或设置一个值，该值表示是否用户可以通过鼠标改变行或者列的尺寸。
collectionView ICollectionView 获取绑定到该行或者列的 ICollectionView 对象。
cssClass string 获取或设置当呈现非行或者列中间的行头或列头单元格时，使用的一个 CSS 类名。
grid FlexGrid 获取拥有该行或者列的 FlexGrid 对象。
index number 获取位于父集合中的行或列的索引。
isContentHtml boolean 获取或设置一个值，该值表示是否位于该行或该列中的单元格包含 HTML 格式的文本内容而不是纯文本。
isReadOnly boolean 获取或设置一个值，该值表示是否位于该行或该列中的单元格可以被编辑。
isSelected boolean 获取或设置一个值，该值表示是否行或者列处于被选中状态。
isVisible boolean

获取一个值，该值表示是否该行或列为可见，且不是收起状态。

该属性为只读状态。为改变一行或一列的可见性，请使用 visible 属性以取代。
pos number 获取该行或列的位置。
renderSize number 获取该行或列的呈现尺寸。 该属性受 visibility，默认尺寸，以及最小值和最大值的影响。
size number 获取或设置行或列的尺寸。 设置该属性为 null 或者负值将使得该元素使用父集合的默认尺寸。
visible boolean 获取或设置一个值，该值表示是否该行或列为可见。
wordWrap boolean 获取或设置一个值，该值表示是否位于该行或该列中的单元格可以折行显示。
