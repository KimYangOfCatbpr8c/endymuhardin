KnockoutJS 绑定用于 PivotPanel 对象。 使用 wjPivotPanel 绑定向你的 KnockoutJS 应用添加 PivotPanel 控件，比如： <div data-bind="wjPivotPanel: { itemsSource: rawData, control: thePanel, initialized: init }"> </div> wjPivotPanel 绑定支持所有 PivotPanel 的读写属性和事件
