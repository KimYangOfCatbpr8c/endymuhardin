表示

WjFlexChart ChartAnimation 对象对应的 Angular 2 组件。

wj-flex-chart-animation 组件必须包含在一个 WjFlexChart 或者 WjFlexPie 组件中。

使用 wj-flex-chart-animation 组件添加 ChartAnimation 对象至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjFlexChartAnimation 组件继承自 ChartAnimation 类，因此继承了其全部的属性，事件和方法。

属性

名称 类型 描述
animationMode AnimationMode 获取或设置是否该绘图区的点是逐点，逐系列或者全部一起开始动画。 全部的动画仍然会在指定的动画期间内完成。
(继承自 ChartAnimation).
axisAnimation boolean 获取或设置一个布尔值，该值表示是否动画应用到坐标轴。
(继承自 ChartAnimation).
duration number 获取或设置整个动画的执行长度，以毫秒为单位。
(继承自 ChartAnimation).
easing Easing 获取或设置应用到动画的缓动方程。
(继承自 ChartAnimation).
