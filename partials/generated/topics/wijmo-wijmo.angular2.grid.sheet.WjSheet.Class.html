WjFlexSheet Sheet 对象对应的 Angular 2 组件。

wj-sheet 组件必须包含在一个 WjFlexSheet 组件内部。

使用 wj-sheet 组件添加 Sheet 对象至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjSheet 组件继承自 Sheet 类，因此继承了其全部的属性，事件和方法。
