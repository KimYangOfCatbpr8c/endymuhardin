RangeSelector 类

RangeSelector 控件显示了一个范围选择器，允许用户选择在指定的 FlexChart 上显示的数据范围。

为使用 RangeSelector 控件，必须指定显示选中数据范围的 FlexChart 控件。

当最小值或者最大值发生变化时，将触发 rangeChanged 事件。 例如： 

 var rangeSelector = new wijmo.chart.interaction.RangeSelector(chart);
 rangeSelector.rangeChanged.addHandler(function () {
    // 执行相关的更新
    // 比方说，更新其他图表的显示范围
    update(rangeSelector.min, rangeSelector.max);
  });

属性

名称 类型 描述
isVisible boolean 获取或设置范围选择器的可见性。
max number 获取或设置范围的最大值。 如果没有设置，则最大值将自动计算。
maxScale number 获取或设置范围选择器的最大范围缩放。 最大缩放介于 0 和 1 之间。
min number 获取或设置范围的最小值。 如果没有设置，则最小值将自动计算。
minScale number 获取或设置范围选择器的最小范围缩放。 最小缩放介于 0 和 1 之间。
orientation Orientation 获取或设置范围选择器的方向。
seamless boolean 获取或设置一个值，该值决定是否最小值和最大值的处理器平滑移动。。
