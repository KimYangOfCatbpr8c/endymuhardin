模块 wijmo/wijmo.angular2.chart.analytics

文件
wijmo.angular2.js

包含 wijmo.chart.analytics 模块的 Angular 2 组件。

wijmo.angular2.chart.analytics 是一个外部的 TypeScript 模块，可以通过其环境模块名称导入到您的代码中。例如：

import * as wjAnalitics from 'wijmo/wijmo.angular2.chart.analytics';

WjFlexChartMovingAverage

WjFlexChart MovingAverage 对象对应的 Angular 2 组件。

wj-flex-chart-moving-average 组件必须包含在一个 WjFlexChart 组件内部。

使用 wj-flex-chart-moving-average 组件添加 MovingAverage 对象至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjFlexChartMovingAverage 组件继承自 MovingAverage 类，因此继承了其全部的属性，事件和方法。
