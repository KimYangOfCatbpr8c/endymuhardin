wijmo.vue.js

Wijmo for Vue 模块。 这个模块提供了 Vue 组件，封装了 Wijmo 控件。 为了使用它，你的程序需要添加对Vue 和 Wijmo 的js 文件和css 文件的引用 。 为了向页面添加Wijmo 控件，需要在 HTML 页面中包含适当的标签。 比如：下面的代码向Vue 页面中添加了一个 InputNumber 控件。

<wj-input-number
  format="c2"
  placeholder="Sales"
  :value.sync="view.currentItem.sales"
  :value-changed="refreshView"
  :min="0"
  :max="10000"
  :step="100"
  :is-required="false">
</wj-input-number>

这个例子说明了以下要点：

  1. Wijmo 控件包含以 'wj' 起始的标签名，随后是小写的控件名称，使用短线连接。
  2. 标签名匹配控件的属性和事件。

  3. 属性名前冒号指示该属性值应该      解释为JavaScript表达式(e.g. :min="0").

  4. 属性名就加 .sync 代表双向绑定。(e.g. :value.sync="view.currentItem.sales").
  5. 指定事件处理程序和单向属性为相同的方式。 (e.g. :value-changed="refreshView").
所有的Vue组件包含一个“control” 伪属性，允许在标记中使用控件的属性，比如：

<wj-list-box style="height:150px;width:250px;" :items-source="countries" control="listBox"> </wj-list-box> <p> selectedIndex: {{ listBox.selectedIndex }}</p> <p> selectedValue: {{ listBox.selectedValue }}</p> 这个标记语言设置控件的 listBox 属性为一个 ListBox 控件的实例，因此它可以在标记语言中使用。 listBox 属性必须定义在 data 包含Vue 上下文的数据对象中，否则组件属性不会正确的更新比如：

// Vue application
var app = new Vue({
  el: '#app',
  data: {
    listBox: null // will receive a reference to the ListBox control
  }
});

WjFlexChart

Vue 组件封装了 FlexChart 控件。

下面的例子展示了如何使用 Vue 标记实例化一个 FlexChart 控件。

<wj-flex-chart
    :items-source="data"
    binding-x="country"
    :header="props.header"
    :footer="props.footer">



    <wj-flex-chart-legend :position="props.legendPosition">
    </wj-flex-chart-legend>
    <wj-flex-chart-axis wj-property="axisX" :title="props.titleX">
    </wj-flex-chart-axis>
    <wj-flex-chart-axis wj-property="axisY" :title="props.titleY">
    </wj-flex-chart-axis>



    <wj-flex-chart-series name="Sales" binding="sales">
    </wj-flex-chart-series>
    <wj-flex-chart-series name="Expenses" binding="expenses">
    </wj-flex-chart-series>
    <wj-flex-chart-series name="Downloads" binding="downloads">
    </wj-flex-chart-series>
</wj-flex-chart>
代码设置 itemsSource 为一个集合，包含图表数据，bindingX 属性设置为数据属性，包含图表的X值。 也可以设置图表的header 和 footer 属性定义在图表上方和下放显示的标题。 wj-flex-chart-legend and wj-flex-chart-axis 组件用来子定义控件的图例和轴。 最终，三个 wj-flex-chart-series 组件用来指定图表中的数据属性。
WjFlexChartAxis FlexChart 控件中的 Axis 组件。
WjFlexChartLegend FlexChart 控件中的 Legend 组件。
WjFlexChartSeries FlexChart 控件中的 a Series 组件。
WjFlexGrid

Vue 组件封装了 FlexGrid 控件。 下面的例子展示了如何使用 Vue 标记实例化一个 wijmo.grid.FlexGrid 控件。

<wj-flex-grid
  :items-source="data">
  <wj-flex-grid-column binding="name" header="Name">
  </wj-flex-grid-column>
  <wj-flex-grid-column binding="sales" header="Sales" format="c0">
  </wj-flex-grid-column>
  <wj-flex-grid-column binding="expenses" header="Expenses" format="c0">
  </wj-flex-grid-column>
  <wj-flex-grid-column binding="active" header="Active">
  </wj-flex-grid-column>
  <wj-flex-grid-column binding="date" header="Date">
  </wj-flex-grid-column>
</wj-flex-grid>
这段代码设置 itemsSource 属性为一个集合，包含表单数据，然后使用wj-flex-grid-column组件展示列。
WjFlexGridColumn 在FlexGrid 控件中代表 Column 的组件。
WjFormat

<p>value: {​{ theAmount | wj-format 'c' }}</p>
<p>date: {​{ theDate | wj-format 'd' }}</p>
WjInclude

Vue组件包含给定的HTML 片段。 wj-include 组件接受一个 src 属性，指定加载的文件并包含在文档中。 比如：

<wj-popup control="modalDialog" :modal="true" :hide-trigger="None">
  <wj-include src="includes/dialog.htm"></wj-include>
</wj-popup>

WjLinearGauge

Vue 组件封装了 LinearGauge 控件。 下面的代码展示了如何实例化wijmo.gauge.LinearGauge 控件。

<wj-linear-gauge
    :min="0" :max="1000" :step="50" :is-read-only="false"
    format="c0" :thumb-size="20"
    :show-ranges="false"
    :value.sync="sales">
    <wj-range wj-property="face" :thickness="0.5">
    </wj-range>
    <wj-range wj-property="pointer" :thickness="0.5">
    </wj-range>
    <wj-range :min="0" :max="333" color="red">
    </wj-range>
    <wj-range :min="333" :max="666" color="gold">
    </wj-range>
    <wj-range :min="666" :max="1000" color="green">
    </wj-range>
</wj-linear-gauge>

代码汇总通过min, max, step, 和 isReadOnly 属性来表示仪表盘的范围并允许用户来编辑它的值， 然后，它绑定仪表盘的 value 属性到控制器中的sales 代码。

然后它设置 format, thumbSize, 和 showRanges属性来定义仪表盘的外观，最终，标记设置设置 face 的粗细，pointer，ranges，extra ranges 来根据仪表盘当前值控制值范围的颜色。
