AngularJS 指令应用于 Calendar 控件.

使用 wj-calendar 指令添加 Calendar 控件到 AngularJS 应用中。 注意指令和参数名称必须使用小写字母+短线语法替代驼峰命名。 示例：

<p>Here is a Calendar control:</p>
<wj-calendar 
  value="theDate">
</wj-calendar>

在线示例

下面的例子创建了一个 Calendar 控件，并绑定到控制器公开的 'date' 变量上。 使用 min 和 max 属性限制日期的有效范围.

wj-calendar 指令支持以下属性:

ng-model
@ 使用Angular的ng-model指令绑定控件的 value 属性。使用ng-model指令绑定属性能够获得一些共通的特性，例如验证，自动添加控件状态到form实例，等等。 使用 wj-model-property 重定义被 ng-model指定绑定的控件属性。
wj-model-property
@ 使用 ng-model 指令指定控件绑定的作用域。
control
= 该指令创建的指向 Calendar 控件的引用。
display-month
= 指定日历显示的月份。
first-day-of-week
@ 指定星期开始日。
initialized
& 该事件在绑定完成控件特性值的初始化后被触发。
is-initialized
= 绑定是否已经完成控件特性值的初始化。
item-formatter
= 自定义显示的格式化函数。
max
@ 最大有效日期 (格式化字符串为 "yyyy-MM-dd")。
min
@ 最小有效日期 (格式化字符串为 "yyyy-MM-dd")。lt;/dd>
month-view
@ 指定显示月视图还是年视图。
show-header
@ 是否显示标题区域。
value
= 编辑的日期。
got-focus
& gotFocus 事件。
lost-focus
& lostFocus 事件。
value-changed
& valueChanged 事件。

min 和 max 属性需要指定为格式化 "yyyy-MM-dd" 的日期字符串， 技术上讲，你可以使用符合W3C [RFC 3339] 定义的任何格式化字符，只要它们是能够被HTML5支持格式化字符。
