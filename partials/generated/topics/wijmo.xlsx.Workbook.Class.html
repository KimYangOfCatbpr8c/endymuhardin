方法

名称 参数 描述
S fromXlsxFormat​(xlsxFormat): string[]

xlsxFormat (string): 一个 Excel 格式字符串，有可能包含用分号分隔的多段格式。

 将 xlsx 多段格式字符串转换为相关的 wijmo 格式的数组。

返回值 .NET 格式的字符串数组，其中每一个元素对应一个独立的Excel 格式。返回的数组至少包含一个元素，它可以是空字符串以防传递的Excel格式为空。
load​(base64)

base64 (string): 包含 xlsx 文件内容的 base64 编码的字符串。

从 base64 编码的字符串或者数据 URL 加载。

例如： 

// 该示例打开一个从 Open File 对话框选择的 xlsx 文件
// 并创建一个工作簿实例以加载该文件。
 
// HTML
<input type="file" 
    id="importFile" 
    accept="application/vnd.openxmlformats-officedocument.spreadsheetml.sheet" 
/>
 
// JavaScript
var workbook, // 接收导入的 IWorkbook
    importFile = document.getElementById('importFile');
 
importFile.addEventListener('change', function () {
    loadWorkbook();
});
 
function loadWorkbook() {
    var reader,
        workbook,
        file = importFile.files[0];
    if (file) {
        reader = new FileReader();
        reader.onload = function (e) {
           workbook = new wijmo.xlsx.Workbook(),
           workbook.load(reader.result);
        };
        reader.readAsDataURL(file);
    }
}
save​(fileName?): string

fileName? (string): 保存的 xlsx 文件的名称。

保存为一个包含 xlsx 文件内容的 base64 编码的字符串，并返回该字符串。如果指定了 fileName，则保存到 xlsx 文件。

例如： 

// 该示例保存一个工作簿实例至 xlsx 文件，在其中一个单元格包含一段 “Hello, Excel!” 的文本。
// 单击。
 
// HTML
<button 
    onclick="saveXlsx('Hello.xlsx')">
    Save
</button>
 
// JavaScript
function exportXlsx(fileName) {
    // 定义一个工作簿的内容。
    var workbook = new wijmo.xlsx.Workbook(),
        worksheet = new wijmo.grid.xlsx.WorkSheet(),
        workbookRow = new wijmo.grid.xlsx.WorkbookRow(),
        workbookCell = new wijmo.grid.xlsx.WorkbookCell();
    workbookCell.value = 'Hello, Excel!';
    workbookRow.cells.push(workbookCell);
    worksheet.rows.push(workbookRow);
    workbook.sheets.push(worksheet);
 
    // 如果指定了 fileName 文件名，则 Workbook.save 方法将导出工作簿至 xlsx 文件并返回该 xlsx 内容的 base64 编码的字符串。
    workbook.save(fileName);
}

返回值 A base-64 string representing the content of the file.
S tableAddress​(xlsxIndex): ITableAddress

xlsxIndex (string): 一个可以包含从 A 开始的字母表示的列索引以及/或者从 1 开始计算的行索引的一个字母数值格式的 Excel 索引标记，形如 "D15"，"D" 或者 "15"。字母的列索引可以为小写字母或大写字母。

 将 Excel 的字母数值形式的单元格，行或者列的索引转换为一对从零开始计算的行/列索引值。

返回值 返回一个对象，包含基于0的行和列索引的row 和 col 属性。如果行或者列没有指定，那么对应的属性为undefined。
S toXlsxDateFormat​(format): string

format (string): Wijmo 日期格式。

 将 wijmo 日期格式转换为 Excel 格式。

返回值 Excel 格式的表示 。
S toXlsxNumberFormat​(format): string

format (string): Wijmo 数值格式。

 将 wijmo 数值格式转换为 xlsx 格式。

返回值 Excel格式表示 。
S xlsxAddress​(row, col, absolute?, absoluteCol?): string

row (number): 从零开始计算的行索引，或者当仅需要转换列索引时，可以为 null。
col (number): 从零开始计算的列索引，或者当仅需要转换行索引时，可以为 null。
absolute? (boolean): True 值表示行和列的索引都应当返回绝对索引格式（形如 $D$7）。absoluteCol 参数允许重新定义列索引的行为
absoluteCol? (boolean): True 值表示列的索引返回绝对值索引格式。

 将从零开始计算索引值的单元格，行或者列的索引转换为 Excel 字母数字表示。

返回值 代表Excel 单元格索引的字母数字。
