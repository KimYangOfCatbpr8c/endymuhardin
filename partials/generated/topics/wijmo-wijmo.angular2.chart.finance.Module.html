模块 wijmo/wijmo.angular2.chart.finance

文件
wijmo.angular2.js

包含 wijmo.chart.finance 模块的 Angular 2 组件。

wijmo.angular2.chart.finance 是一个外部的 TypeScript 模块，可以通过其环境模块名称导入到您的代码中。例如：

import * as wjFinance from 'wijmo/wijmo.angular2.chart.finance';
 
@Component({
    directives: [wjFinance.WjFinancialChart, wjFinance.WjFinancialChartSeries],
    template: `
      <wj-financial-chart [itemsSource]="data" [bindingX]="'x'">
          <wj-financial-chart-series [binding]="'y'"></wj-financial-chart-series>
      </wj-financial-chart>`,
    selector: 'my-cmp',
})
export class MyCmp {
    data: any[];
}

类

名称 描述
WjFinancialChart

FinancialChart控件对应的 Angular 2 组件。

使用 wj-financial-chart 组件添加 FinancialChart 控件至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjFinancialChart 组件继承自 FinancialChart 控件，因此继承了其全部的属性，事件和方法。

wj-financial-chart 组件可以包含 WjFlexChartAxis， WjFlexChartLegend，WjFinancialChartSeries 子组件，同时可以包含全部的来自于 wijmo.angular2.chart.finance.analytics 模块的组件。
WjFinancialChartSeries

WjFinancialChart FinancialSeries 对象对应的 Angular 2 组件。

wj-financial-chart-series 组件必须包含在一个 WjFinancialChart 组件内部。

使用 wj-financial-chart-series 组件添加 FinancialSeries 对象至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjFinancialChartSeries 组件继承自 FinancialSeries 类，因此继承了其全部的属性，事件和方法。
