FlexGrid FlexGridFilter 对象对应的 Angular 2 组件。

wj-flex-grid-filter 组件必须包含在一个 WjFlexGrid 组件内部。

使用 wj-flex-grid-filter 组件添加 FlexGridFilter 对象至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjFlexGridFilter 组件继承自 FlexGridFilter 类，因此继承了其全部的属性，事件和方法。

属性

名称 类型 描述
defaultFilterType FilterType

获取或设置所使用的默认筛选器类型。

该值可以在指定列的筛选器上被覆盖。例如，以下代码创建一个筛选器，在全部的列上 按照指定条件筛选，除了“ByValue”列以外：

var f = new wijmo.grid.filter.FlexGridFilter(flex);
f.defaultFilterType = wijmo.grid.filter.FilterType.Condition;
var col = flex.columns.getColumn('ByValue'),
    cf = f.getColumnFilter(col);
cf.filterType = wijmo.grid.filter.FilterType.Value;

filterChanged CellRangeEventArgs

在一个列筛选器被用户编辑完成之后触发。

使用该事件参数获取拥有该筛选器的列，并确定是否该变更被应用或取消。


(继承自 FlexGridFilter).
filterChanging CellRangeEventArgs

当一个列筛选器即将被用户编辑时触发。

如果您希望覆盖筛选器默认的设置，您可以通过该事件自定义该列筛选器。

比如说，以下代码当筛选器条件使用的操作为符 null 时，设置该操作符为 ‘contains’：

filter.filterChanging.addHandler(function (s, e) {
  var cf = filter.getColumnFilter(e.col);
  if (!cf.valueFilter.isActive && cf.conditionFilter.condition1.operator == null) {
    cf.filterType = wijmo.grid.filter.FilterType.Condition;
    cf.conditionFilter.condition1.operator = wijmo.grid.filter.Operator.CT;
  }
});
