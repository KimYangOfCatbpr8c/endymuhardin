Aggregate 枚举

模块
wijmo
文件
wijmo.js

指定聚合的类型来计算在一个组中的值。

值

名称 描述
Avg 返回一个组中数值的平均值。
Cnt 返回一个组中值不为null的数量。
CntAll Returns the count of all values in the group (including nulls).
Max 返回一个组中的最大值。
Min 返回一个组中的最小值。
None 没有集合。
Rng 返回一个组中最大数值和最小数值之间的不同。
Std 返回一个组中数值的样本标准偏差。 (使用基于n-1 的公式)。
StdPop 返回一个组中值的人口标准偏差。 (使用基于n的公式).
Sum 返回一个组中数值的集合。
Var 返回一个组中数值的样本标准偏差。 (使用基于n-1 的公式).
VarPop 返回一个组中值的人口变量。 (使用基于n的公式).
