PrintDocument 类

模块
wijmo
文件
wijmo.js

类可以创建自定义文档的打印。

PrintDocument 类可以创建打印或输出为PDF文件。大多数浏览器允许你选择纸张大小，方向，空间，以及是否包含页眉和页脚。

为了使用，实例化一个PrintDocument，使用append 方法添加内容，并且通过调用print 方法来完成。

例如： 

// 创建文档
var doc = new wijmo.PrintDocument({
  title: 'PrintDocument Test'
});



// 添加一些简单的文本
doc.append('
Printing Example
');
doc.append('
This document was created using the PrintDocument class.
');



// 添加一些存在的元素
doc.append(document.getElementById('gaugeControl'));



// 打印文档(或者输出它为 PDF)
doc.print();
