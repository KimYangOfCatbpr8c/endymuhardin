颜色类。

Color 类指定CSS字符串来解析颜色，并且暴露了它们的红，绿，蓝，和alpha通道作为读写属性。

Color 类也提供了fromHsb 和fromHsl 方法，使用HSB和HSL颜色模型代替RGB创建颜色。 并且用see:getHsb 和getHsl 方法来检索使用这些颜色模型的颜色组件。

最后，Color 类提供了一个interpolate 方法来通过插值使用HSL模型和两种颜色之间 创建颜色。这种方法是特别有用的，用于创建彩色动画的方法。

构造函数

名称 参数 描述
constructor​(color): Color

color (string): CSS color specification.

 Initializes a new Color from a CSS color specification.

属性

名称 类型 描述
a number 获取或设置Color的alpha组件，从0到1的一个范围（零是透明的，一是光滑的）。
b number 获取或设置Color的蓝色组件，从0到255的一个范围。
g number 获取或设置Color的绿色组件，从0到255的一个范围。
r number 获取或设置Color的红色组件，从0到255的一个范围。

方法

名称 参数 描述
equals​(clr): boolean

clr (Color): Color 与Color 进行比较。

 如果一个Color与Color具有相同的值，则返回真值。
S fromHsb​(h, s, b, a?): Color

h (number): Hue值，从0到1。
s (number): Saturation值，从0到1。
b (number): Brightness值，从0到1。
a? (number): Alpha值，从0到1。
(默认值: 1)

 使用指定的HSB值创建一个新的Color。
S fromHsl​(h, s, l, a?): Color

h (number): Hue值，从0到1。
s (number): Saturation值，从0到1。
l (number): Lightness值，从0到1。
a? (number): Alpha值，从0到1。
(默认值: 1)

 使用指定的HSL值创建一个新的Color。
S fromRgba​(r, g, b, a?): Color

r (number): 红色通道的值，从0到255。
g (number): 绿色通道的值，从0到255。
b (number): 蓝色通道的值，从0到255。
a? (number): alpha通道的值，从0到1。
(默认值: 1)

 创建一个使用RGBA颜色通道的值指定新Color。
S fromString​(value): Color

value (string): 字符串包含指定的CSS颜色规范。

 从一个CSS颜色字符串中创建新的Color 。

返回值 一个新的Color，如果该字符串不能被解析成一个颜色则为null。
getHsb​(): number[] 获取该颜色的HSB组件的数组。
getHsl​(): number[] 获取该颜色的HSL组件的数组。
S interpolate​(c1, c2, pct): Color

c1 (Color): 第一个颜色。
c2 (Color): 第二个颜色。
pct (number): 零和一个值之间的值，该值决定了该插值应该接近于第二色。

 通过插值的颜色创建一个Color。
S toOpaque​(c, bkg?): Color

c (any): Color 被转换成一个不透明的颜色（该颜色也可以被指定为字符串）。
bkg? (any): 在去除透明时使用的背景颜色 （默认值为白色）。

 获取给定颜色的最接近的不透明颜色。
toString​(): string 获取Color的字符串表示形式。
