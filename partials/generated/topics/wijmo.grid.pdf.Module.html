模块 wijmo.grid.pdf

文件
wijmo.grid.pdf.js

定义用来导出 FlexGrid 至 PDF 的 FlexGridPdfConverter 类。

类

名称 描述
FlexGridPdfConverter 提供导出 FlexGrid 至 PDF 的功能。

接口

名称 描述
ICellStyle 表示单元格的外观。
IFlexGridDrawSettings 表示 draw 以及 drawToPosition 方法所使用的设置。
IFlexGridExportSettings 表示 export 方法所使用的设置。
IFlexGridStyle 表示待导出的 FlexGrid 的外观。

枚举

名称 描述
ExportMode 指定是整个表单还是仅仅其中某一个部分应当呈现。
ScaleMode 指定表单内容如何拉伸以适应页面大小。
