Clipboard 类

提供剪贴板操作的实用方法的静态类。

Clipboard 类提供了静态 copy 和paste 方法，它可以使用控件来自定义剪贴板操作中的剪贴板内容。

例如，下面的代码演示了如何控制可以截取剪贴板快捷键，并提供自定义剪贴板处理：

rootElement.addEventListener('keydown', function(e) {
  // 拷贝： ctrl+c 或者 ctrl+Insert
  if (e.ctrlKey && (e.keyCode == 67 || e.keyCode == 45)) {
    var text = this.getClipString();
    Clipboard.copy(text);
    return;
  }
  // 粘贴： ctrl+v 或者 shift+Insert
  if ((e.ctrlKey && e.keyCode == 86) || (e.shiftKey && e.keyCode == 45)) {
    Clipboard.paste(function (text) {
      this.setClipString(text);
    });
    return;
  }
});

方法

名称 参数 描述
S copy​(text)

text (string): 文本复制到剪切板上。

拷贝字符串到剪切板上。

此方法只适用在用户按下剪贴板复制命令后立即调用该方法（例如ctrl+c）。
S paste​(callback)

callback (Function): 在检索剪贴板内容时调用的函数。函数接收剪贴板内容作为参数。

从剪切板上获取字符串。

此方法只适用在用户按下剪贴板粘贴命令后立即调用该方法（例如ctrl+v）。
