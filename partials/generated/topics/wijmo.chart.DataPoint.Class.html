DataPoint 类

派生类
WjFlexChartDataPoint
模块
wijmo.chart
文件
wijmo.chart.js

表示一个数据点（带有 x 和 y 坐标轴）的类。

X 和 Y 轴可以指定为一个数值或者一个日期对象（基于时间的数据）。

构造函数

名称 参数 描述
constructor​(x?, y?): DataPoint

x? (any): 新数据点的X坐标。
(默认值: 0)
y? (any): 新数据点的Y坐标。
(默认值: 0)

 初始化类 DataPoint 的一个新实例。

属性

名称 类型 描述
F x any 获取或设置该 DataPoint 的 X 轴值。
F y any 获取或设置该 DataPoint 的 Y 轴值。
