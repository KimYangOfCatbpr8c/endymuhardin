WjFlexChart Legend控件对应的 Angular 2 组件。

wj-flex-chart-legend 组件必须包含在一个 WjFlexChart 组件内部。

使用 wj-flex-chart-legend 组件添加 Legend 对象至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjFlexChartLegend 组件继承自 Legend 类，因此继承了其全部的属性，事件和方法。

属性

名称 类型 描述
position Position 获取或设置一个枚举值，该值决定是否显示图例以及相对于图表图例区出现的位置。
(继承自 Legend).
