模块 wijmo/wijmo.angular2.chart.interaction

文件
wijmo.angular2.js

包含 wijmo.chart.interaction 模块的 Angular 2 组件。

wijmo.angular2.chart.interaction 是一个外部的 TypeScript 模块，可以通过其环境模块名称导入到您的代码中。例如：

import * as wjInteraction from 'wijmo/wijmo.angular2.chart.interaction';
import * as wjChart from 'wijmo/wijmo.angular2.chart';
 
@Component({
    directives: [wjChart.WjFlexChart, wjInteraction.WjFlexChartRangeSelector, wjChart.WjFlexChartSeries],
    template: `
      <wj-flex-chart [itemsSource]="data" [bindingX]="'x'">
          <wj-flex-chart-range-selector></wj-flex-chart-range-selector>
          <wj-flex-chart-series [binding]="'y'"></wj-flex-chart-series>
      </wj-flex-chart>`,
    selector: 'my-cmp',
})
export class MyCmp {
    data: any[];
}

类

名称 描述
WjFlexChartGestures

WjFlexChart ChartGestures 对象对应的 Angular 2 组件。

wj-flex-chart-gestures 组件必须包含在一个 WjFlexChart 组件内部。

使用 wj-flex-chart-gestures 组件添加 ChartGestures 对象至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjFlexChartGestures 组件继承自 ChartGestures 类，因此继承了其全部的属性，事件和方法。
WjFlexChartRangeSelector

WjFlexChart RangeSelector 对象对应的 Angular 2 组件。

wj-flex-chart-range-selector 组件必须包含在一个 WjFlexChart 组件内部。

使用 wj-flex-chart-range-selector 组件添加 RangeSelector 对象至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjFlexChartRangeSelector 组件继承自 RangeSelector 类，因此继承了其全部的属性，事件和方法。
