WjFlexChart Axis控件对应的 Angular 2 组件。

wj-flex-chart-axis 组件必须包含在一个 WjFlexChart 组件内部。

使用 wj-flex-chart-axis 组件添加 Axis 对象至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjFlexChartAxis 组件继承自 Axis 类，因此继承了其全部的属性，事件和方法。

itemFormatter Function

获取或设置坐标轴标签的 itemFormatter 函数。

如果指定，该函数接受两个参数：

  • render engine: 用作格式化标签的 IRenderEngine 对象。
  • current label: 具有以下属性的字符串值：
    • value: 待格式化的坐标轴标签的值。
    • text: 在该标签上使用的文本。
    • pos: 该标签待呈现的位置，位于控件坐标系。
    • cls: 待应用给该标签的 CSS 类。

该函数返回属性发生变化的标签的标签参数。

例如： 

chart.axisY.itemFormatter = function(engine, label) {
    if(label.val > 5){
        engine.textFill = 'red'; // 红色文本
        label.cls = null;// 没有默认的 CSS
     }
    return label;
}
