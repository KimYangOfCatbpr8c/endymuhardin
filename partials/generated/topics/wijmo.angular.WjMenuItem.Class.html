AngularJS 指令应用于 menu items.

wj-menu-item 指令必须包含于 WjMenu 指令. 支持以下特性:

cmd
= 项目被点击时控制器中执行的函数。
cmd-param
= 项目被点击时传递给执行函数cmd的参数。
value
= 项目被点击时选中的值 (通过 this 或 cmd 获得).

菜单项显示的内容有可能是包含 AngularJS 绑定和指令的 HTML 代码段。 也可以使用 ng-repeat 和 ng-if 指令填充菜单项。 除了控制器上任何可用的属性变量之外， $item, $itemIndex 和 $control 模板变量也可用于 AngularJS 绑定中，用以指定数据项，索引，以及父控件等。
