AngularJS 指令应用于 ListBox 和 Menu 项目模板。

wj-item-template 指令必须包含在一个 WjListBox 或者 WjMenu 指令中。

wj-item-template 定义了一个 ListBox 的项目模板，并绑定到 Menu 控件上。 该模板包含使用了 AngularJS 绑定和指令的任意 HTML 代码段。除了控制器上任何可用的属性变量之外， $item, $itemIndex 和 $control 模板变量也可用于 AngularJS 绑定中，用以指定数据项，索引，以及父控件等。

注意指令和参数名称必须使用小写字母+短线语法替代驼峰命名。 示例：

<p>Here is a ListBox control with an item template:</p>
<wj-list-box items-source="musicians">
    <wj-item-template>
        {​{$itemIndex}}. <b>{​{$item.name}}</b>
        <br />
        <img ng-src="{​{$item.photo}}"/>
    </wj-item-template>
</wj-list-box>
