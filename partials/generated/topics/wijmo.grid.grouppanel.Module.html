模块 wijmo.grid.grouppanel

文件
wijmo.grid.grouppanel.js

提供在绑定状态的 FlexGrid 控件下的一个用来编辑分组的拖放界面的扩展包。

类

名称 描述
GroupPanel

该 GroupPanel 控件提供在绑定状态的 FlexGrid 控件下的一个用来编辑分组的拖放界面。

它允许用户从 FlexGrid 拖拽一列至该面板，并在该面板范围内移动分组。 用户也可以在该面板内单击分组标记，以基于该分组列进行排序，或者移除分组。

为了使用一个 GroupPanel，添加其至一个包含一个 FlexGrid 控件的页面， 并设置该面板的 grid 属性至该 FlexGrid 控件。例如：

// 创建一个 FlexGrid 
var flex = new wijmo.grid.FlexGrid('#flex-grid');
flex.itemsSource = getData();
// 添加一个 GroupPanel  以编辑数据分组
var groupPanel = new wijmo.grid.grouppanel.GroupPanel('#group-panel');
groupPanel.placeholder = "Drag columns here to create groups.";
groupPanel.grid = flex;
