表示 AnnotationLayer 支持的批注对象的基类。

构造函数

名称 参数 描述
constructor​(options?): AnnotationBase

options? (): 包含初始化数据的Javascript对象。

 初始化类 AnnotationBase 的一个新实例。

属性

名称 类型 描述
attachment AnnotationAttachment 获取或设置批注的附件。
isVisible boolean 获取或设置批注的可见性。
name string 获取或设置批注的名称。
offset Point 获取或设置批注相对于 point 的偏移量。
point DataPoint 获取或设置批注的位置。 位置的坐标系由 attachment 属性决定。 更多细节，请参见 AnnotationAttachment。
pointIndex number 获取或设置批注的数据点索引。 仅当 attachment 属性设置为 DataIndex 时应用该设置。
position AnnotationPosition 获取或设置批注的位置。 该位置相对于 point。
seriesIndex number 获取或设置批注的数据系列索引。 仅当 attachment 属性设置为 DataIndex 时应用该设置。
style any 获取或设置批注的样式。
tooltip string 获取或设置批注的工具提示。
