模块 wijmo.gauge

文件
wijmo.gauge.js

定义 RadialGauge，LinearGauge，以及 BulletGraph 控件。

和其他诸多仪表控件不同，Wijmo 仪表专注于展示的数据， 几乎没有多余的颜色和标记元素。它们被设计为易于使用 和读取，尤其是在小屏幕设备上。

Wijmo 仪表由 Range 对象组成。每一个 Wijmo 仪表 至少具有两个范围：“face” 和 “pointer”。

  • “face” 表示仪表的背景。面板范围的“min”和“max”属性关联到仪表控件的“min”和“max”属性， 它们将限制着仪表可以显示的数值范围。
  • “pointer”是一个指示仪表当前值的范围。该 pointer 范围的“max”属性关联到仪表的“value” 属性。

除了这两个特殊的范围之外，仪表还可以具有一些其他的额外范围，这些范围将添加到 “ranges”集合。这些额外的范围可用来做两类事情：

  • 默认情况下，额外的范围显示为仪表背景的一部分。通过这种方式您可以在图表上显示 不同的区域，比如说“good”，“average”，“bad”。
  • 如果您设置了仪表的“showRanges”属性为 false，则额外的范围将不会显示。取而代之，它们将用来 自动地基于当前值设置“pointer”的颜色。

BulletGraph

BulletGraph 是线性图表的一种，特别设计用作 dashboard 中的使用场景。 它显示一个单独的关键指标，随同一个测量指标，以及一个定量的范围，以便可以立即测定出 该测量值是 好的，不好的或者其他一些状态。

Bullet Graphs 是由 dashboard 设计大师 Stephen Few 创建并推广。您可以在 维基百科上找到更多的详细介绍和示例。

LinearGauge

LinearGauge 显示了一个线性的缩放，具有一个表示单一值的指示器，可选地可以有一些范围可以表示参考值。

如果您设置了仪表的 isReadOnly 属性为 false，则用户可以通过单击仪表编辑值。

Range

定义 Gauge 控件所使用的范围。

Range 对象具有 min 以及 max 属性以定义该范围的领地，同时还提供了 color 和 thickness 属性，以定义该范围的外观。

每一个 Gauge 控件至少具有两个范围： ‘face’ 定义了改仪表的最小值和最大值，而‘pointer’显示该仪表的当前值。

除了内建的范围之外，仪表也可以具有额外的范围，这些范围用来显示重要的区域范围（比如说，过低，中等，过高值）。
