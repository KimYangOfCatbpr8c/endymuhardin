ChartAnimation 类

派生类
WjFlexChartAnimation
模块
wijmo.chart.animation
文件
wijmo.chart.animation.js

表示 FlexChart，FinancialChart 以及 FlexPie 的动画。

ChartAnimation 提供了内置的动画，用来在加载和更新图表时显示动画。 该动画可以通过若干属性由用户进行定制，包括动画期间，缓动函数，动画模式。

构造函数

名称 参数 描述
constructor​(chart, options?): ChartAnimation

chart (FlexChartBase): ChartAnimation 依附的图表。
options? (any): 包含控件初始化数据的Javascript对象。

 初始化 ChartAnimation 类的一个新实例。

属性

名称 类型 描述
animationMode AnimationMode 获取或设置是否该绘图区的点是逐点，逐系列或者全部一起开始动画。 全部的动画仍然会在指定的动画期间内完成。
axisAnimation boolean 获取或设置一个布尔值，该值表示是否动画应用到坐标轴。
duration number 获取或设置整个动画的执行长度，以毫秒为单位。
easing Easing 获取或设置应用到动画的缓动方程。
