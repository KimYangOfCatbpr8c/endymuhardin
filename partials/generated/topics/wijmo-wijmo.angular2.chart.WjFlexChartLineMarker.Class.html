WjFlexChart LineMarker控件对应的 Angular 2 组件。

wj-flex-line-marker 组件必须包含在一个 WjFlexChart 组件内部。

使用 wj-flex-line-marker 组件添加 LineMarker 对象至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjFlexChartLineMarker 组件继承自 LineMarker 类，因此继承了其全部的属性，事件和方法。

属性

名称 类型 描述
alignment LineMarkerAlignment

获取或设置 LineMarker 内容的对齐方式。

默认情况下，LineMarker　靠右显示，位于目标点的底部。 使用 ‘|’ 联合对齐方式的设置。

// 设置对齐方式为靠左。
marker.alignment = wijmo.chart.LineMarkerAlignment.Left;
// 设置对齐方式为左上。
marker.alignment = wijmo.chart.LineMarkerAlignment.Left | wijmo.chart.LineMarkerAlignment.Top;
