使用 KnockoutJS 绑定 InputNumber 控件。

使用 wjInputNumber 绑定添加 InputNumber 控件，示例如下：

<p>Here is an InputNumber control:</p>
<div data-bind="wjInputNumber: {
  value: theNumber,
  min: 0,
  max: 10,
  format: 'n0',
  placeholder: 'number between zero and ten' }">
</div>

wjInputNumber 支持 InputNumber 控件的所有可读写属性及事件。以下属性支持双向绑定：

  • value
  • text
