模块 wijmo.olap

文件
wijmo.olap.js

包含提供了 OLAP 功能的组件，比如说 pivot table 以及图表。

PivotEngine 类负责摘要原始数据至 pivot 视图。

PivotPanel 控件提供通过拖拽字段至视图列表并编辑其属性的方式编辑 pivot 视图的 UI 界面。

PivotGrid 控件扩展了 FlexGrid 以显示具有可以折叠收起的行和列分组的 Pivot Table。

PivotChart 控件提供了 pivot table 的可视化展示，具有分级的坐标轴。

类

名称 描述
DetailDialog 表示一个用来显示一个表单单元格详细内容的对话框。
PivotChart

提供 olap pivot table 的可视化表示。

为使用该控件，需设置其 itemsSource 属性为一个 PivotPanel 控件或者一个 PivotEngine 实例。
PivotCollectionView 扩展 CollectionView 类，以防止小计行在排序时位置发生变化。
PivotEngine

提供一个交互式转换常规数据表至 Olap pivot table 的用户界面。

按照字段列表，以表格形式列出位于 itemsSource 集合中的数据并且 创建包含汇总数据的 pivotView 集合。

Pivot table 将数据分组成一个或多个维度。不同的维度由表格中的行和列表示，数据保存在表格中的单元格中。
PivotField 表示位于 wijmo.olap 数据源中的一个项目的属性。
PivotFieldCollection 表示一个 PivotField 对象的集合。
PivotFieldEditor PivotField 对象的编辑器。
PivotFilter 表示用做在 PivotField 中选择值的一个筛选器。
PivotFilterEditor PivotFilter 对象的编辑器。
PivotGrid

扩展 FlexGrid 控件以显示 pivot table。

为使用该控件，需要将其 itemsSource 属性设置为一个 PivotPanel 控件或者一个 PivotEngine 的实例。
PivotPanel

提供一个将常规数据表交互式转换至 Olap pivot table 的用户界面。

Olap Pivot table 将数据分组成一个或多个维度。不同的维度由表格中的行和列表示，汇总数据保存在表格中的单元格中。

使用 itemsSource 属性设置源数据，同时使用 pivotView 属性获取包含汇总数据的输出表。
ProgressEventArgs 为 progress 事件提供参数。

枚举

名称 描述
