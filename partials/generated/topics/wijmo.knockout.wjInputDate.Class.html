使用 KnockoutJS 绑定 InputDate 控件。

使用 wjInputDate 绑定 InputDate 控件，示例如下：

<p>Here is an InputDate control:</p>
<div data-bind="wjInputDate: {
  value: theDate,
  format: 'M/d/yyyy' }">
</div>

wjInputDate 支持 InputDate 控件的所有可读写属性及事件，以下属性支持双向绑定：

  • isDroppedDown
  • text
  • value
