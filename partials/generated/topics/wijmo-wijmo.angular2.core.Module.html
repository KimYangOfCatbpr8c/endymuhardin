模块 wijmo/wijmo.angular2.core

文件
wijmo.angular2.js

包含 wijmo 模块的 Angular 2 组件。

wijmo.angular2.core 是一个外部的 TypeScript 模块，可以通过其环境模块名称导入到您的代码中。例如：

import * as wjCore from 'wijmo/wijmo.angular2.core';
 
@Component({
    directives: [wjCore.WjTooltip],
    template: '<span [wjTooltip]="'Greeting'">Hello</span>',
    selector: 'my-cmp',
})
export class MyCmp {
}

WjTooltip

Tooltip 类的Angular 2 Directive 指令。

通过 wjTooltip Directive 指令添加工具提示元素至页面。 wjTooltip Directive 指令支持 HTML 内容，智能位置摆放以及触摸。

wjTooltip Directive 指令以添加到该工具提示应用到的元素的参数形式指定。 参数的值为文本或者包含该文本的元素的 id。例如：

<p [wjTooltip]="'#fineprint'" >
    Regular paragraph content...</p>
...
<div id="fineprint" style="display:none">
  <h3>Important Note</h3>
  <p>
    Data for the current quarter is estimated 
    by pro-rating etc.</p>
</div>
