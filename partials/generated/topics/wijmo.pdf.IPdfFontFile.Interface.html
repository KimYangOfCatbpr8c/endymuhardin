IPdfFontFile 类

实现
IPdfFontAttributes
模块
wijmo.pdf
文件
wijmo.pdf.js

表示由 registerFont 以及 registerFontAsync 方法注册的字体的设置。

属性

名称 类型 描述
F family? string 一个可选的参数，决定 TrueType Collection 或者 Datafork TrueType 字体集。
F name string 待使用的字体名称。
F source any 一个包含二进制数据或者加载字体的 URL 的 ArrayBuffer。 支持以下字体格式：TrueType (.ttf)，TrueType Collection (.ttc)， Datafork TrueType (.dfont)。
F style? string 字体的样式。取值可以为下列值之一：'normal'，'italic'，'oblique'。
F weight? string 字体粗细。取值可以为下列值之一：'normal'，'bold'，'100'，'200'， '300'，'400'，'500'，'600'，'700'，'800'，'900'。
