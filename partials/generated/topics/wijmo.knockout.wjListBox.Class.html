使用 KnockoutJS 绑定 ListBox 控件。

使用 wjListBox 绑定添加 ListBox 控件，示例如下:

<p>Here is a ListBox control:</p>
<div data-bind="wjListBox: {
  itemsSource: countries,
  selectedItem: theCountry }">
</div>

wjListBox 支持 ListBox 控件的所有可读写属性及事件。以下属性支持双向绑定：

  • selectedIndex
  • selectedItem
  • selectedValue
