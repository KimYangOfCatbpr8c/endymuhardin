AngularJS 指令应用于 FlexGrid Column 对象。

wj-flex-grid-column 指令必须包含于 WjFlexGrid 指令。 支持以下属性:

aggregate
@ Aggregate 对象，用于显示合计行和合计列。
align
@ 设置数据列的水平对齐方式。
allow-dragging
@ 指定拖拽行为。
allow-sorting
@ 是否允许排序。
allow-resizing
@ 是否重置行列尺寸
allow-merging
@ 是否允许列合并。
binding
@ 数据源绑定。
css-class
@ 渲染列的 css 样式类。
data-map
= DataMap 对象，用于编辑数据行的值。
data-type
@ DataType 枚举值，指定列存储的数据类型。
format
@ 用于显示行值的格式化字符串 (参见 Globalize)。
header
@ 列标题。
input-type
@ 列的输入类型。
is-content-html
@ 以纯文本还是HTML显示内容。
is-read-only
@ 是否只读。
is-selected
@ 是否选中。
mask
@ 编辑列时显示的掩饰字符。
max-width
@ 最大列宽。
min-width
@ 最小列宽。
name
@ 列名称。
required
@ 是否是必须列，即不能是null值。
show-drop-down
@ 是否下拉列表编辑基于 DataMap 的值。
visible
@ 列是否可见。
width
@ 列宽。
word-wrap
@ 单元格字符折行方式。

wj-flex-grid-column 指令中的任何 HTML 内容将被处理成列上的单元格模板。模板仅适用于正常的单元格，如果你想指定特殊的单元格如列头或分组标题，那么需要使用 WjFlexGridCellTemplate 指令。
