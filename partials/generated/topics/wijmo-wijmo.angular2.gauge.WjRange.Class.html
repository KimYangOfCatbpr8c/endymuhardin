WjFlexChart Range 对象对应的 Angular 2 组件。

wj-range 组件必须包含在一个 WjLinearGauge，WjRadialGauge 或者 WjBulletGraph 组件中。

使用 wj-range 组件添加 Range 对象至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjRange 组件继承自 Range 类，因此继承了其全部的属性，事件和方法。
