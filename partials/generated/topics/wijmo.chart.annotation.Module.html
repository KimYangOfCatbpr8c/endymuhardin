模块 wijmo.chart.annotation

文件
wijmo.chart.annotation.js

定义 AnnotationLayer 以及 see:FlexChart 和 FinancialChartand 上使用的各种批注。

类

名称 描述
AnnotationBase 表示 AnnotationLayer 支持的批注对象的基类。
AnnotationLayer

表示 FlexChart 以及 FinancialChart 上使用的批注层。

该批注层包含一个包含各种批注元素的集合：文本，直线，图像，矩形，等等。 为使用 AnnotationLayer，创建一个标注并将其放入该批注层的 items 属性。
Circle 表示 AnnotationLayer 的一个圆形批注。
Ellipse 表示 AnnotationLayer 的一个椭圆形批注。
Image 表示 AnnotationLayer 的一个图像批注。
Line 表示 AnnotationLayer 的一个直线批注。
Polygon 表示 AnnotationLayer 的一个多边形批注。
Rectangle 表示 AnnotationLayer 的一个矩形批注。
Shape 表示 AnnotationLayer 的一个形状批注的基类。
Square 表示 AnnotationLayer 的一个方形批注。
Text 表示 AnnotationLayer 的一个文本批注。

枚举

名称 描述
AnnotationAttachment 指定批注的附件。
AnnotationPosition 指定批注的位置。
