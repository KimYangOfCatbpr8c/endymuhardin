PdfPaths 类

模块
wijmo.pdf
文件
wijmo.pdf.js

提供方法创建 graphics path，用来进行图形绘制或者将其用做 Clip。

路径创建方法调用必须通过 stroke，fill，fillAndStroke 或者 clip 方法结束。 任何不会直接应用到路径创建/绘制/Clip的方法（改变画笔，绘制一个文本，保存图形上下文状态等等）需要禁止调用，直至该路径完成操作。 lineTo，bezierCurveTo 以及 quadraticCurveTo 方法不会开始路径操作，它们必须在 moveTo 之后进行操作。

该方法支持链式方法： 

doc.paths.moveTo(0, 0).lineTo(100, 100).stroke();

该类没有被设计用来在您的代码中实例化。

bezierCurveTo​(cp1x, cp1y, cp2x, cp2y, x, y): PdfPaths

cp1x (number): 第一个控件点的 X-轴坐标，单位是点。

绘制一个从当前点到一个新点的一条贝塞尔曲线，使用 (cp1x, cp1y) 以及 (cp2x, cp2y) 做为控制点。

fill​(brushOrColor?, rule?): PdfPaths

brushOrColor? (any): 待使用的画刷或颜色。
rule? (PdfFillRule): 待使用的填充规则。

使用指定的画刷和规则填充路径。 如果画刷未指定，则将使用默认的文档画刷（参见 setBrush 方法）。

brushOrColor 参数可以接受以下值：

  • 一个 PdfBrush 对象。
  • 为一个 Color 对象或者任何可以被 fromString 方法接受的字符串。 在这种情形下，具有指定颜色的 PdfBrush 对象将在内部被创建。



返回值 The PdfPaths object.
fillAndStroke​(brushOrColor?, penOrColor?, rule?): PdfPaths

brushOrColor? (any): 待使用的画刷或颜色。
penOrColor? (any): 待使用的画笔或颜色。
rule? (PdfFillRule): 待使用的填充规则。

使用指定的画刷和画笔以及填充规则绘制路径的填充和笔画。 如果画刷和画笔未指定，则使用默认的文档上的画刷和画笔（参见 setBrush，setPen 方法）。

brushOrColor 参数可以接受以下值：

  • 一个 PdfBrush 对象。
  • 为一个 Color 对象或者任何可以被 fromString 方法接受的字符串。 在这种情形下，具有指定颜色的 PdfBrush 对象将在内部被创建。

penOrColor 参数可以接受以下值：

  • 一个 PdfPen 对象。
  • 为一个 wijmo.Color 对象或者任何可以被 wijmo.Color.fromString 方法接受的字符串。 在这种情形下，具有指定颜色的 PdfPen 对象将在内部被创建。



返回值 The PdfPaths object.
