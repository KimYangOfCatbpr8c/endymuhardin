AutoComplete

AutoComplete 控件是一种允许调用者基于用户输入自定义下拉框显示项目的输入控件。

该控件和 ComboBox 类似，除了项目源是一个函数（itemsSourceFunction）而不是一个静态的列表。 例如，您可以在远程的数据库上，按照用户的输入查找项目。

以下示例创建了一个 AutoComplete 控件，并使用一个 ‘countries’ 数组填充数据。 AutoComplete 将根据用户输入搜索国家信息，并且缩小匹配当前用户输入的国家列表。

ComboBox

ComboBox 控件允许用户从列表中选取字符串。

该控件可以自动地按照用户输入完成当前条目的填写，同时允许用户显示一个下拉列表，显示全部可用的项目。

使用 selectedIndex 或者 text 属性以确定当前哪一个项目被选中。

isEditable 属性决定是否用户可以输入当前在列表中不存在的值。

以下示例演示创建一个 ComboBox 控件，并使用一个国家的列表填充数据。当用户进行输入时， ComboBox 将依照用户的输入，查找对应的国家。 isEditable 属性被设置为 false，因此用户被强制从列表中选择项目。

该示例也展示了如何通过一个具有子 <option;> 项目的 HTML <select;> 元素创建并生成一个 ComboBox。

Menu

Menu 控件显示一个具有一个下拉命令列表的一个文本元素，当用户通过单击或触摸点击时，调用这些命令。

Menu 控件继承自 ComboBox，因此您可以使用和操作 ComboBox 一样的方式填充数据以及 定制样式（参见 itemsSource 属性）。

Menu 控件添加了一个 itemClicked 事件，该事件在用户从菜单上选择了一个项目时触发。 该事件处理函数可以检查 Menu 以确定哪一个项目被单击。例如：

var menu = new wijmo.input.Menu(hostElement);
menu.header = 'Main Menu';
menu.itemsSource = ['option 1', 'option 2', 'option 3'];
menu.itemClicked.addHandler(function(sender, args) {
var menu = sender;
  alert('Thanks for selecting item ' + menu.selectedIndex + ' from menu ' + menu.header + '!');
});

以下示例演示您如何可以创建一个 value picker，基于命令的菜单，以及响应 itemClicked 事件的菜单。该示例中的菜单基于 HTML <select;> 以及 <option;> 元素。

MultiSelect

MultiSelect 控件允许用户从包含自定义对象或者简单字符串的下拉列表框选择多个项目。

MultiSelect 扩展自 ComboBox，继承了全部的常见属性，包括 itemsSource 和 displayMemberPath。

和 ListBox 控件类似，它具有一个 checkedMemberPath 属性，该属性用来指定用作确定某一个项目是否为复选选中状态的 属性名称。

当前复选（选中）的项目可以通过 checkedItems 属性获取。

控件的 header 是完全可定制的。默认情况下，最多显示前两个选中的项目以及总共选中的项目个数。 您可以改变所显示的项目个数（maxHeaderItems），当没有任何的项目被选中时所显示的信息（placeholder）， 以及用来显示项目个数的格式化字符串（headerFormat）。或者您可以提供一个函数，用来基于任何您的应用程序所需要 的条件生成 header 的内容（headerFormatter）。
Popup

将某个元素以弹出窗体的方式显示的类。

Popup 可能具有 owner 元素，在这种情况下，它的行为类似于一个 rich tooltip，可以基于 showTrigger 以及 hideTrigger 属性 指定的动作显示或则隐藏。

不具有 owner 元素的 Popup 的行为类似于对话框。它们相对于屏幕居中显示， 并且通过调用 show 方法显示。

关闭 Popup，可以调用 hide 方法。除此之外，Popup 内部任何可以点击的元素，只要是具有 ‘wj-hide’ class，就可以在发生单击时 隐藏该 Popup。例如，以下 Popup 可以在用户按下 OK 或 Cancel 按钮时隐藏：

<button id="btnPopup">Show Popup</button>
<wj-popup owner="#btnPopup" style="padding:12px">
  <p>按下以下按钮中的任意一个可以隐藏 Popup。</p>
  <hr/>
  <button class="wj-hide" ng-click="handleOK()">OK</button>
  <button class="wj-hide">Cancel</button>
</wj-popup>
