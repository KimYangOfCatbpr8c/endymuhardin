AngularJS 指令应用于 Range 对象。

wj-range 指令必须包含于 WjLinearGauge, WjRadialGauge 或者 WjBulletGraph 指令。 它添加 Range 对象到父指令的 'ranges' 数组属性上。 也可以使用 wj-property 特别指定属性名，用于使用父指令的其他 Range 类型属性。

示例： 

<wj-radial-gauge 
    min="0" 
    max="200" 
    step="20" 
    value="theValue" 
    is-read-only="false">
    <wj-range 
      min="0" 
      max="100" 
      color="red">
    </wj-range>
    <wj-range 
      min="100" 
      max="200" 
      color="green">
    </wj-range>
    <wj-range 
      wj-property="pointer" 
      color="blue">
    </wj-range>
</wj-radial-gauge>

wj-range 指令支持以下属性:

min
@ 范围的最小值。
max
@ 范围的最大值。
color
@ 范围显示的颜色。
thickness
@ 测量值的厚度，从0到1的值。
name
@ 范围的名称。
wj-property
@ 使用当前指令初始化的属性名称。
