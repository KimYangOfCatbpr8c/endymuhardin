模块 wijmo.grid.detail

文件
wijmo.grid.detail.js

为 FlexGrid 控件提供明细行的扩展。

类

名称 描述
DetailMergeManager

该 FlexGridDetailProvider 类使用的合并管理器类。

DetailMergeManager 将明细单元格（DetailRow 中的单元格）合并为单一的 明细单元格，该单元格横跨全部的表单列。
DetailRow 包含横跨全部表单列的单一明细单元格所在的行。
FlexGridDetailProvider

实现 FlexGrid 控件的明细行。

为了向 FlexGrid 添加明细行，需要创建一个 FlexGridDetailProvider 的实例，并 设置 createDetailCell 属性为一个函数，该函数创建用来在明细单元格中显示的元素。

例如：

// 创建一个 FlexGrid 以显示不同分类
var gridCat = new wijmo.grid.FlexGrid('#gridCat');
gridCat.itemsSource = getCategories();
// 添加明细行，以显示每一个分类下不同的产品
var detailProvider = new wijmo.grid.detail.FlexGridDetailProvider(gridCat);
detailProvider.createDetailCell = function (row) {
  var cell = document.createElement('div');
  var gridProducts = new wijmo.grid.FlexGrid(cell);
  gridProducts.itemsSource = getProducts(row.dataItem.CategoryID);
  return cell;
}

FlexGridDetailProvider 提供了一个 detailVisibilityMode 属性，该属性决定 何时显示该明细行。该属性的默认值为 ExpandSingle，这将添加一组 展开/收起图表至行头区域。

枚举

名称 描述
DetailVisibilityMode 指定何时以及如何显示明细行。
