Popup控件对应的 Angular 2 组件。

使用 wj-popup 组件添加 Popup 控件至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

弹出窗体的内容可以在 wj-popup 内进行指定，并且可以 包含任意的带有 Angular 2 绑定的 HTML 片段，组件以及 Directive 指令。

WjPopup 组件继承自 Popup 控件，因此继承了其全部的属性，事件和方法。

这个方法应该被模板化控件的构造器调用。它的职责是绑定模板部分到相应的控件成员上。

例如，下面的代码应用一个模板到InputNumber 控件的实例上。这个模板必须包含有'wj-part'属性的元素，该属性值可以被设置为'input', 'btn-inc'，和'btn-dec'。 该控件的成员'_tbx', '_btnUp', 和 '_btnDn'将被分配一些引用到这些元素上。

this.applyTemplate('wj-control wj-inputnumber', template, {
  _tbx: 'input',
  _btnUp: 'btn-inc',
  _btnDn: 'btn-dec'
}, 'input');

initialize​(options)

options (any): Object that contains the initialization data.

通过从给定的对象中拷贝属性来初始化控件。

该方法允许你使用普通的数据对象代替在代码中为每一个属性设置值来初始化控件。

例如: 

grid.initialize({
  itemsSource: myList,
  autoGenerateColumns: false,
  columns: [
    { binding: 'id', header: 'Code', width: 130 },
    { binding: 'name', header: 'Name', width: 60 } 
  ]
});
// 相当于
grid.itemsSource = myList;
grid.autoGenerateColumns = false;
// 等等.

初始化数据被应用时，它作为一种类型检查。如果初始化的对象包含不知名的属性名称或者无效的数据类型时，该方法将被抛出。

 使能够引起异步刷新的控件无效。

S invalidateAll​(e?)

e? (HTMLElement): Container element. If set to null, all Wijmo controls on the page will be invalidated.

使所有包含在HTML元素中的Wijmo控件无效。

当你的应用包含有动态的面板，且该面板能改变控件的可见性或者面积时，可以使用该方法。例如，splitters，accordions， 和tab控件通常改变控件内容元素的可见性。在这种情况下，通知包含在该元素中的控件失败时，将会引起它们适当地停止工作。

如果此情况发生，你必须处理在动态容器中合适的事件，并且调用Control 使所有的方法无效，这样包含的Wijmo控件将适当地更新它们的布局信息。

show​(modal?, handleResult?)

modal? (boolean): 是否将 popup 显示为一个模态对话框。如果提供了该参数，将 设置 modal 属性的值。
handleResult? (Function):

Callback invoked when the popup is hidden. If provided, this should be a function that receives the popup as a parameter.

The handleResult callback allows callers to handle the result of modal dialogs without attaching handlers to the hidden event. For example, the code below shows a dialog used to edit the current item in a CollectionView. The edits are committed or canceled depending on the dialogResult value. For example:

$scope.editCurrentItem = function () {
  $scope.data.editItem($scope.data.currentItem);
  $scope.itemEditor.show(true, function (e) {
    if (e.dialogResult == 'wj-hide-ok') {
      $scope.data.commitEdit();
    } else {
      $scope.data.cancelEdit();
    }
  });
}
