Binding 类

模块
wijmo
文件
wijmo.js

提供绑定到复杂的属性（例如'customer.address.city'）

构造函数

名称 参数 描述
constructor​(path): Binding

path (string): Name of the property to bind to.

 初始化类 Binding 的一个新实例。

属性

名称 类型 描述
path string

获取或者设置绑定的路径。

在最简单的例子中，该路径是用于绑定源对象的属性名称（例如 'street'）。

一个属性的子属性可以被指定为一个在JavaScript中相似的语法（例如 'address.street'）。

方法

名称 参数 描述
getValue​(object): any

object (any): 包含要检索的数据的对象。

获取一个给定对象的绑定值。

如果该对象不包含绑定path 指定的属性，该方法返回null。
setValue​(object, value)

object (any): 包含被设置数据的对象。
value (any): 设置数据值。

为一个给定的对象设置绑定值。

如果该对象不包含绑定path 指定的属性，不用设置该值。
