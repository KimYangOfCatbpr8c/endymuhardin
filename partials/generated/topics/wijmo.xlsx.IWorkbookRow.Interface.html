IWorkbookRow 类

模块
wijmo.xlsx
文件
wijmo.xlsx.js

表示 Workbook 对象模型的行定义。

属性

名称 类型 描述
F cells? IWorkbookCell[]

获取或设置该行中单元格的数组。

数组中的每一个 IWorkbookCell 对象描述了位于一行指定位置处的一个单元格， 比如说，位于索引 0 处的单元格 属于具有索引值为 A 的列，具有索引值 1 的单元格属于 Excel 中具有索引值为 B 的列，以此类推。如果一个特定的单元格 在 xlsx 文件中没有定义（empty），则在导入导出操作时相关联的数组元素为 undefined。
F collapsed? boolean TBD: 表示该行为收起 outline 状态。
F groupLevel? number 获取或设置行的分组级别。
F height? number

获取或设置设备无关像素点（1/96英寸）表示的行高。

如果高度未指定，则应用默认的行高。
F style? IWorkbookStyle

获取或设置行的样式。

该属性定义的样式可以应用到该行中全部的单元格，也可以被特定的单元格样式覆盖。
F visible? boolean 获取或设置行的可见性。
