用于 FinancialSeries 的AngularJs指令。

wj-financial-chart-series 指令可能包含在 WjFinancialChart 指令中。. 它支持下面的特性：

axis-x
@系列的X轴。
axis-y
@系列的X轴。
binding
@ 包含系列Y轴值的属性名。 这个值覆盖任何绑定在图表上的设置。
binding-x
@ 包含系列X轴值的属性名。 这个值覆盖任何绑定在图表上的设置。
chart-type
@ 用于为这个系列绘制的图表类型。这个值覆盖任何绑定在图表上的设置。参见 FinancialChartType.
css-class
@ 用于这个系列的CSS类。
items-source
= 包含这几个系列数据的数组或者ICollectionView 对象。
name
@ 展示在图例上的系列名称。
style
= 系列样式。 使用ng-attr-style 指令系列样式对象。参见 ngAttr 属性绑定 The series style. Use ng-attr-style to specify the series style object as an object. AngularJS 创建自定义指令。 and the FlexChart 101 Styling Series
altStyle
= 系列可替代风格。
symbol-marker
@ 这个系列使用的标记形状，这个值覆盖了设置在图表上的默认值。
symbol-size
@ 用于 Scatter, LineSymbols, 或 SplineSymbols 图表的数据点符号的尺寸，单位是像素。 这个值覆盖了图标级别上的设置。
symbol-style
=

用于 Scatter, LineSymbols, 或 SplineSymbols 图表的数据点符号的样式，单位是像素。 这个值覆盖了图标级别上的设置。

visibility
= SeriesVisibility 指定是否和在哪里展示系列。
在大多数情况下， wj-financial-chart-series 仅指定 name 和 binding 属性。 保留的值是集成自父 wj-financial-chart 指令。
