Size 类

模块
wijmo
文件
wijmo.js

类代表了一个尺寸(宽和高)。

构造函数

名称 参数 描述
constructor​(width?, height?): Size

width? (number): 新 Size 的宽度。
(默认值: 0)
height? (number): 新 Size 的高度。
(默认值: 0)

 初始化类 Size 的一个新实例。

属性

名称 类型 描述
F height number 获取或者设置该尺寸 的高。
F width number 获取或者设置该尺寸 的宽。

方法

名称 参数 描述
clone​(): Size 创建该尺寸 的一个副本。
equals​(sz): boolean

sz (Size): 尺寸 比较该 尺寸。

 如果一个尺寸 与该尺寸 有同样的面积，则返回true。
