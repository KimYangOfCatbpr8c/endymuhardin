IWorkbookColumn 类

模块
wijmo.xlsx
文件
wijmo.xlsx.js

表示 Workbook 对象模型的列定义。

属性

名称 类型 描述
F autoWidth? boolean 获取或设置一个值，该值指示是否列宽自动调整以适应其单元格内容。
F style? IWorkbookStyle

获取或设置列的样式。

该属性定义的样式可以应用到该列中全部的单元格，也可以被特定的单元格样式覆盖。
F visible? boolean 获取或设置列的可见性。
F width? any

获取或设置以设备无关像素（1/96英寸）或者字符数表示的列的的宽度。

数值类型的值定义以像素为单位的宽度。导入时，宽度始终以像素数表示。

一个带有‘ch’后缀的数值组成的字符串，比如说‘10ch’，定义以字符个数表示的宽度。 这和通过 Excel UI 定义的列宽具有相同的含义。仅当导出操作时，宽度可以被指定为字符数。

如果宽度未指定，则应用默认的列宽。
