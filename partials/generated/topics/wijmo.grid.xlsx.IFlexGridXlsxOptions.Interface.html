模块
wijmo.grid.xlsx
文件

FlexGrid Xlsx 转换选项

属性

名称 类型 描述
F activeWorksheet? any 在 xlsx 文件中，活动表单的索引或名称。
F includeCellStyles? boolean 表示是否在生成的 xlsx 文件中包含单元格样式。
F includeColumnHeaders? boolean 表示是否在生成的 xlsx 文件的第一行包含列头。
F includeRowHeaders? boolean 表示是否在生成的 xlsx 文件的第一行包含列头。
F sheetIndex? number 工作簿中表单的索引。
F sheetName? string 表单 Sheet 的名称。
F sheetVisible? boolean 表单的可见性。
