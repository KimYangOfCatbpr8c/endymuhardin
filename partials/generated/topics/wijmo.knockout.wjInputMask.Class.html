使用 KnockoutJS 绑定 InputMask 控件。

使用 wjInputMask 绑定添加 InputMask 控件，示例如下：

<p>Here is an InputMask control:</p>
<div data-bind="wjInputMask: {
  mask: '99/99/99',
  promptChar: '*' }">
</div>

wjInputMask 支持 InputMask 控件的所有可读写属性及事件。value 属性提供了双向绑定。
