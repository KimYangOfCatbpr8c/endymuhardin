模块 wijmo.grid.filter

文件
wijmo.grid.filter.js

为 FlexGrid 控件提供 Excel 样式的筛选器界面的扩展包。

类

名称 描述
ColumnFilter

定义一个 FlexGrid 控件上某一列的筛选器。

ColumnFilter 包含一个 ConditionFilter 以及一个 ValueFilte； 同一时刻只能有一个处于活动状态。

该类被 FlexGridFilter 类所使用；您一般不会直接使用。
ColumnFilterEditor

用来检查和修改列筛选器的编辑器。

该类被 FlexGridFilter 类所使用；您将几乎不会直接使用到该类。
ConditionFilter

为 FlexGrid 上的列定义一个条件筛选器。

条件筛选器包含两个条件，这两个条件可以通过 ‘and’ 或者一个 ‘or’ 操作符连接在一起。

该类被 FlexGridFilter 类所使用；您将很少有机会直接使用到。
ConditionFilterEditor

该编辑器用作检查并修改 ConditionFilter 对象。

该类被 FlexGridFilter 类所使用；您将很少可能会直接使用到。
FilterCondition

定义一个筛选器条件。

该类被 FlexGridFilter 类所使用；您将几乎不会直接使用。
FlexGridFilter

为 FlexGrid 控件实现一个 Excel 样式的筛选器。

为了在 FlexGrid 控件上启用筛选，需要创建一个 FlexGridFilter 的实例， 并将 grid 作为一个参数传递给构造器。例如：

// 创建 FlexGrid
var flex = new wijmo.grid.FlexGrid('#gridElement');
// 在 FlexGrid 上启用筛选
var filter = new wijmo.grid.filter.FlexGridFilter(flex);

一旦这一过程完成，则表单的列头上将添加一个筛选器图表。 单击该图表将显示一个编辑器，用户可以在此编辑该列的筛选 条件。

该 FlexGridFilter 类依赖 wijmo.grid 以及 wijmo.input 模块。
ValueFilter

定义一个 FlexGrid 控件上某一列的值筛选器。

值筛选器包含一个明确的值列表，由表单显示。
ValueFilterEditor

用来检查并修改 ValueFilter 对象的编辑器。

该类被 FlexGridFilter 类所使用；您将很少有机会直接用到该接口。

定义一个 FlexGrid 控件上某一列的筛选器。

该类被 FlexGridFilter 类所使用；您将很少有机会直接使用到它。
