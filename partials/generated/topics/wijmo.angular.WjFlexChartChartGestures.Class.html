AngularJS 指令应用于 FlexChart ChartGestures 对象。

wj-flex-chart-gestures 指令必须包含于 WjFlexChart 指令或者 WjFinancialChart 指令。 支持以下属性:

mouse-action
@ 鼠标操作是缩放或者移动。
interactive-axes
@ 轴是否交互。
enable
@ 是否支持手势操作。
scale-x
@ x 轴的最大最小区间。
scale-y
@ y 轴的最大最小区间。
pos-x
@ x 轴初始位置。
pos-y
@ y 轴初始位置。
