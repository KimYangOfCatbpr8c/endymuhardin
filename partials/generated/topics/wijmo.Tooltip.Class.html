提供一个弹出窗口，该窗口显示页面上的其他元素的附加信息。

Tooltip 类可以在两个模式中使用：

Automatic 模式： 使用setTooltip 方法连接Tooltip 到页面上的一个或者更多个元素。 Tooltip 自动监听事件，并且当用户执行操作时触发提示信息。例如：

var tt = new wijmo.Tooltip();
tt.setTooltip('#menu', 'Select commands.');
tt.setTooltip('#tree', 'Explore the hierarchy.');
tt.setTooltip('#chart', '#idChartTooltip');

Manual 模式： 调用者使用show 和hide 方法负责显示和隐藏工具提示。例如：

var tt = new wijmo.Tooltip();
element.addEventListener('click', function () {
  if (tt.isVisible) {
    tt.hide();
  } else {
    tt.show(element, 'This is an important element!');
  }
});

构造函数

constructor​(options?): Tooltip

options? (any): JavaScript object containing initialization data for the Tooltip.

 初始化类 Tooltip 的一个新实例。

属性

名称 类型 描述
gap number 获取或设置工具提示和目标元素之间的距离。
hideDelay number 获取或设置延时，以毫秒为单位，在隐藏工具提示鼠标离开目标元素后。
isContentHtml boolean 获取或设置一个值，决定是否提示内容应该显示为纯文本或HTML。
isVisible boolean 获取提示信息当前是否可见。
showAtMouse boolean 获取或设置一个值，决定是否提示应该定位相对于鼠标的位置而不是目标元素。
showDelay number 获取或设置延时，以毫秒为单位，在鼠标进入目标元素后显示的工具提示。

方法

名称 参数 描述
hide​() 如果提示信息当前可见，则隐藏它。
onPopup​(e)

e (TooltipEventArgs): TooltipEventArgs 包含事件数据。

 引发popup 事件。
setTooltip​(element, content)

element (any): 提示信息解释元素，元素id，或者控件。
content (string): 元素的提示信息内容或者id包含提示信息内容。

指定工具提示内容到一个特定的页面上的元素。

同样的提示可以显示任意数量的元素在页面中的信息。为了从元素上移除提示信息， 调用setTooltip 并且制定内容为null。
show​(element, content, bounds?)

element (any): 提示信息解释元素，元素id，或者控件。
content (string): 元素的提示信息内容或者id包含提示信息内容
bounds? (Rect): 可选的元素定义提示目标的范围。如果没有被提供，元素的边界被使用（由 getBoundingClientRect方法报告）。

 显示指定内容的提示，紧挨着指定的元素。

事件

名称 参数 描述
popup TooltipEventArgs

提示信息被显示之前发生。

事件处理程序可以自定义提示内容或改变的事件参数压制工具提示显示。
