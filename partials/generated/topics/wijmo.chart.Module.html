模块 wijmo.chart

文件
wijmo.chart.js

定义 FlexChart 控件及其相关类。

以下示例创建了一个 FlexChart 并将其绑定至一个数据数组。 该图表具有三个系列，每一个系列关联到包含在数据源数组中对象的一个属性。 该示例中的最后一个系列通过 chartType 属性 覆盖了其他系列使用的默认图表类型。

FlexChart

FlexChart 提供了一个强大且灵活的方式可视化显示数据。

您可以使用 FlexChart 控件创建以不同格式显示数据的图表，包括条形图，折线图，符号，气泡图以及其他。

为使用 FlexChart 控件，须设置 itemsSource 属性为一个包含数据的数组，接下来向 series 属性添加一个或更多的 Series 对象。

使用 chartType 属性定义全部系列使用的 ChartType。 您可以通过设置每一个 Series 对象上的 chartType 属性覆盖该默认设置的图表类型。
FlexChartBase FlexChart 以及 FlexPie 派生自的基类 FlexChartBase 控件。
FlexChartCore FlexChart 核心的图表控件。
FlexPie

FlexPie 控件提供了切片可选中的饼图和环形图。

使用 FlexPie 控件，设置 itemsSource 属性为一个包含数据的数组， 并使用 binding 以及 bindingName 属性设置包含项目值和名称的属性。
HitTestInfo 包含指定的页面坐标位置处，FlexChart 的某个部分的信息。
Legend 表示图表图例。
LineMarker

表示 FlexChart 的 LineMarker 的一个扩展。

LineMarker 由一个显示数据点的值的一个文本区域，一个可选的水平或垂直线（或者两者同时显示，以达到十字线效果）组成，位于绘图区域上方。 它可以是静态的（interaction = None)，或者是跟随鼠标或者触摸点（interaction = move），或者当用户拖拽线形时移动（interaction = Drag）。 例如： 

  // 创建一个可交互的标记，显示水平线和 Y 值
  var lm = new wijmo.chart.LineMarker($scope.ctx.chart, {
      lines: wijmo.chart.LineMarkerLines.Horizontal,
      interaction: wijmo.chart.LineMarkerInteraction.Move,
      alignment : wijmo.chart.LineMarkerAlignment.Top
  });
  lm.content = function (ht) {
      // 显示 Y 值
      return lm.y.toFixed(2);
  }

Palettes

以下为图表 Series 对象准备的预定义的调色板。

为创建自定义的调色板，可以指定一组字符串或者 rgba 值的数组。

您可以为 FlexChart 以及 FlexPie 控件指定调色板。 例如：

chart.palette = Palettes.light;

有以下预定义调色板：

  • standard (default)
  • cocoa
  • coral
  • dark
  • highcontrast
  • light
  • midnight
  • minimal
  • modern
  • organic
  • slate
PieDataLabel FlexPie 的点数据标签。
PlotArea

表示图表中的一个绘图区域。

图表可以具有多个绘图区域以及多组坐标轴。 通过 Axis.plotArea 属性向绘图区添加坐标轴。例如： 

 // 创建一个绘图区
 var pa = new wijmo.chart.PlotArea();
 pa.row = 1;
 chart.plotAreas.push(pa);
 // 创建辅助 Y 轴
 var ay2 = new wijmo.chart.Axis(wijmo.chart.Position.Left);
 ay2.plotArea = pa; // 将坐标轴关联到绘图区
 chart.axes.push(ay2);
 // 沿 Y 轴绘制第一个系列
 chart.series[0].axisY = ay2;

PlotAreaCollection 表示 FlexChartCore 中的 PlotArea 对象集合。
RenderEventArgs 为 Series 事件提供事件参数。
Series

表示显示在图表中的一系列数据点。

Series 类支持全部的基本图表类型。您可以为添加到 FlexChart 系列集合中的每一个 Series 对象定义一个不同的图表类型。这将覆盖该集合中为全部 Series 对象设置的默认的 chartType 属性。
SeriesBase 表示显示在图表中的一系列数据点。
SeriesEventArgs 为 Series 事件提供事件参数。
