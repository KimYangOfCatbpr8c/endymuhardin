AngularJS 指令应用于 Menu 控件.

使用 wj-menu 指令添加 drop-down menus 到 AngularJS 应用中。 注意指令和参数名称必须使用小写字母+短线语法替代驼峰命名。 示例：

<p>Here is a Menu control used as a value picker:</p>
<wj-menu header="Tax" value="tax">
  <wj-menu-item value="0">Exempt</wj-menu-item>
  <wj-menu-item value=".05">5%</wj-menu-item>
  <wj-menu-item value=".1">10%</wj-menu-item>
  <wj-menu-item value=".15">15%</wj-menu-item>
</wj-menu>

下面的例子创建了3个 Menu 控件。第一个作为数据采集器，第二个使用了包含参数的命令列表，第三个绑定了控制器上的 itemClicked 事件处理函数。

wj-menu 指令扩展了 WjComboBox ，提供以下特性：

command-path
@ 项目被点击时执行的命令。
command-parameter-path
@ 命令参数名。
header
@ 控件显示文本。
is-button
@ 是否响应控件标题区域的点击事件。
value
@ 选中的 wj-menu-item 值。
item-clicked
& itemClicked 事件。
got-focus
& gotFocus 事件。
lost-focus
& lostFocus 事件。

wj-menu 指令可能包含以下子指令: WjMenuItem, WjMenuSeparator 和 WjItemTemplate(任意菜单绑定控件).
