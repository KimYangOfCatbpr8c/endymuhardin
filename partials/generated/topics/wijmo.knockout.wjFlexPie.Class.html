使用 KnockoutJS 绑定 FlexPie 控件。

使用 wjFlexPie 绑定 FlexPie 控件，示例如下：

<p>Here is a FlexPie control:</p>
<div data-bind="wjFlexPie: {
        itemsSource: data,
        binding: 'value',
        bindingName: 'name',
        header: 'Fruit By Value' }">
    <div data-bind="wjFlexChartLegend : { position: 'Top' }"></div>
</div>

wjFlexPie 绑定可以包含 wjFlexChartLegend 自绑定。

wjFlexPie 支持 FlexPie 控件的所有可读写属性及事件
