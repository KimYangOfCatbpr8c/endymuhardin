RowColFlags 枚举

模块
wijmo.grid
文件
wijmo.grid.js

指定一个表单行或列的状态的 Flag。

值

名称 描述
AllowDragging 该行或列可以通过鼠标拖放至一个新的位置。
AllowMerging 该行或列可以包含合并的单元格。
AllowResizing 该行或列可以被改变尺寸。
AllowSorting 该列可以通过鼠标单击列头进行排序。
AutoGenerated 该列为自动生成。
Collapsed 该分组行为收起状态。
ColumnDefault 新列的默认设置。
HtmlContent 该行或列中的单元格包含 HTML 格式的文本。
ParentCollapsed 该行具有一个收起状态的父分组。
ReadOnly 该行或列为只读（无法被编辑）。
RowDefault 新行的默认设置。
Selected 该行或列为被选中状态。
Visible 该行或列为可见。
WordWrap 位于该行和列的单元格可能包含可折行的文本。
