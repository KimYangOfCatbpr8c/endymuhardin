模块 wijmo.chart.interaction

文件
wijmo.chart.interaction.js

定义向图表添加交互功能的相关类型。

类

名称 描述
ChartGestures

ChartGestures 控件允许用户在通过 Zoom 或者 Pan 操作缩放指定的 FlexChart。

为使用 ChartGestures 控件，指定需要 zoom 或 pan 操作的 FlexChart。

 var chartGestures = new wijmo.chart.interaction.ChartGestures(chart);

RangeSelector

RangeSelector 控件显示了一个范围选择器，允许用户选择在指定的 FlexChart 上显示的数据范围。

为使用 RangeSelector 控件，必须指定显示选中数据范围的 FlexChart 控件。

当最小值或者最大值发生变化时，将触发 rangeChanged 事件。 例如： 

 var rangeSelector = new wijmo.chart.interaction.RangeSelector(chart);
 rangeSelector.rangeChanged.addHandler(function () {
    // 执行相关的更新
    // 比方说，更新其他图表的显示范围
    update(rangeSelector.min, rangeSelector.max);
  });

枚举

名称 描述
InteractiveAxes 指定图表手势的交互坐标轴。
MouseAction 指定图表手势的鼠标动作。
Orientation 指定范围选择器的方向。
