名称 参数 描述

g (FlexGrid): 拥有该集合的 FlexGrid 对象。
defaultSize (number): 集合中元素的默认尺寸。

 初始化类 RowColCollection 的一个新实例

属性

名称 类型 描述
defaultSize number 获取或设置集合中元素的默认尺寸。
frozen number

获取或设置集合中冻结行或者冻结列的个数。

冻结的行或者列不会滚动，会保持在表单的顶部或者左侧，紧邻着固定单元格。 和固定单元格不同，冻结的单元格可以像普通单元格一样被选择和编辑。
isUpdating 获取一个值，该值指示当前是否暂停通知（参见beginUpdate 和endUpdate）。
(继承自 ObservableArray).
maxSize number 获取或设置集合中元素的最大尺寸。
minSize number 获取或设置集合中元素的最小尺寸。
