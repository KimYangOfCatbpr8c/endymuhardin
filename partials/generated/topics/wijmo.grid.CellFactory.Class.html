创建表示在一个 FlexGrid 控件内部单元格的 HTML 元素。

方法

名称 参数 描述
disposeCell​(cell)

cell (HTMLElement): 表示该单元格的元素。

 销毁一个单元格元素并释放全部关联的资源。
updateCell​(p, r, c, cell, rng?, updateContent?)

p (GridPanel): 包含该单元格的 GridPanel。
r (number): 包含该单元格的行的索引。
c (number): 包含该单元格的列的索引。
cell (HTMLElement): 表示该单元格的元素。
rng? (CellRange): 包含单元格合并范围的 CellRange 对象，或者当单元格没有合并时返回 null。
updateContent? (boolean): 是否更新单元格的内容以及其位置和样式。

 创建或更新表单中的一个单元格。
