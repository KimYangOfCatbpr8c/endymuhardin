AngularJS 指令用于FlexRadar FlexRadarAxis Axis。 wj-flex-radar-axis 指令必须包含在 WjFlexRadar directive 指令中。 支持下面的特性：

wj-property
@ 定义 FlexChart属性名，axis-x 或 axis-y,使用指令初始化。.

axis-line
@ 这个值指定坐标轴是否可见。
binding
@ 为 itemsSource 属性获取或设置以逗号分割的属性名以用于坐标轴标签，第一个名称指定坐标轴上的值，第二只值代表对in个的坐标轴标签，默认值为：‘value，text’。
format
@ 用于坐标轴表亲啊的格式化字符串。(参见 Globalize).
item-formatter
= 格式化函数自定义坐标轴标签的外观。
items-source
= 用于坐标轴标签的数据源。
labels
@ 指示坐标轴标签是否可见。
label-angle
@ 坐标轴标签的旋转角度。
label-align
@ 坐标轴标签的对齐方式。
label-padding
@ 坐标轴标签的填充。
major-grid
@ 决定坐标轴是否包含表格线。
major-tick-marks
@ 定义坐标轴刻度线的外观。 see TickMark.
major-unit
@ 坐标轴标间的单位数量。
max
@ 坐标轴上显示的最小值。
min
@ 坐标轴标签上的最大值。
minor-grid
@ 决定坐标值是否包含次网格线。
minor-tick-marks
@ 定义坐标轴上次要刻度线的外观。 (see TickMark).
minor-unit
@ 次要坐标轴间的单位数量。

origin
@ 坐标原点。
overlappingLabels
@ OverlappingLabels 指示是否处理重叠轴标签。
position
@ 指定坐标轴的位置。
title
@ 坐标轴旁的标题文本。
