FlexSheetPanel 类

基类
GridPanel
模块
wijmo.grid.sheet
文件
wijmo.grid.sheet.js
表示

定义 GridPanel 类的扩展，该类由 FlexSheet 使用，而 t其基类 FlexGrid 类则使用 GridPanel。例如， cells 属性将返回该类型的一个实例。

构造函数

名称 参数 描述
constructor​(grid, cellType, rows, cols, element): FlexSheetPanel

grid (FlexGrid): 拥有该面板的 FlexGrid 对象。
cellType (CellType): 该面板中单元格的类型。
rows (RowCollection): 显示在该面板中的行。
cols (ColumnCollection): 显示在该面板中的列。
element (HTMLElement): 托管控件中单元格的 HTMLElement。

 初始化 FlexSheetPanel 类的一个新实例

属性

名称 类型 描述
cellType CellType 获取包含在该面板中的单元格类型。
(继承自 GridPanel).
columns ColumnCollection 获取列集合的面板。
(继承自 GridPanel).
grid FlexGrid 获取拥有该面板的表单。
(继承自 GridPanel).
height number 获取面板中的内容的总高度。
(继承自 GridPanel).
hostElement HTMLElement 获取面板的托管容器元素。
(继承自 GridPanel).
rows RowCollection 获取行集合的面板。
(继承自 GridPanel).
viewRange CellRange 获取表示当前在面板上可见的单元格范围的 CellRange。
(继承自 GridPanel).
width number 获取面板中的内容的总宽度。
(继承自 GridPanel).

方法

名称 参数 描述
getCellBoundingRect​(r, c, raw?): Rect

r (number): 包含该单元格的行的索引。
c (number): 包含该单元格的列的索引。
raw? (boolean): 是否返回的矩形框是按照原始的面板坐标系而不是 viewport 坐标系。

获取位于 viewport 坐标系的单元格范围。

返回的值为一个包含位于 viewport 坐标系的单元格的位置和尺寸的 Rect 对象。 viewport 坐标系和 getBoundingClientRect 方法使用的相同。


(继承自 GridPanel).
getCellData​(r, c, formatted): any

r (number): 该单元格的行索引。
c (any): 包含该单元格的列的索引，名称或者绑定。
formatted (boolean): 是否格式化该值用于显示目的。

 获取保存在面板中单元格中间的值。
(继承自 GridPanel).
getSelectedState​(r, c, rng): SelectedState

r (number): 指定单元格的行索引。
c (number): 指定单元格的列索引。
rng (CellRange): 包括应当被包含的单元格的 CellRange 单元格范围。

获取一个 SelectedState 值，该值表示单元格的选中状态。

覆盖此方法以支持 FlexSheet 上的 showSelectedHeaders 的多选。
setCellData​(r, c, value, coerce?): boolean

r (number): 包含该单元格的行的索引。
c (any): 包含该单元格的列的索引，名称或者绑定。
value (any): 待存储在单元格中的值。
coerce? (boolean): A value indicating 是否自动地改变值以匹配该列的数据类型。
(默认值: true)

 设置面板中单元格的内容。

返回值 如果该值成功存储则返回 true，否则返回 false（转换失败）。
