使用 KnockoutJS 绑定 ColorPicker 控件。

使用 wjColorPicker 绑定添加 ColorPicker 控件，示例如下：

<p>Here is a ColorPicker control:</p>
<div 
  data-bind="wjColorPicker: { value: theColor }">
</div>

wjColorPicker 支持 ColorPicker 控件的所有可读写属性及事件。以下属性支持双向绑定：

  • value
