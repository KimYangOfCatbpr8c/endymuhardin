options any

获取或设置各种图表选项。

支持以下选项：

plotMargin any

获取或设置绘图区的边距，单位是像素。

绘图区的边距，表示控件的边框和绘图区之间的区域。

默认情况下，该值基于坐标轴标签所需的区域自动计算，但是如果您希望控制绘图区在控件内的精确位置，您可以覆盖该值（例如，当将多个图表在一页中进行布局）。

您可以设置该属性为一个数值类型的值或者一个通过 CSS-样式指定的边距。例如：

// 将绘图区的边距四边均设置为 20 个像素
chart.plotMargin = 20;
// 分别设置绘图区域边距的顶部，右侧，底部和左侧。
chart.plotMargin = '10 15 20 25';
// 分别设置绘图区域边距的顶部/底部为 10 像素，左侧/右侧为 20 像素。
chart.plotMargin = '10 20';

tooltip ChartTooltip

获取图表的 Tooltip 对象。

工具提示内容通过一个模板生成，其中可以包含以下任意参数：

  • propertyName: 由该点所表示的数据对象的任意属性。
  • seriesName: 包含该数据点的系列名称（仅 FlexChart）。
  • pointIndex: 数据点的索引。
  • value: 数据点的值（对于 FlexChart 而言是 y-值，而对于 FlexPie 而言是项目值）。
  • x: 数据点的 x-值（仅 FlexChart）。
  • y: 数据点的 y-值（仅 FlexChart）。
  • name: 数据点的名称（对于 FlexChart 而言是 x-值，而对于 FlexPie 而言是图例项）。

为修改模板，请分配一个新值给工具提示的 content 属性。 例如：

chart.tooltip.content = '<b>{seriesName}</b> ' +
   '<img src="resources/{x}.png"/><br/>{y}';

通过设置该模板为一个空白字符串可以禁用图表工具提示。

您同样也可以使用 tooltip 属性自定义工具提示的属性，比如说 showDelay 以及 hideDelay：

chart.tooltip.showDelay = 1000;

更多细节和不同选项的说明，请参见 ChartTooltip 属性。

initialize​(options)

options (any): Object that contains the initialization data.

通过从给定的对象中拷贝属性来初始化控件。

该方法允许你使用普通的数据对象代替在代码中为每一个属性设置值来初始化控件。

例如: 

grid.initialize({
  itemsSource: myList,
  autoGenerateColumns: false,
  columns: [
    { binding: 'id', header: 'Code', width: 130 },
    { binding: 'name', header: 'Name', width: 60 } 
  ]
});
// 相当于
grid.itemsSource = myList;
grid.autoGenerateColumns = false;
// 等等.

初始化数据被应用时，它作为一种类型检查。如果初始化的对象包含不知名的属性名称或者无效的数据类型时，该方法将被抛出。

S invalidateAll​(e?)

e? (HTMLElement): Container element. If set to null, all Wijmo controls on the page will be invalidated.

使所有包含在HTML元素中的Wijmo控件无效。

当你的应用包含有动态的面板，且该面板能改变控件的可见性或者面积时，可以使用该方法。例如，splitters，accordions， 和tab控件通常改变控件内容元素的可见性。在这种情况下，通知包含在该元素中的控件失败时，将会引起它们适当地停止工作。

如果此情况发生，你必须处理在动态容器中合适的事件，并且调用Control 使所有的方法无效，这样包含的Wijmo控件将适当地更新它们的布局信息。
