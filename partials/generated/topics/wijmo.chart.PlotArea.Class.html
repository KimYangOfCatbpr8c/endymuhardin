PlotArea 类

派生类
WjFlexChartPlotArea
模块
wijmo.chart
文件
wijmo.chart.js

表示图表中的一个绘图区域。

图表可以具有多个绘图区域以及多组坐标轴。 通过 Axis.plotArea 属性向绘图区添加坐标轴。例如： 

 // 创建一个绘图区
 var pa = new wijmo.chart.PlotArea();
 pa.row = 1;
 chart.plotAreas.push(pa);
 // 创建辅助 Y 轴
 var ay2 = new wijmo.chart.Axis(wijmo.chart.Position.Left);
 ay2.plotArea = pa; // 将坐标轴关联到绘图区
 chart.axes.push(ay2);
 // 沿 Y 轴绘制第一个系列
 chart.series[0].axisY = ay2;

属性

名称 类型 描述
column number 获取或设置绘图区域的列号。 通过 column 属性，您可以设置绘图区在图表中的垂直位置。
height any

获取或设置绘图区的高度。

高度可以指定为一个数值（按像素数指定宽度）或者具有'{number}*'格式的字符串（星号尺寸标记）。
name string 获取或设置绘图区域的名称。
row number 获取或设置绘图区域的行号。 通过 row 属性，您可以设置绘图区在图表中的水平位置。
style any

获取或设置绘图区域的样式。

通过 style 属性，您可以设置绘图区的外观。 例如： 

  pa.style = { fill: 'rgba(0,255,0,0.1)' };

width any

获取或设置绘图区的宽度。

宽度可以指定为一个数值（按像素数指定宽度）或者具有'{number}*'格式的字符串（星号尺寸标记）。
