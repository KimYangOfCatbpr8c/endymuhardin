AngularJS 指令应用于 InputDate 控件。

使用 wj-input-date 指令添加 InputDate 到 AngularJS 应用中。 注意指令和参数名称必须使用小写字母+短线语法替代驼峰命名。 示例：

<p>Here is an InputDate control:</p>
<wj-input-date 
  value="theDate" 
  format="M/d/yyyy">
</wj-input-date>

下面的例子使用 InputDate 和 an InputTime 控件呈现 Date 值 (包含日期和时间信息)，需要注意两个控件是怎样绑定到同一个控制器属性变量上的，并每个编辑数据的不同部分。这个例子同时展示了 Calendar 控件可以通过单击选择日期。

wj-input-date 指令支持以下属性:

ng-model
@ 使用Angular的ng-model指令绑定控件的 value 属性。使用ng-model指令绑定属性能够获得一些共通的特性，例如验证，自动添加控件状态到form实例，等等。 使用 wj-model-property 重定义被 ng-model指定绑定的控件属性。
wj-model-property
@ 使用 ng-model 指令指定控件绑定的作用域。
control
= 该指令创建的指向 InputDate 的引用
format
@ 日期编辑时的格式化字符串 (参见 Globalize).
mask
@ 用于验证用户输入的蒙版 (参见 InputMask).
is-dropped-down
@ 指示下拉当前是否可见。
initialized
& 该事件在绑定完成控件特性值的初始化后被触发。
is-initialized
= 绑定是否已经完成控件特性值的初始化。
max
@ 最大有效日期 (格式化字符串为 "yyyy-MM-dd")。
min
@ 最小有效日期 (格式化字符串为 "yyyy-MM-dd")。
place-holder
@ 描述输入字段预期值的提示信息。
required
@ 是否阻止null值。
show-drop-down-button
@ 是否显示下拉按钮。
text
= 控件显示文本。
value
= 编辑的日期至。
got-focus
& gotFocus 事件。
lost-focus
& lostFocus 事件。
is-dropped-down-changing
& isDroppedDownChanging 事件。
is-dropped-down-changed
& isDroppedDownChanged 事件。
text-changed
& textChanged 事件。
value-changed
& valueChanged 事件。

min 和 max 属性需要指定为格式化 "yyyy-MM-dd" 的日期字符串， 技术上讲，你可以使用符合W3C [RFC 3339] 定义的任何格式化字符，只要它们是能够被HTML5支持格式化字符。
