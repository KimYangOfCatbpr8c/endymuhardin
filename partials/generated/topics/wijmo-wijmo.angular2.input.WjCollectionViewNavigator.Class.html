ICollectionView navigator元素对应的 Angular 2 组件。

通过 wj-collection-view-navigator 组件添加一个元素，允许用户遍历一个 ICollectionView 中的项目。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。例如：

<wj-collection-view-navigator [cv]="myCollectionView"> </wj-collection-view-navigator>
