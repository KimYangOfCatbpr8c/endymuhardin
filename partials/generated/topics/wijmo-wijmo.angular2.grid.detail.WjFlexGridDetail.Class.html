表示

FlexGrid DetailRow 模板的Angular 2 Directive 指令。

wj-flex-grid-detail 指令必须在一个包含在 wj-flex-grid 组件中包含的一个 <template> 模板元素。

wj-flex-grid-detail 指令继承自 FlexGridDetailProvider 类，可以管理详细行的可见性，详细行的内容可以在 directive 指令标签中定义为任意的 HTML 片段。 该片段可以包含 Angular 2 绑定，组件以及指令。 row 和 item 模板变量可以在 Angular 2 绑定中使用，分别表示详细行的父 Row 以及 Row.dataItem 对象。

属性

名称 类型 描述
createDetailCell Function

获取或设置创建明细单元格的回调函数。

该回调函数接受一个 Row 对象作为参数，并返回一个表示该行明细的 HTML 元素。 例如：

// 为指定的行创建明细单元格
dp.createDetailCell = function (row) {
  var cell = document.createElement('div');
  var detailGrid = new wijmo.grid.FlexGrid(cell, {
    itemsSource: getProducts(row.dataItem.CategoryID),
    headersVisibility: wijmo.grid.HeadersVisibility.Column
  });
  return cell;
};


(继承自 FlexGridDetailProvider).
detailVisibilityMode DetailVisibilityMode 获取或设置一个值，该值决定何时显示行明细。
(继承自 FlexGridDetailProvider).
disposeDetailCell Function

获取或设置用来销毁明细单元格的回调函数。

该回调函数接受一个 Row 对象做为参数，并将销毁任何关联到该 明细单元格的资源。

该函数为可选。当 createDetailCell 函数申请了一些无法被自动垃圾回收的资源时 需要用到该函数。


(继承自 FlexGridDetailProvider).
grid FlexGrid 获取拥有该 FlexGridDetailProvider 的 FlexGrid。
(继承自 FlexGridDetailProvider).
maxHeight number 获取或设置行明细的最大高度，单位是像素。
(继承自 FlexGridDetailProvider).
rowHasDetail Function

获取或设置一个回调函数，该函数决定是否一行具有明细。

该回调函数接受一个 Row 对象作为参数， 并返回一个布尔值，该值表示是否该行具有明细。例如：

// 移除具有偶数值的 CategoryID 的项目的明细
dp.rowHasDetail = function (row) {
  return row.dataItem.CategoryID % 2 == 0;
};

设置该属性的值为 null 表示全部的行都具有明细。


(继承自 FlexGridDetailProvider).
