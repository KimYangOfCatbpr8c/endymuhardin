ITableAddress 类

模块
wijmo.xlsx
文件
wijmo.xlsx.js

使用从零开始计算索引值的行和列部分组成的一个单元格索引，同时还包含了表示是否该索引部分是绝对（例如“$D”）的还是相对的（例如“D”）的属性。

属性

名称 类型 描述
F absCol boolean 表示是否原始的列索引是绝对表示（比如说 “$D”）还是相对表示（比如说“D”）。
F absRow boolean 表示是否原始的行索引是绝对表示（比如说 “$15”）还是相对表示（比如说“15”）。
F col number 从零开始计算的列索引。
F row number 从零开始计算的行索引。
