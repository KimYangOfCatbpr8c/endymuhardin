名称 参数 描述
constructor​(engine): PivotFieldCollection

engine (PivotEngine): 包含这个 PivotFieldCollection 的PivotEngine 。

 初始化 PivotFieldCollection 类的一个新实例。

属性

名称 类型 描述
engine PivotEngine 得到拥有这个 PivotFieldCollection 的 PivotEngine 的一个引用。
isUpdating 获取一个值，该值指示当前是否暂停通知（参见beginUpdate 和endUpdate）。
(继承自 ObservableArray).
maxItems number

获取或设置该集合中最大允许的字段数。

该属性默认设置为 null，表示接受任意数量的项目。

deferUpdate​(fn)

fn (Function): 被执行的没有更新的函数。

在beginUpdate/endUpdate 块中执行函数。

直到函数完成前集合将不会被刷新。该方法确保endUpdate 会被调用即使函数会抛出异常。


(继承自 ObservableArray).
