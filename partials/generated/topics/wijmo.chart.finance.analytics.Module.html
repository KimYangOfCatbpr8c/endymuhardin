模块 wijmo.chart.finance.analytics

文件
wijmo.chart.finance.analytics.js

FinancialChart 的分析扩展。

类

名称 描述
ATR

表示 FinancialChart 的一个真实波动幅度均值指标系列。

真实波动幅度均值用来衡量资产的波动性。真实波动幅度均值不提供价格趋势的任何指标，而是价格的 波动程度。
BollingerBands

表示 FinancialChart 的一个 Bollinger Bands® 覆盖系列。

Bollinger Bands 是一个 John Bollinger 所持有的注册商标。
CCI

表示 FinancialChart 的一个商品通道指数指标系列。

商品通道指数指标是是衡量资产的现行价格振荡 相对于平均价格水平以上的指定时间段的水平。
Envelopes

表示 FinancialChart 的一个移动平均包围覆盖系列。

移动平均包围是一种移动平均值，设置为移动平均值的上下限。 高于/低于移动平均值的范围由百分比指定，最终由 size 属性决定。
Fibonacci

表示 FinancialChart 的一个斐波那契回调线工具。 该工具允许在不同的警告级别计算和绘图，在金融图表中非常有用。

为了向 FinancialChart 控件添加一个斐波那契工具，创建一个 Fibonacci 的实例并将其添加至 图表的 series 集合。 例如：

// 创建图表
var chart = new wijmo.chart.finance.FinancialChart('#chartElement');
// 创建斐波那契工具
var ftool = new wijmo.chart.finance.analytics.Fibonacci();
chart.series.push(ftool);

FibonacciArcs 表示 FinancialChart 的一个斐波纳契弧线工具。
FibonacciFans 表示 FinancialChart 的一个斐波那契 Fans 工具。
FibonacciTimeZones 表示 FinancialChart 的一个斐波那契 Time Zones 工具。
Macd

表示 FinancialChart 的一个移动平均汇总/分离（MACD）指标系列。

MCAD指标旨在揭示资产的价格趋势的强度，方向，动量以及持续时间。
MacdBase Macd 以及 MacdHistogram 系列的基类（抽象类）。
MacdHistogram

表示 FinancialChart 的一个移动平均汇总/分离（MACD）历史的指标系列。

MCAD指标旨在揭示资产的价格趋势的强度，方向，动量以及持续时间。
OverlayIndicatorBase Overlay 以及指标系列的基类（抽象类）。
RSI

表示 FinancialChart 的一个相对强弱指数指标系列。

相对强弱指数是一个动量振荡器，被设计用来根据近期的交易时段的收盘价，衡量资产 当前以及历史的强弱。
SingleOverlayIndicatorBase 仅绘制一个系列的 Overlay 以及指标系列的基类（抽象类）。
Stochastic

表示 FinancialChart 的一个随机摆动指标系列。

随机指标旨在通过资产的收盘价相比其高低范围来预测价格拐点的动量指标。

Stochastic 系列可以用在快速（默认值），慢速以及完全随机振荡器上。 为创建一个慢速或者完全随机振荡器，请设置 smoothingPeriod 属性的值为一个大于 1 的整数值； 慢速振荡器通常是将 smoothingPeriod 设置为 3。 为了创建或者回复到快速随机震荡，设置 smoothingPeriod 属性为整数值 1。
WilliamsR

表示 FinancialChart 的一个Willaims％R指标系列。

Williams %R 指标是一种动能指标，是一个快速随机的逆振荡器（Stochastic）。 Williams %R 指标被设计为告诉一个资产的交易是否已经接近高或者低的交易区间。
