ICollectionView
模块
wijmo.collections

wijmo.js

定义扩展的ICollectionView以提供具有翻页功能的方法和属性。

属性

名称 类型 描述
F canChangePage boolean 获取一个值，该值表明pageIndex 值是否可以改变。
F isPageChanging boolean 获取一个值，该值指示该索引是否正在更改。
F itemCount number

获取视图中的项目的数目，并在账目中分页。

使用totalItemCount 属性获得的项目总数。

注意，它不同于.NET IPagedCollectionView, itemCount 和 totalItemCount 都在分页被应用之前返回数量。
F pageChanged Event 页面索引更改后发生。
F pageChanging Event 在页面索引更改之前发生。
F pageIndex number 获取当前页的基于零的索引。
F pageSize number 获取或设置页面上显示的项的数量。
F totalItemCount number

在使用分页之前获取视图中的项目总数。

要获得当前视图中的项目数，不考虑分页，使用itemCount 属性，不进行账目分页。

注意，它不同于.NET IPagedCollectionView, itemCount 和totalItemCount 都在分页被应用之前返回数量。
