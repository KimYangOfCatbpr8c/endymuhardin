wjFlexChartWilliamsR 类

基类
WjBaseSingleOverlayIndicator
模块
wijmo.knockout
文件
wijmo.knockout.js
表示

KnockoutJS 绑定用于 WilliamsR 对象。

使用 wjFlexChartWilliamsR 绑定向你的 KnockoutJS 应用程序添加 WilliamsR 对象。比如： 

<p>Here is a WilliamsR:</p>
   <div data-bind="wjFinancialChart: { itemsSource: fData, bindingX: 'date'}">
        <div data-bind="wjFlexChartWilliamsR: { binding: 'high,low,open,close',period:20 }"></div>
  </div>

wjFlexChartWilliamsR 绑定支持所有 WilliamsR 的读写属性和事件
