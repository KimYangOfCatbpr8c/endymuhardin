DataMap 类

模块
wijmo.grid
文件
wijmo.grid.js

表示列的 dataMap 属性的数据映射。

数据列向表单提供了自动查找的能力。比如说，您希望显示一个客户的名称，而不是 ID，或者 一个颜色值，而不是其原始的 RGB 值。

以下代码将一个表单绑定到一组产品的集合，之后指定一个 DataMap 至表单的 ‘CategoryID’ 列，因此表单显示分类的名称而不是其原始的 ID。

// 绑定表单至产品
var flex = new wijmo.grid.FlexGrid();
flex.itemsSource = products;
// 映射 CategoryID 列以显示分类的名称而不是原始的 ID
var col = flex.columns.getColumn('CategoryID');
col.dataMap = new wijmo.grid.DataMap(categories, 'CategoryID', 'CategoryName');
