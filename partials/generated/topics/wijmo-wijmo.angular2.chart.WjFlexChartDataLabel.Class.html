WjFlexChart DataLabel控件对应的 Angular 2 组件。

wj-flex-chart-data-label 组件必须包含在一个 WjFlexChart 组件内部。

使用 wj-flex-chart-data-label 组件添加 DataLabel 对象至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjFlexChartDataLabel 组件继承自 DataLabel 类，因此继承了其全部的属性，事件和方法。

content any

获取或设置数据标签的内容。

该内容可以指定为一个字符串，或者一个接受 HitTestInfo 对象作为参数的函数。

当标签的内容为一个字符串时，它可以包含任意以下参数：

  • seriesName: 包含该数据点的系列名称（仅 FlexChart）。
  • pointIndex: 数据点的索引。
  • value: 数据点的值。
  • x: 数据点的 x-值（仅 FlexChart）。
  • y: 数据点的 y-值（仅 FlexChart）。
  • name: 数据点的 Name。
  • propertyName: 数据对象的任意属性。

参数必须包含在一对花括号中，比如说 'x={x}, y={y}'。

以下示例中，我们在标签上展示数据点的 y 值。

 
 // 创建一个图表，并在数据点位置上方的标签中显示 y 值。
 var chart = new wijmo.chart.FlexChart('#theChart');          
 chart.initialize({
     itemsSource: data,
     bindingX: 'country',
     series: [
         { name: 'Sales', binding: 'sales' },
         { name: 'Expenses', binding: 'expenses' },
         { name: 'Downloads', binding: 'downloads' }],                            
 });
 chart.dataLabel.position = "Top";
 chart.dataLabel.content = "{country} {seriesName}:{y}";

下一个示例展示如何通过一个函数设置数据标签的内容。

 
 // 设置数据标签内容
 chart.dataLabel.content = function (ht) {
   return ht.name + ":" + ht.value.toFixed();
 }
