RangeMode 枚举

模块
wijmo.chart.finance
文件
wijmo.chart.finance.js

指定 Kagi 以及 Renko 图表类型的单位。

值

名称 描述
ATR 为 Kagi 图表的转回金额或者 Renko 图表的 box 尺寸使用一个平均真实范围值。当使用了 ATR 时，转回金额或者 box 尺寸，这些图表的选项必须是一个整数，并且将会被用作 ATR 计算的周期。
Fixed 为 Kagi 图表的转回金额或者 Renko 图表的 box 尺寸使用一个固定的，正数的数值。
Percentage 为 Kagi 图表的转回金额使用一个百分比。RangeMode.Percentage 当前不被 Renko 图表类型支持。
