IFlexGridStyle 类

模块
wijmo.grid.pdf
文件
wijmo.grid.pdf.js

表示待导出的 FlexGrid 的外观。

属性

名称 类型 描述
F altCellStyle? ICellStyle 表示应用到 FlexGrid 偶数行的单元格样式。
F cellStyle? ICellStyle 指定应用到 FlexGrid 内部的单元格样式。
F groupCellStyle? ICellStyle 表示应用到 FlexGrid 分组行的单元格样式。
F headerCellStyle? ICellStyle 表示应用到 FlexGrid 行头和列头的单元格样式。
