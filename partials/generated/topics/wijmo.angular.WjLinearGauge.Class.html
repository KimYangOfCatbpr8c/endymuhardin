AngularJS 指令应用于 LinearGauge 控件。

使用 wj-linear-gauge 指令添加 linear gauges 到 AngularJS 应用中。 注意指令和参数名称必须使用小写字母+短线语法替代驼峰命名。 示例：

<wj-linear-gauge 
  value="ctx.gauge.value" 
  show-text="Value"
  is-read-only="false">
  <wj-range 
    wj-property="pointer" 
    thickness="0.2">
    <wj-range 
      min="0" 
      max="33" 
      color="green">
    </wj-range>
    <wj-range 
      min="33" 
      max="66" 
      color="yellow">
    </wj-range>
    <wj-range 
      min="66" 
      max="100" 
      color="red">
    </wj-range>
  </wj-range>
</wj-linear-gauge>

wj-linear-gauge 指令支持以下属性:

ng-model
@ 使用Angular的ng-model指令绑定控件的 vlue 属性。使用ng-model指令绑定属性能够获得一些共通的特性，例如验证，自动添加控件状态到form实例，等等。 使用 wj-model-property 重定义被 ng-model指定绑定的控件属性。
wj-model-property
@ 使用 ng-model 指令指定控件绑定的作用域。
control
= 该指令创建的指向 LinearGauge 的引用
direction
@ GaugeDirection 值，测量值增长的方向。
format
@ 以文本显示测量值时的格式化字符串。
has-shadow
@ 是否显示阴影。
initialized
& 该事件在绑定完成控件特性值的初始化后被触发。
is-initialized
= 绑定是否已经完成控件特性值的初始化。
is-animated
@ 是否使用动画显示测量值的变化。
is-read-only
@ 是否只读。
min
@ 显示测量值的最大值。
max
@ 显示测量值的最小值。
show-text
@ ShowText 值，测量值的显示文本。
step
@ 用户点击增加或减少按钮时的步进值。
thickness
@ 测量值的厚度，从0到1的值。
value
= 显示的测量值。
got-focus
& gotFocus 事件。
lost-focus
& lostFocus 事件。

wj-linear-gauge 指令可能包含一个或多个 WjRange 指令。
