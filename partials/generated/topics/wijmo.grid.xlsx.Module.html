模块 wijmo.grid.xlsx

文件
wijmo.grid.xlsx.js

为 FlexGrid 控件提供客户端 Excel xlsx 格式文件导入/导出能力的定义了 FlexGridXlsxConverter 类的扩展包。

类

名称 描述
FlexGridXlsxConverter 该类型提供了静态的 import 以及 export 方法，以便从 Excel xlsx 格式的文件导入和导出 FlexGrid 控件。

接口

名称 描述
IExtendedSheetInfo 定义额外的可以被 FlexGrid 实例上动态的 wj_sheetInfo 属性访问的工作簿属性。
IFlexGridXlsxOptions FlexGrid Xlsx 转换选项

属性

名称 类型 描述
F activeWorksheet number xlsx 文件中活动表单的索引。
F dataType DataType 数据类型
F format string 格式
F hAlign string 水平对齐方式
F includeCellStyles boolean 表示是否在生成的 xlsx 文件中包含单元格样式。
F includeColumnHeader boolean 表示是否将 xlsx 文件的第一行认作包含列头名称。
F includeColumnHeaders boolean 表示是否在生成的 xlsx 文件的第一行包含列头。
F includeRowHeaders boolean 表示是否在生成的 xlsx 文件的第一列包含行头。
