FilterCondition 类

模块
wijmo.grid.filter
文件
wijmo.grid.filter.js

定义一个筛选器条件。

该类被 FlexGridFilter 类所使用；您将几乎不会直接使用。

属性

名称 类型 描述
isActive boolean 获取一个值，该值表示是否该条件为活动状态。
operator Operator 获取或设置该 FilterCondition 使用的操作符。
value any 获取或设置该 FilterCondition 使用的值。

方法

名称 参数 描述
apply​(value): boolean

value (): 待测试的值。

 返回一个值，该值表示是否给定的值通过了该 FilterCondition。
clear​() 清除全部条件。
