文件
wijmo.xlsx.js

该模块依赖外部的 JSZip 库， 需要在 HTML 页面中使用类似以下的标记添加引用： 

<script src="http://cdnjs.cloudflare.com/ajax/libs/jszip/2.5.0/jszip.min.js"></script>

接口

名称 描述
ITableAddress 使用从零开始计算索引值的行和列部分组成的一个单元格索引，同时还包含了表示是否该索引部分是绝对（例如“$D”）的还是相对的（例如“D”）的属性。
IWorkbook

表示 Excel 工作簿。该接口是 Excel Workbook 对象模型（WOM）的一个根定义， 提供了定义存储在 xlsx 文件中数据和属性的简单方式。

为了创建一个 xlsx 文件，您需要创建一个符合 IWorkbook 接口定义的 JavaScript 对象，同时创建子对象定义工作簿表单以及其中的列，行以及单元格。 接着将该对象传递给 XlsxConverter 类的 export 方法，该方法会返回给您一个相关的 xlsx 文件的内容，该内容可以被保存在磁盘 或上传到服务器。

为了读取一个 xlsx 文件数据，您需要将 xlsx 文件的内容传递给 XlsxConverter 类的 import 方法， 该方法将返回一个符合 IWorkbook 接口的 JavaScript 对象，该对象允许您容易地检查工作簿的数据和属性。您可以以各种 方式进一步地使用该对象，例如，将其内容绘制到 HTML 页面或者将其保存为一个数据数组，以便进一步处理或用于数据绑定。

属性

名称 类型 描述
F absCol boolean 表示是否原始的列索引是绝对表示（比如说 “$D”）还是相对表示（比如说“D”）。
F absRow boolean 表示是否原始的行索引是绝对表示（比如说 “$15”）还是相对表示（比如说“15”）。
F base64 string 表示导出结果的 base64 编码的字符串。
F base64Array Uint8Array base64 编码字符串结果转换的 int 型数组。
F col number 从零开始计算的列索引。
F href Function 导出结果的下载链接。
F row number 从零开始计算的行索引。
