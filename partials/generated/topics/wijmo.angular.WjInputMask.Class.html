AngularJS 指令应用于 InputMask 控件。

使用 wj-input-mask 指令添加 InputMask 到 AngularJS 应用中。 注意指令和参数名称必须使用小写字母+短线语法替代驼峰命名。 示例：

<p>Here is an InputMask control:</p>
<wj-input-mask
  mask="99/99/99"
  mask-placeholder="*">
</wj-input-mask>

wj-input-mask 指令支持以下属性:

ng-model
@ 使用Angular的ng-model指令绑定控件的 value 属性。使用ng-model指令绑定属性能够获得一些共通的特性，例如验证，自动添加控件状态到form实例，等等。 使用 wj-model-property 重定义被 ng-model指定绑定的控件属性。
wj-model-property
@ 使用 ng-model 指令指定控件绑定的作用域。
control
= 该指令创建的指向 InputNumber 的引用
initialized
& 该事件在绑定完成控件特性值的初始化后被触发。
is-initialized
= 绑定是否已经完成控件特性值的初始化。
mask
@ 格式化用户输入的掩饰字符。
prompt-char
@ 提示用户输入位置的字符。
place-holder
@ 描述输入字段预期值的提示信息。
value
= 编辑的字符值。
raw-value
= 编辑的字符值，移除了占位符等。
got-focus
& gotFocus 事件。
lost-focus
& lostFocus 事件。
value-changed
& valueChanged 事件。
