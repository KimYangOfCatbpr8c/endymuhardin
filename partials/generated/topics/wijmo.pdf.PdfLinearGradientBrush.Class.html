PdfLinearGradientBrush 类

基类
PdfGradientBrush
模块
wijmo.pdf
文件
wijmo.pdf.js
表示

表示通过线性渐变填充一个区域的画刷。

构造函数

名称 参数 描述
constructor​(x1, y1, x2, y2, stops, opacity): PdfLinearGradientBrush

x1 (number): 线性渐变的起始X坐标 。
y1 (number): 线性渐变的起始Y坐标 。
x2 (number): 线性渐变的结束X坐标 。
y2 (number): 线性渐变的结束X坐标 。
stops (PdfGradientStop[]): 设置在画刷上的 PdfGradientStop 数组。
opacity (number): 这个画刷的透明度 。

 初始化 PdfLinearGradientBrush 类的一个新实例。

属性

名称 类型 描述
opacity number 获取或设置画刷的不透明度。 值必须位于范围 [0, 1] 之间， 这里 0 表示画刷完全透明，而 1 表示画刷完全不透明。 默认值为 1。
(继承自 PdfGradientBrush).
stops PdfGradientStop[] 获取或设置一个 PdfGradientStop 对象的集合，每一个对象表示在画刷的渐变轴上的一个颜色，偏移量以及不透明度。 默认值为一个空白集合。
(继承自 PdfGradientBrush).
x1 number 获取或设置线性渐变起始点的 X-轴坐标， 位于页面区域坐标系，单位是点。
x2 number 获取或设置线性渐变结束点的 X-轴坐标， 位于页面区域坐标系，单位是点。
y1 number 获取或设置线性渐变起始点的 Y-轴坐标， 位于页面区域坐标系，单位是点。
y2 number 获取或设置线性渐变结束点的 Y-轴坐标， 位于页面区域坐标系，单位是点。

方法

名称 参数 描述
clone​(): PdfLinearGradientBrush 创建该 PdfLinearGradientBrush 的一份拷贝。

返回值 该画刷的一个拷贝
equals​(value): boolean

value (PdfLinearGradientBrush): 待比较的 PdfLinearGradientBrush 对象。

 确定是否指定的 PdfLinearGradientBrush 实例和当前的实例相等。

返回值 如果指定对象与当前对象相等，则返回真，否则返回假。
