表示

表示页面的运行时标题，比如说页眉和页脚。

该类没有被设计用来在您的代码中实例化。

方法

名称 参数 描述
drawImage​(url, x?, y?, options?): PdfPageArea

url (string): 一个 URL，用于从中获取一个图像，或者是一个 Data URI，包含一个 base64 编码的图像的字符串。
x? (number): 绘制图像目标位置的点的 X-轴坐标，单位是点。
y? (number): 绘制图像目标位置的点的 Y-轴坐标，单位是点。
options? (IPdfImageDrawSettings): 确定图像的绘制选项。

使用给定的选项绘制一个 JPG 或者 PNG 格式的图像。

如果 x 和 y 未指定，则相应地使用 x 以及 y 替代。 最终，该方法更新 y 的值。 于是，任何后续的文本或者图像将从该点开始进行绘制。



返回值 The PdfPageArea object.
(继承自 PdfPageArea).
drawSvg​(url, x?, y?, options?): PdfPageArea

url (string): A string containing the URL to get the SVG image from or the data URI containing a base64 encoded SVG image.
x? (number): The x-coordinate of the point to draw the image at, in points.
y? (number): The y-coordinate of the point to draw the image at, in points.
options? (IPdfSvgDrawSettings): Determines the SVG image drawing options.

 用给定的选项绘制一个SVG图形。 如果 x 和 y 没有指定，那么使用 x 和 y 来代替。 这个方法使用svg元素的宽高属性值，通过选项的options.width 和 options.height属性来决定缩放因素。 如果参数忽略，那么缩放将不会生效，图片会按照原始大小绘制。 最终，如果图片使用文本流，那么方法会更新y. Hence。 任何连续的文本或图片都会从这个点开始绘制。 增长值使用 options.height 属性提供，或者 outermost svg 元素的高度，取决于谁先定义。 如果上面的值都没有提供，那么 y 将会保持不变。 这个方法提供限制的SVG 功能集合，主要用于绘制WIjmo5控件。

返回值 The PdfPageArea object.
(继承自 PdfPageArea).
drawText​(text, x?, y?, options?): IPdfTextMeasurementInfo

text (string): 待绘制的文本。
x? (number): 待绘制文本的点的 X-轴坐标，单位是点。
y? (number): 待绘制文本的点的 Y-轴坐标，单位是点。
options? (IPdfTextDrawSettings): 确定文本的绘制选项。

使用指定的选项绘制一个文本，并返回文本的测量信息。

如果 pen，brush 或者 font 属性的值被忽略，则使用当前文档上的 画笔，画刷或者字体（setPen，setBrush，setFont）。

文本在一个指定的矩形区域内绘制，左上角，宽度和高度分别由 x，y，options.width 以及 options.height 定义。 如果 x 和 y 没有定义，则相应地使用 x 以及 y 替代。

文本自动地在该区域内折行并裁剪。 如果没有指定 options.height，并且文本超出了 body 底部边缘，则文本将自动地扩展到一个新页面上。

最终，该方法更新 x 以及 y 的值。 于是，任何后续的文本或者图像将从该点开始进行绘制（取决于 options.continued 的设置）。

测量结果并不反映文本实际上可以被拆分到多页或者多列的事实；该文本将被看作一个单一的块对待。



返回值 A IPdfTextMeasurementInfo object determines the measurement information.
(继承自 PdfPageArea).
lineHeight​(font?): number

font? (PdfFont): 用来获取行高的字体。

通过给定的字体获取行高。

如果字体未指定，则使用当前文档上使用的字体。



返回值 行高，单位是像素。
(继承自 PdfPageArea).
measureText​(text, font?, options?): IPdfTextMeasurementInfo

text (string): 待测量的文本。
font? (PdfFont): 待应用到该文本的字体。
options? (IPdfTextSettings): 确定文本的绘制选项。

使用指定的字体以及文本绘制选项测量一段文本，无须将其真实呈现。

如果字体未指定，则使用当前文档上使用的字体。

该方法使用和 drawText 相同的文本呈现引擎，因此当 option.width 未指定时，它以相同的方式关联到 x 以及右侧页面边距。 测量结果将不会反映文本实际上可以拆分成多页或多列的事实；文本被当作一个单一的块对待。



返回值 A IPdfTextMeasurementInfo object determines the measurement information.
(继承自 PdfPageArea).

angle (number): 旋转的角度，单位是度。
origin? (Point): 旋转的中心点 Point，单位是点。如果没有提供，则使用左上角。

 顺时针旋转图形上下文为一个指定的角度。

scale​(xFactor, yFactor?, origin?): PdfPageArea

xFactor (number): 缩放 X 轴维度的缩放因子。
yFactor? (number): 缩放 Y 轴维度的缩放因子。如果没有提供，则假定和 xFactor 相等。
(默认值: xFactor)
origin? (Point): 缩放的中心点的 Point，单位是点。如果没有提供，则使用左上角。

按照一个指定的缩放因子缩放图形上下文。

介于 [0 ,1] 范围之间的缩放因子表示尺寸将缩小。 值大于 1 的缩放因子表示尺寸将增大。



返回值 The PdfPageArea object.

transform​(a, b, c, d, e, f): PdfPageArea

a (number): 第一行和第一列的值。
b (number): 第一行和第二列的值。
c (number): 第二行和第一列的值。
d (number): 第二行和第二列的值。
e (number): 第三行和第一列的值。
f (number): 第三行和第二列的值。

通过给定的六个值，该值表示一个 3 乘以 3 的转换矩阵，对一个图形上下文进行变换。

一个变换矩阵书写格式如下：

ef1
