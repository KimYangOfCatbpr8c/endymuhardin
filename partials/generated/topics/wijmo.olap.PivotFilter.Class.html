PivotFilter 类

模块
wijmo.olap
文件
wijmo.olap.js

表示用做在 PivotField 中选择值的一个筛选器。

构造函数

名称 参数 描述
constructor​(field): PivotFilter

field (PivotField): 拥有这个过滤器的 PivotField 。

 初始化 PivotFilter 类的一个新实例。

属性

名称 类型 描述
conditionFilter ConditionFilter 获取 PivotFilter 中的 ConditionFilter。
filterType FilterType

获取或设置该筛选器提供的筛选的类型。

设置该属性的值为 null 将使得该筛选器使用由 owner 筛选器的 defaultFilterType 属性定义的值。
isActive boolean 获取一个值，该值表示是否一个筛选器处于活动状态。
valueFilter ValueFilter 获取 PivotFilter 中的 ValueFilter。

方法

名称 参数 描述
apply​(value): boolean

value (): 待测试的值。

 获取一个值，该值表示是否测试值通过了该筛选器的筛选。
clear​(): void 清除该筛选器。
