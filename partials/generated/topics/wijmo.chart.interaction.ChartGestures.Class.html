ChartGestures 类

派生类
WjFlexChartGestures
模块
wijmo.chart.interaction
文件
wijmo.chart.interaction.js

ChartGestures 控件允许用户在通过 Zoom 或者 Pan 操作缩放指定的 FlexChart。

为使用 ChartGestures 控件，指定需要 zoom 或 pan 操作的 FlexChart。

 var chartGestures = new wijmo.chart.interaction.ChartGestures(chart);

构造函数

名称 参数 描述
constructor​(chart, options?): ChartGestures

chart (FlexChartCore): The FlexChart that allows the user to zoom or pan.
options? (): A 包含控件初始化数据的Javascript对象。

 初始化类 ChartGestures 的一个新实例。.

属性

名称 类型 描述
enable boolean 获取或设置 ChartGestures 的启用状态。
interactiveAxes InteractiveAxes 获取或设置 ChartGestures 的交互坐标轴。
mouseAction MouseAction 获取或设置 ChartGestures 的鼠标动作。
posX number 获取或设置 X 坐标轴的初始位置。 该值表示当 Scale 小于 1 时，坐标轴的初始位置。 否则，该 Value 没有效果。 该值应当介于 0 和 1 之间。
posY number 获取或设置 Y 坐标轴的初始位置。 该值表示当 Scale 小于 1 时，坐标轴的初始位置。 否则，该 Value 没有效果。 该值应当介于 0 和 1 之间。
scaleX number 获取或设置 X 坐标轴的初始缩放。 缩放应当大于 0 并且小于等于 1。 缩放指定介于 Min 和 Max 之间的哪一部分将显示。 当缩放值为 1（默认值），整个坐标轴范围可见。
scaleY number 获取或设置 Y 坐标轴的初始缩放。 缩放应当大于 0 并且小于等于 1。 缩放指定介于 Min 和 Max 之间的哪一部分将显示。 当缩放值为 1（默认值），整个坐标轴范围可见。

方法

名称 参数 描述
remove​() 从图表上移除 ChartGestures 控件。
reset​() 重置图表的坐标轴。
