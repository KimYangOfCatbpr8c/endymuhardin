AngularJS 指令应用于 InputNumber 控件。

使用 wj-input-number 指令添加 InputNumber 到 AngularJS 应用中。 注意指令和参数名称必须使用小写字母+短线语法替代驼峰命名。 示例：

<p>Here is an InputNumber control:</p>
<wj-input-number
  value="theNumber"
  min="0"
  max="10"
  format="n0"
  placeholder="number between zero and ten">
</wj-input-number>

下面的例子创建了几个 InputNumber 控件，并且展示了不同的格式化字符，范围，步进值的效果。

wj-input-number 指令支持以下属性:

ng-model
@ 使用Angular的ng-model指令绑定控件的 value 属性。使用ng-model指令绑定属性能够获得一些共通的特性，例如验证，自动添加控件状态到form实例，等等。 使用 wj-model-property 重定义被 ng-model指定绑定的控件属性。
wj-model-property
@ 使用 ng-model 指令指定控件绑定的作用域。
control
= 该指令创建的指向 InputNumber 的引用
format
@ 显示数字的格式化字符 (参见 Globalize).
input-type
@ 输入控件的类型。
initialized
& 该事件在绑定完成控件特性值的初始化后被触发。
is-initialized
= 绑定是否已经完成控件特性值的初始化。
max
@ 最大有效数值。
min
@ 最小有效数值。
place-holder
@ 描述输入字段预期值的提示信息。
required
@ 是否阻止null值。
show-spinner
@ 是否显示按钮用于改变 step 值。
step
@ 用户点击按钮时增加或较少的值。
text
= 控件显示文本。
value
= 编辑的数字。
got-focus
& gotFocus 事件。
lost-focus
& lostFocus 事件。
text-changed
& textChanged 事件。
value-changed
& valueChanged 事件。
