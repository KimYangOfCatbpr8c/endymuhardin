Rect 类

模块
wijmo
文件
wijmo.js

类代表矩形(左, 上, 宽, 和高)。

构造函数

名称 参数 描述
constructor​(left, top, width, height): Rect

left (number): 新 Rect的左坐标。
top (number): 新 Rect的上坐标。
width (number): 新 Rect的宽度。
height (number): 新 Rect的高度。

 初始化类 Rect 的一个新实例。

属性

名称 类型 描述
bottom number 获取Rect 的底坐标
F height number 获取或者设置Rect 的高度。
F left number 获取或者设置Rect 的左坐标。
right number 获取Rect 的右坐标。
F top number 获取或者设置Rect 的上坐标。
F width number 获取或者设置Rect 的宽度。

方法

名称 参数 描述
clone​(): Rect 创建Rect 的一个副本.
contains​(pt): boolean

pt (any): Point 或者 Rect 的检查。

 巨鼎一个矩形是否包含给定的点或者矩形。
equals​(rc): boolean

rc (Rect): 待与Rect 比较的 Rect 。

 当一个 Rect 与这个拥有相同的坐标和外形尺寸时返回true 。
S fromBoundingRect​(rc): Rect

rc (any): 通过调用DOM的getBoundingClientRect 或者 GetBoundingBox 方法获取矩形。

 从 ClientRect 或者 SVGRect 对象创建一个Rect。
inflate​(dx, dy): Rect

dx (number): 扩大或缩小矩形的左右两侧的量。
dy (number): 扩大或缩小矩形的上下两侧的量。

 创建一个矩形，该结果是通过指定的量来扩大或者缩小一个矩形而来的。
S intersection​(rc1, rc2): Rect

rc1 (Rect): 第一个矩形
rc2 (Rect): 第二个矩形

 获取表示两个矩形的交集的矩形。
S union​(rc1, rc2): Rect

rc1 (Rect): 第一个矩形。
rc2 (Rect): 第二个矩形。

 获取一个矩形，该矩形代表两个矩形的合并。
