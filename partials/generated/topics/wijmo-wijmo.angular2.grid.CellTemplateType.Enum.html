CellTemplateType 枚举

模块
wijmo/wijmo.angular2.grid
文件
wijmo.angular2.js

定义应用该模板的单元格的类型。该值指定在 WjFlexGridCellTemplate 指令的 cell-type attribute 属性中。

值

名称 描述
BottomLeft Defines a bottom left cell (at the intersection of the row header and column footer cells).
Cell 定义一个普通的（数据）单元格。
CellEdit 定义一个处于编辑状态下的单元格。
ColumnFooter Defines a column footer cell.
ColumnHeader 定义一个列头单元格。
Group 定义一个位于分组行中间的普通单元格。
GroupHeader 定义一个位于分组行中的分组行头单元格。
NewCellTemplate 定义一个位于新行模板中的单元格。
RowHeader 定义一个行头单元格。
RowHeaderEdit 定义一个处于编辑模式下的行头单元格。
TopLeft 定义左上角的单元格。
