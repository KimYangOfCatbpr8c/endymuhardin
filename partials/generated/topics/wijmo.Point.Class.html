类代表一个点(x坐标和y坐标)。

构造函数

名称 参数 描述
constructor​(x?, y?): Point

x? (number): 新点的X坐标。
(默认值: 0)
y? (number): 新点的Y坐标。
(默认值: 0)

 初始化类 Point 的一个新实例。

属性

名称 类型 描述
F x number 获取或者设置该点 的x坐标。
F y number 获取或者设置该点 的y坐标。

方法

名称 参数 描述
clone​(): Point 创建该点 的一个副本。
equals​(pt): boolean

pt (Point): 点 比较该 点。

 如果一个点 与这个点 有同样的坐标，则返回true。
