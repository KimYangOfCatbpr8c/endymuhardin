模块 wijmo.pdf

文件
wijmo.pdf.js

定义 PdfDocument 类以及其相关联的类型。

类

名称 描述
PdfBrush

表示一个抽象基类，表示全部画刷的基类。 从该类派生的任何类的实例用来填充区域和文本。

该类没有被设计用来在您的代码中实例化。
PdfDashPattern 表示用做绘制路径的虚线样式。
PdfDocument 表示一个基于 PDFKit JavaScript 类库的一个 PDF 文档对象。
PdfDocumentEndedEventArgs 为 end 事件提供事件参数。
PdfFont 表示一个字体。
PdfGradientBrush

表示一个做为 PdfLinearGradientBrush 以及 PdfRadialGradientBrush 类的基类的抽象类。

该类没有被设计用来在您的代码中实例化。
PdfGradientStop 表示一个对象，该对象决定一个渐变的过渡点。
PdfLinearGradientBrush 表示通过线性渐变填充一个区域的画刷。
PdfPageArea

表示页面上具有其自主坐标系的一个区域，(0, 0)点表示左上角。 为绘制文本，图像，路径和平移提供方法。

该类没有被设计用来在您的代码中实例化。
PdfPaths

提供方法创建 graphics path，用来进行图形绘制或者将其用做 Clip。

路径创建方法调用必须通过 stroke，fill，fillAndStroke 或者 clip 方法结束。 任何不会直接应用到路径创建/绘制/Clip的方法（改变画笔，绘制一个文本，保存图形上下文状态等等）需要禁止调用，直至该路径完成操作。 lineTo，bezierCurveTo 以及 quadraticCurveTo 方法不会开始路径操作，它们必须在 moveTo 之后进行操作。

该方法支持链式方法： 

doc.paths.moveTo(0, 0).lineTo(100, 100).stroke();

该类没有被设计用来在您的代码中实例化。
PdfPen 决定一个用来绘制路径和文本笔画的对象。
PdfRadialGradientBrush 表示通过放射状渐变填充一个区域的画刷。
PdfRunningTitle

表示页面的运行时标题，比如说页眉和页脚。

该类没有被设计用来在您的代码中实例化。
PdfRunningTitleDeclarativeContent 表示运行标题的声明式内容。
PdfSolidBrush 表示使用单色填充一个区域的画刷。

接口

名称 描述
IPdfBufferedPageRange 表示由 bufferedPageRange 方法返回的一个范围内的缓冲页面。
IPdfDocumentInfo 表示 info 属性使用的文档信息。
IPdfFontAttributes 表示字体属性。
IPdfFontFile 表示由 registerFont 以及 registerFontAsync 方法注册的字体的设置。
IPdfImageDrawSettings

表示 drawImage 方法所使用的图像绘制设置。

如果宽度和高度选项均未指定，则图像将按照其原始尺寸绘制。 如果仅提供了宽度，那么图像将被按比例缩放以适应指定的宽度。 如果仅提供了高度，则图像将被按比例缩放以适应指定的高度。 如果同时指定了宽度和高度，则图像将按照 stretchProportionally 属性的设置，对图像进行拉伸。
IPdfPageMargins 表示页面页边距。
IPdfPageSettings 表示页面设置。
IPdfSvgDrawSettings drawSvg 方法画SVG图形时所使用的设置。
IPdfTextDrawSettings 表示在使用指定的 PdfPen 以及 PdfBrush 绘制文本时，drawText 所使用的设置。
IPdfTextMeasurementInfo 表示由 measureText 方法返回的文本测量结果信息。
IPdfTextSettings 表示 drawText 以及 measureText 方法使用的文本设置。

枚举

名称 描述
PdfFillRule 指定一个决定是否某个点位于一个封闭路径的内部的规则。
PdfImageHorizontalAlign 指定图像的水平对齐方式。
PdfImageVerticalAlign 指定图像的垂直对齐方式。
PdfLineCapStyle 指定当绘制线形时，用做非封闭的子路径（以及长划线，如果存在的话）两端的形状。
PdfLineJoinStyle 指定绘制路径时，拐角处的形状。
PdfPageOrientation 指定页面方向。
PdfPageSize 指定页面尺寸，单位是点。
PdfTextHorizontalAlign 指定文本内容的水平对齐方式。
