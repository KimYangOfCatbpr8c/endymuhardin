IFlexGridDrawSettings 类

模块
wijmo.grid.pdf
文件
wijmo.grid.pdf.js

表示 draw 以及 drawToPosition 方法所使用的设置。

属性

名称 类型 描述
F embeddedFonts? IPdfFontFile[]

表示将内嵌到文档中的自定义字体的数组。

该示例演示如何设置 FlexGridPdfConverter 使用两种自定义字体，Cuprum-Bold.ttf 以及 Cuprum-Regular.ttf。 第一个仅应用到行头或列头单元格，而第二个字体将应用到其余所有的单元格。

	wijmo.grid.pdf.FlexGridPdfConverter.export(flex, fileName, {
		embeddedFonts: [
			{
				source: 'resources/ttf/Cuprum-Bold.ttf',
				name: 'cuprum',
				style: 'normal',
				weight: 'bold'
			},
			{
				source: 'resources/ttf/Cuprum-Regular.ttf',
				name: 'cuprum',
				style: 'normal',
				weight: 'normal'
			}
		],
		styles: {
			cellStyle: {
				font: {
					family: 'cuprum'
				}
			},
			headerCellStyle: {
				font: {
					weight: 'bold'
				}
			}
		}
	});

F exportMode? ExportMode 确定导出模式。
F maxPages? number 确定最大导出页数。
F recalculateStarWidths? boolean 确定是否星号尺寸标记的列宽应当基于 PDF 页面的宽度重新计算而不是使用表单的宽度。
F repeatMergedValuesAcrossPages? boolean 指示当合并的范围切分到多个页面时，是否合并的值应当跨页面重复。
F styles? IFlexGridStyle 表示导出的 FlexGrid 的外观。
