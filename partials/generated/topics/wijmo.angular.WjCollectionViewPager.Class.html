AngularJS directive for an ICollectionView pager element.

使用 wj-collection-view-pager 指令添加元素，允许用户通过 ICollectionView 中的页面进行导航。 注意指令和参数名称必须使用小写字母+短线语法替代驼峰命名。 示例：

Here is a CollectionViewPager:</p>
<wj-collection-view-pager
  cv="myCollectionView">
</wj-collection-view-pager>

在线示例

这个例子创建了一个拥有100,000个项目和每页20个项目的 CollectionView ，它定义了一个导航器用以选择当前页，当前项目，并使用 FlexGrid 呈现数据。

wj-collection-view-pager 指令只有一个属性：

cv
= 指向 paged ICollectionView 对象的引用。
