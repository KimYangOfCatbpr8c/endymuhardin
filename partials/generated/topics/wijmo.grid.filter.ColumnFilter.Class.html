定义一个 FlexGrid 控件上某一列的筛选器。

ColumnFilter 包含一个 ConditionFilter 以及一个 ValueFilte； 同一时刻只能有一个处于活动状态。

该类被 FlexGridFilter 类所使用；您一般不会直接使用。

构造函数

名称 参数 描述
constructor​(owner, column): ColumnFilter

owner (FlexGridFilter): 拥有该列筛选器的 FlexGridFilter。
column (Column): 待筛选的 Column。

 初始化 ColumnFilter 类的一个新实例

属性

名称 类型 描述
column Column 获取待筛选的 Column。
conditionFilter ConditionFilter 获取该 ColumnFilter 中的 ConditionFilter。
filterType FilterType

获取或设置该筛选器提供的筛选类型。

设置该属性的值为 null 将使得该筛选器使用由 owner 筛选器的 defaultFilterType 属性定义的值。
isActive boolean 获取一个值，该值表示是否该筛选器为活动状态。
valueFilter ValueFilter 获取该 ColumnFilter 中的 ValueFilter。
