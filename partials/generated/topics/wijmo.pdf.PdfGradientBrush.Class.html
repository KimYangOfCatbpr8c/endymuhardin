表示

表示一个做为 PdfLinearGradientBrush 以及 PdfRadialGradientBrush 类的基类的抽象类。

该类没有被设计用来在您的代码中实例化。

构造函数

名称 参数 描述
constructor​(stops?, opacity?): PdfGradientBrush

stops? (PdfGradientStop[]): The PdfGradientStop array to set on this brush.
opacity? (number): The opacity of this brush.

 初始化 PdfGradientBrush 类的一个新实例。

属性

名称 类型 描述
opacity number 获取或设置画刷的不透明度。 值必须位于范围 [0, 1] 之间， 这里 0 表示画刷完全透明，而 1 表示画刷完全不透明。 默认值为 1。
stops PdfGradientStop[] 获取或设置一个 PdfGradientStop 对象的集合，每一个对象表示在画刷的渐变轴上的一个颜色，偏移量以及不透明度。 默认值为一个空白集合。

返回值 这个画刷的一个拷贝。
