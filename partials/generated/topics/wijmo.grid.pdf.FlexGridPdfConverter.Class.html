FlexGridPdfConverter 类

模块
wijmo.grid.pdf
文件
wijmo.grid.pdf.js

提供导出 FlexGrid 至 PDF 的功能。

方法

名称 参数 描述
S draw​(flex, doc, width?, height?, settings?): void

flex (FlexGrid): 待导出的 FlexGrid 实例。
doc (PdfDocument): 绘制的目标 PdfDocument 实例。
width? (number): 绘制区域的宽度，单位是点。
height? (number): 绘制区域的高度，单位是点。
settings? (IFlexGridDrawSettings): 绘制相关的设置。

将 FlexGrid 绘制到一个现有的 PdfDocument 文档的 (0, @wijmo.pdf.PdfDocument.y)坐标位置。

如果宽度未指定，则表单将以实际尺寸呈现，并按照需要拆分多个页面。 如果高度未指定，则表单将拉伸以适应页面宽度，并按照需要在垂直方向 拆分成多个页面。如果高度和宽度同时指定，则表单将拉伸以适应指定的 矩形区域，此时不需要做任何的分页。

	var doc = new wijmo.pdf.PdfDocument({
		ended: function (sender, args) {
			wijmo.pdf.saveBlob(args.blob, 'FlexGrid.pdf');
		}
	});



	wijmo.grid.pdf.FlexGridPdfConverter.draw(grid, doc, null, null, {
		maxPages: 10,
		styles: {
			cellStyle: {
				backgroundColor: '#ffffff',
				borderColor: '#c6c6c6'
			},
			headerCellStyle: {
				backgroundColor: '#eaeaea'
			}
		}
	});

S drawToPosition​(flex, doc, point, width?, height?, settings?): void

flex (FlexGrid): 待导出的 FlexGrid 实例。
doc (PdfDocument): 绘制的目标 PdfDocument 实例。
point (Point): 绘制的目标位置，单位是点。
width? (number): 绘制区域的宽度，单位是点。
height? (number): 绘制区域的高度，单位是点。
settings? (IFlexGridDrawSettings): 绘制相关的设置。

在指定的坐标位置绘制 FlexGrid 至现有的 PdfDocument 实例。

如果没有指定宽度，则表单将以实际尺寸绘制，且没有任何分页。 如果没有指定宽度，则表单将缩放以适应页面宽度，且没有分页。 如果同时指定了宽度和高度，则表单将缩放以适应指定的矩形区域，不需要分页显示。

	var doc = new wijmo.pdf.PdfDocument({
		ended: function (sender, args) {
			wijmo.pdf.saveBlob(args.blob, 'FlexGrid.pdf');
		}
	});



	wijmo.grid.pdf.FlexGridPdfConverter.drawToPosition(grid, doc, new wijmo.Point(0, 0), null, null, {
		maxPages: 10,
		styles: {
			cellStyle: {
				backgroundColor: '#ffffff',
				borderColor: '#c6c6c6'
			},
			headerCellStyle: {
				backgroundColor: '#eaeaea'
			}
		}
	});

S export​(flex, fileName, settings?): void

flex (FlexGrid): 待导出的 FlexGrid 实例。
fileName (string): 导出的文件名。
settings? (IFlexGridExportSettings): 导出的相关设置。

导出 FlexGrid 至 PDF。

	wijmo.grid.pdf.FlexGridPdfConverter.export(grid, 'FlexGrid.pdf', {
		scaleMode: wijmo.grid.pdf.ScaleMode.PageWidth;
		maxPages: 10,
		styles: {
			cellStyle: {
				backgroundColor: '#ffffff',
				borderColor: '#c6c6c6'
			},
			headerCellStyle: {
				backgroundColor: '#eaeaea'
			}
		},
		documentOptions: {
			info: {
				title: 'Sample'
			}
		}
	});
