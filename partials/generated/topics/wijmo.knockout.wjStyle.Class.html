wijmo.knockout.js

使用 KnockoutJS 绑定设置 FlexGrid Column 单元格条件格式化。

使用 wjStyle 绑定和 wjFlexGridColumn 绑定提供单元格的条件格式化，示例如下：

<div data-bind="wjFlexGridColumn: { 
        header: 'Change',
        binding: 'changePercent',
        format: 'p2',
        width: '*'
        },
        wjStyle: { color: getAmountColor($item().change) }"></div>

wjStyle 使用 KnockoutJS 原生语法 style 设置绑定。 在视图模型中添加属性，在绑定表达式中使用监控属性：

$item
到绑定到当前行的一个引用
$row
行索引。
$col
列索引。
