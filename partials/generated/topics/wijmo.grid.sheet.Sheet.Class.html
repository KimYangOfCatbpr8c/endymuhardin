模块
wijmo.grid.sheet

表示 FlexSheet 中的一个表单。

构造函数

名称 参数 描述
constructor​(owner?, grid?, sheetName?, rows?, cols?): Sheet

owner? (FlexSheet): 该对象的所有者 @see: FlexSheet 控件。
grid? (FlexGrid): 用来存储该表单的值的 FlexGrid 控件。如果未指定，则将会创建一个新的 FlexGrid 控件。
sheetName? (string): FlexSheet 内的表单名称。
rows? (number): 该表单的行数。
cols? (number): 该表单的列数。

 初始化 FlexSheet 类的一个新实例

属性

名称 类型 描述
columnCount number 获取或设置该表单中列的个数。
grid FlexGrid 获取用来存储该表单数据的关联的 FlexGrid 控件。
itemsSource any 获取或设置表单的 FlexGrid 实例的 ICollectionView 的数组。
name string 获取或设置表单的名称。
rowCount number 获取或设置该表单上行的个数。
selectionRanges ObservableArray 获取选中范围的数组。
visible boolean 获取或设置表单的可见性。

方法

名称 参数 描述
getCellStyle​(rowIndex, columnIndex): ICellStyle

rowIndex (number): 指定单元格的行索引。
columnIndex (number): 指定单元格的列索引。

 获取指定单元格的样式。

e (EventArgs)

 触发 sheetNameChanged 事件。
onVisibleChanged​(e)

e (EventArgs)

 触发 visibleChanged 事件。

事件

名称 参数 描述
nameChanged EventArgs 在表单名称发生变化之后触发。
visibleChanged EventArgs 在表单的可见状态发生变化之后触发。
