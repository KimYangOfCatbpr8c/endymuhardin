构造函数

属性

名称 类型 描述
F bold boolean 表示是否当前的字体为粗体。
F color string

获取或设置字体的颜色。

对于导出操作，颜色可以通过任何合法的 HTML 格式指定，比如说六位字符的 dash notation 标记法或者 rgb/rgba/hsl/hsla 函数格式。在 rgba/hsla 表示的情况下，任何指定的 alpha 通道的值都将被忽略。

对于导入操作，一个颜色值将始终表示为 HTML 六位字符的 dash notation 格式，比如‘#afbfcf’。
F family string 获取或设置字体集名称。
F italic boolean 表示是否当前的字体应用了斜体样式。
F size number 获取或设置以设备无关像素值（1/96英寸）表示的字体尺寸。
F underline boolean 表示是否当前的字体具有下划线。
