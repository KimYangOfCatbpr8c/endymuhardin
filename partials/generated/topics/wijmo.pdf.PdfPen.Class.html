PdfPen 类

模块
wijmo.pdf
文件
wijmo.pdf.js

决定一个用来绘制路径和文本笔画的对象。

构造函数

名称 参数 描述
constructor​(colorOrBrushOrOptions?, width?, dashPattern?, cap?, join?, miterLimit?): PdfPen

colorOrBrushOrOptions? (any): 将要使用的颜色，画刷或者Javascript对象。
width? (number): 将要使用的宽度。
dashPattern? (PdfDashPattern): 要使用的虚线模式。
cap? (PdfLineCapStyle): 要使用的线帽样式。
join? (PdfLineJoinStyle): 要使用的线交汇风格。
miterLimit? (number): 要使用的斜线限制。

 初始化 PdfPen 类的一个新的实例，使用指定的颜色或者画刷或者Javascript对象。 第一个参数可以接受下面的值：
  • Color 对象或者可以被fromString 方法接受的任意字符串。
  • PdfBrush 对象。
  • 可以包含初始化属性的Javascript对象（所有的参数都被忽略）。

属性

名称 类型 描述
brush PdfBrush 获取或设置描绘路径的画刷。 如果定义，优先于 color 属性。
cap PdfLineCapStyle 获取或设置用做路径笔画两端的形状。 默认值为 Butt 。
color Color 获取或设置描绘路径的颜色。 默认的颜色为黑色。
dashPattern PdfDashPattern 获取用做绘制路径的虚线样式。 默认值为一条实线。
join PdfLineJoinStyle 获取或设置用做路径笔画接角的形状。 默认值为 Miter 。
miterLimit number 确定斜切长度占线宽比的最大值，当线形连接是从斜切转换成斜角。 默认值为 10。
width number 获取或设置用来绘制路径的线宽，单位是点。 默认的宽度为 1。

方法

名称 参数 描述
clone​(): PdfPen 创建该 PdfPen 的一份拷贝。

返回值 这个笔的一个拷贝。
equals​(value): boolean

value (PdfPen): 待比较的 PdfPen 对象。

 确定是否指定的 PdfPen 实例和当前的实例相等。

返回值 如果指定对象与当前对象相等，则返回真，否则返回假。
