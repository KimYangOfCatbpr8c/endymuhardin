AngularJS 指令应用于 InputTime control.

使用 wj-input-time 指令添加 InputTime 到 AngularJS 应用中。 注意指令和参数名称必须使用小写字母+短线语法替代驼峰命名。 示例：

<p>Here is an InputTime control:</p>
<wj-input-time 
  value="theDate" 
  format="h:mm tt"
  min="09:00" max="17:00"
  step="15">
</wj-input-time>

在线示例

这个例子使用 InputDate 和 an InputTime 控件呈现 Date 值 (包含日期和时间信息)，需要注意两个控件是怎样绑定到同一个控制器属性变量上的，并每个编辑数据的不同部分。这个例子同时展示了 Calendar 控件可以通过单击选择日期。

wj-input-time 指令扩展了 WjComboBox ，添加了如下特性:

ng-model
@ 使用Angular的ng-model指令绑定控件的 value 属性。使用ng-model指令绑定属性能够获得一些共通的特性，例如验证，自动添加控件状态到form实例，等等。 使用 wj-model-property 重定义被 ng-model指定绑定的控件属性。
wj-model-property
@ 使用 ng-model 指令指定控件绑定的作用域。
control
= 该指令创建的指向 InputDate 的引用。
format
@ 选中事件的格式化字符串。
mask
@ 用户输入的验证蒙版 (参见 InputMask)。
max
@ 最小有效时间（格式化为 "hh:mm"）。
min
@ 最大有效时间（格式化为 "hh:mm"）。
step
@ 下拉列表的步进值。
value
= 编辑的时间 (以 Data 对象的形式).
value-changed
& @see: valueChanged 事件。

min 和 max 属性需要指定为格式化 "yyyy-MM-dd" 的日期字符串， 技术上讲，你可以使用符合W3C [RFC 3339] 定义的任何格式化字符，只要它们是能够被HTML5支持格式化字符。
