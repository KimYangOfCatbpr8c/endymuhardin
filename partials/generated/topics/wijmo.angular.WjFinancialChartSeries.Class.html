AngularJS 指令应用于 FinancialChart FinancialSeries 对象。

wj-financial-chart-series 指令必须包含于 WjFinancialChart 指令。 支持以下属性:

axis-x
@ X轴。
axis-y
@ Y轴。
binding
@ 包含Y轴数据的属性名。该值可被任意重写以绑定图表数据。
binding-x
@ 包含X轴数据的属性名。该值可被任意重写以绑定图表数据。
chart-type
@ 图表默认类型，可以重写这个。参见 ChartType。
css-class
@ 渲染序列的 CSS 样式类。
items-source
= 一个数组或者 ICollectionView 对象，其中包含序列数据。
name
@ 序列显示的图例名称。
style
= 渲染序列的样式，使用 ng-attr-style 指定序列对象。参见 ngAttr AngularJS Creating Custom Directives 以及 FlexChart 101 Styling Series 上的实例以获取更多信息。
altStyle
= 序列交替样式。
symbol-marker
@ 标记序列的形状。可重写默认值。 参见 Marker。
symbol-size
@ 渲染数据点的像素级尺寸。可以重写这个。
symbol-style
= 渲染数据点的样式。可以重写这个。
visibility
= SeriesVisibility 值，用于指定是否显示序列。

多数场景下，the wj-financial-chart-series 仅指定 name 和 binding 属性。其他的继承自父指令 wj-financial-chart 指令。
