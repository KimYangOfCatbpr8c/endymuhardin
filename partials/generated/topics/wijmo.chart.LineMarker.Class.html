LineMarker 类

派生类
WjFlexChartLineMarker
模块
wijmo.chart
文件
wijmo.chart.js

表示 FlexChart 的 LineMarker 的一个扩展。

LineMarker 由一个显示数据点的值的一个文本区域，一个可选的水平或垂直线（或者两者同时显示，以达到十字线效果）组成，位于绘图区域上方。 它可以是静态的（interaction = None)，或者是跟随鼠标或者触摸点（interaction = move），或者当用户拖拽线形时移动（interaction = Drag）。 例如： 

  // 创建一个可交互的标记，显示水平线和 Y 值
  var lm = new wijmo.chart.LineMarker($scope.ctx.chart, {
      lines: wijmo.chart.LineMarkerLines.Horizontal,
      interaction: wijmo.chart.LineMarkerInteraction.Move,
      alignment : wijmo.chart.LineMarkerAlignment.Top
  });
  lm.content = function (ht) {
      // 显示 Y 值
      return lm.y.toFixed(2);
  }

属性

名称 类型 描述
alignment LineMarkerAlignment

获取或设置 LineMarker 内容的对齐方式。

默认情况下，LineMarker　靠右显示，位于目标点的底部。 使用 ‘|’ 联合对齐方式的设置。

// 设置对齐方式为靠左。
marker.alignment = wijmo.chart.LineMarkerAlignment.Left;
// 设置对齐方式为左上。
marker.alignment = wijmo.chart.LineMarkerAlignment.Left | wijmo.chart.LineMarkerAlignment.Top;

chart FlexChartCore 获取拥有该 LineMarker 的 FlexChart 对象。
content Function 获取或设置允许您自定义 LineMarker 文本内容的内容函数。
dragContent boolean 获取或设置一个布尔值，该值表示当交互模式设置为“Drag”时，标记的内容是否可拖拽。
dragLines boolean 获取或设置一个布尔值，该值表示当交互模式设置为“Drag”，且水平和垂直线被拖拽是，线形是连接的。
dragThreshold number 获取或设置标记可以被拖动的距离水平线和垂直线的最大距离。
horizontalPosition number

获取或设置相对于绘图区域 LineMarker 的水平位置。

其值范围为（0，1）。 如果该值为 null 或者 undefined 并且 interaction 设置为 wijmo.chart.LineMarkerInteraction.Move 或者 wijmo.chart.LineMarkerInteraction.Drag， 标记的水平位置将基于点的位置自动计算。
interaction LineMarkerInteraction 获取或设置 LineMarker 的交互模式。
isVisible boolean 获取或设置 LineMarker 的可见性。
lines LineMarkerLines 获取或设置 LineMarker 线形的可见性。
seriesIndex number 获取或设置显示该 LineMarker 的图表中系列的索引。 当 interaction 属性设置为 wijmo.chart.LineMarkerInteraction.Move 或者 wijmo.chart.LineMarkerInteraction.Drag 时，该属性生效。
verticalPosition number

获取或设置相对于绘图区域 LineMarker 的垂直位置。

其值范围为（0，1）。 如果该值为 null 或者 undefined 并且 interaction 设置为 wijmo.chart.LineMarkerInteraction.Move 或者 wijmo.chart.LineMarkerInteraction.Drag， 标记的垂直位置将基于点的位置自动计算。
x number 获取基于图表数据坐标的当前的 X 值。
y number 获取基于图表数据坐标的当前的 Y 值。
