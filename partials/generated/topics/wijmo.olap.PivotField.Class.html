属性

名称 类型 描述
aggregate Aggregate 获取或设置该字段如何进行概括。
binding string 获取或设置该字段绑定的属性的名称。
collectionView ICollectionView 获取绑定至该字段的 ICollectionView。
dataType DataType 获取或设置字段的数据类型。
descending boolean 获取或设置一个值，该值决定是否该字段上的键应当按照降序顺序进行排序。
engine PivotEngine 获取拥有该 PivotField 的 PivotEngine 对象的一个引用。
filter PivotFilter 获取到用来对该字段的值进行筛选的 PivotFilter 的对象的一个引用。
format string 获取或设置当显示字段值时，所使用的格式。
header string 获取或设置在用户界面上表示该字段的字符串。
isActive boolean

获取或设置一个值，该值决定是否该字段当前是否在视图中被使用。

设置该属性的值为 true 将使得该字段添加到视图的 rowFields 或者 valueFields，取决于字段本身的数据类型。
isContentHtml boolean 获取或设置一个值，该值表示是否该字段中的项目包含 HTML 格式的文本而不是纯文本。
parentField PivotField

获取该字段的父字段。

当您多次拖拽了相同的字段至 Values 列表，则该字段的不同拷贝将被创建，您可以用不同的参数使用这个绑定。 这些拷贝保存了一份到其父字段的引用。
showAs ShowAs 获取或设置字段结果如何被格式化。
weightField PivotField

获取或设置该字段的 PivotField，做为该字段上计算汇总的一个权重。

如果该属性设置为 null，则全部的值都假定具有相同的权重 1。

该属性允许您计算加权平均值和总量。 比如说，假定数据包含‘Quantity’字段以及'Price'字段， 您可以将‘Price’字段用做值字段，同时将‘Quantity’字段做为权重。 输出将包含数据的加权平均。
width number 获取或设置当在用户界面上显示该字段时，所使用的首选宽度。
wordWrap boolean Gets or sets a value that indicates whether the content of this field should be allowed to wrap within cells.
