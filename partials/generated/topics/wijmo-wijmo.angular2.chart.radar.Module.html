模块 wijmo/wijmo.angular2.chart.radar

文件
wijmo.angular2.js

Contains Angular 2 components for the wijmo.chart.radar module.

wijmo.angular2.chart.radar is an external TypeScript module that can be imported to your code using its ambient module name. For example:

import * as wjRadar from 'wijmo/wijmo.angular2.chart.radar';
 
@Component({
    directives: [wjRadar.WjFlexRadar, wjRadar.WjFlexRadarSeries],
    template: `
      <wj-flex-radar [itemsSource]="data" [bindingX]="'x'">
          <wj-flex-radar-series [binding]="'y'"></wj-flex-radar-series>
      </wj-flex-radar>`,
    selector: 'my-cmp',
})
export class MyCmp {
    data: any[];
}

WjFlexRadar

Angular 2 组件用于 FlexRadar 控件 。

使用 wj-flex-radar 组件向你的 Angular2 应用添加 FlexRadar 控件。关于跟多 Angular 2标记语言，请参见Angular 2 Markup。

WjFlexRadar 组件继承自 FlexRadar 控件，继承了全部的属性，事件和方法 。

The wj-flex-radar component may contain WjFlexChartAxis, WjFlexChartLegend, WjFlexRadarSeries child components.
WjFlexRadarAxis

Angular 2 component for the WjFlexRadar FlexRadarAxis object.

wj-flex-radar-axis 组件必须包含在 WjFlexRadar 组件中。

Use the wj-flex-radar-axis component to add FlexRadarAxis objects to your Angular 2 applications. 更多关于 Angular 2标记语言，请参见Angular 2 Markup。

WjFlexRadarAxis 组件继承自 FlexRadarAxis 类，并继承了全部的属性，事件和方法。
