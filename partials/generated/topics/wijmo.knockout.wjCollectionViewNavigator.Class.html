使用 KnockoutJS 绑定 ICollectionView 导航元素。

使用 wjCollectionViewNavigator 指令，可以实现 ICollectionView 内部元素导航，示例如下：

Here is a CollectionViewNavigator:</p>
<div 
  data-bind="wjCollectionViewNavigator: { cv: myCollectionView }">
</div>

wjCollectionViewNavigator 指令只有一个属性

cv
引用 ICollectionView 对象用于导航。
