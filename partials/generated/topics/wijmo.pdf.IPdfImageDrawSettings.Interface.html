IPdfImageDrawSettings 类

模块
wijmo.pdf
文件
wijmo.pdf.js

表示 drawImage 方法所使用的图像绘制设置。

如果宽度和高度选项均未指定，则图像将按照其原始尺寸绘制。 如果仅提供了宽度，那么图像将被按比例缩放以适应指定的宽度。 如果仅提供了高度，则图像将被按比例缩放以适应指定的高度。 如果同时指定了宽度和高度，则图像将按照 stretchProportionally 属性的设置，对图像进行拉伸。

属性

名称 类型 描述
F align? PdfImageHorizontalAlign 在等比例拉伸时，决定图像的水平对齐方式。
F height? number 决定图像的高度，单位是点。
F stretchProportionally? boolean 表示是否当同时指定了宽度和高度选项时，是否按比例拉伸图像。
F vAlign? PdfImageVerticalAlign 在等比例拉伸时，决定图像的垂直对齐方式。
F width? number 决定图像的宽度，单位是点。
