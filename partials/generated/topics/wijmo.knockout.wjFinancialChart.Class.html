KnockoutJS 绑定用于 FinancialChart 控件。

使用 wjFinancialChart 绑定向你的 KnockoutJS 应用添加 FinancialChart 控件，比如：

<p>这是一个 FinancialChart 控件:</p>
<div data-bind="wjFinancialChart: { itemsSource: data, chartType: 'Candlestick' }">
    <div data-bind="wjFlexChartLegend : { 
        position: 'Top' }">
    </div>
    <div data-bind="wjFinancialChartSeries: {
         name: 'close', 
        binding: 'high,low,open,close' }">
    </div>
    </div>
</div>
wjFinancialChart 绑定可能包含下面的子绑定： wjFlexChartAxis, wjFinancialChartSeries, wjFlexChartLegend. wjFinancialChart 绑定支持 FinancialChart 控件的所有的读写属性和事件。 并且支持额外的 tooltipContent 属性，为 FinancialChart.tooltip.content 属性分配一个值。 selection 属性支持双向的数据绑定。
