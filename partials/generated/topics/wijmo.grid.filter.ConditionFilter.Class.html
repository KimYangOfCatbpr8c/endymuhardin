为 FlexGrid 上的列定义一个条件筛选器。

条件筛选器包含两个条件，这两个条件可以通过 ‘and’ 或者一个 ‘or’ 操作符连接在一起。

该类被 FlexGridFilter 类所使用；您将很少有机会直接使用到。

属性

名称 类型 描述
and boolean 获取一个值，该值表示是否通过 AND 或 OR 操作符将两个条件联合在一起。
column Column 获取待筛选的 Column 对象。
condition1 FilterCondition 获取筛选器中的第一个条件。
condition2 FilterCondition 获取筛选器中的第二个条件。
isActive boolean

获取一个值，该值表示是否筛选器为活动状态。

如果两个条件中的至少一个条件具有操作符，并且数值设置为一个合理的组合时，该筛选器为活动状态。
