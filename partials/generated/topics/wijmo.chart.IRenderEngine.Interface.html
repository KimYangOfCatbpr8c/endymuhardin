IRenderEngine 类

模块
wijmo.chart
文件
wijmo.chart.js

表示一个执行基本的绘图程序的呈现引擎。

属性

名称 类型 描述
F element Element 获取呈现的元素。
F fill string 获取或设置用来填充元素的颜色。
F fontFamily string 获取或设置输出文本的字体集。
F fontSize string 获取或设置输出文本的字体尺寸。
F stroke string 获取或设置用来绘制元素边框的颜色。
F strokeWidth number 获取或设置轮廓线的线宽。
F textFill string 获取或设置文本色。

方法

名称 参数 描述
beginRender​() 清除 viewport 并开始呈现周期。
endRender​() 结束呈现周期。
setViewportSize​(w, h)

w (number): Viewport 宽度。
h (number): Viewport 高度。

 设置 viewpoint 的尺寸。
