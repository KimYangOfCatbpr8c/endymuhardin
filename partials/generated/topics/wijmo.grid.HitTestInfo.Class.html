HitTestInfo 类

模块
wijmo.grid
文件
wijmo.grid.js

包含 FlexGrid 控件上，位于指定页面坐标位置的部分的信息的类。

构造函数

名称 参数 描述
constructor​(grid, pt): HitTestInfo

grid (any): 待调查的 FlexGrid 控件或者 GridPanel 控件。
pt (any): 待调查的位于页面坐标系的 Point 对象。

 初始化类 HitTestInfo 的一个新实例

属性

名称 类型 描述
cellType CellType 获取指定位置处的单元格类型。
col number 获取指定位置所在的单元格的列索引。
edgeBottom boolean 获取一个值，该值表示是否鼠标靠近单元格的底部边缘。
edgeLeft boolean 获取一个值，该值表示是否鼠标靠近单元格的左侧边缘。
edgeRight boolean 获取一个值，该值表示是否鼠标靠近单元格的右侧边缘。
edgeTop boolean 获取一个值，该值表示是否鼠标靠近单元格的顶部边缘。
panel GridPanel 获取指定位置处的表单面板。
point Point 获取该 HitTestInfo 所指代的位于控件坐标系的点。
range CellRange 获取指定位置的单元格范围。
row number 获取指定位置所在的单元格的行索引。
