ICollectionView pager元素对应的 Angular 2 组件。

通过 wj-collection-view-pager 组件添加一个元素，允许用户遍历一个分页的 ICollectionView 中的所有页。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。例如：

<wj-collection-view-pager [cv]="myCollectionView"> </wj-collection-view-pager>
