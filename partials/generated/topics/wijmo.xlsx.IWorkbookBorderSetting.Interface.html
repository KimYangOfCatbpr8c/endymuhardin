边框颜色。

对于导出操作，颜色可以通过任何合法的 HTML 格式指定，比如说六位字符的 dash notation 标记法或者 rgb/rgba/hsl/hsla 函数格式。在 rgba/hsla 表示的情况下，任何指定的 alpha 通道的值都将被忽略。 will be ignored.

对于导入操作，一个颜色值将始终表示为 HTML 六位字符的 dash notation 格式，比如‘#afbfcf’。
