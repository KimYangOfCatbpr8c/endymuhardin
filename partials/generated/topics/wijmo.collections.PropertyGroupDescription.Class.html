PropertyGroupDescription 类

基类
GroupDescription
模块
wijmo.collections
文件
wijmo.js
表示

以属性名为标准来描述项目分组。

例如，下面的代码会引起致CollectionView 通过它们的‘country’属性的第一个字母来进行分组: 

var cv = new wijmo.collections.CollectionView(items);
var gd = new wijmo.collections.PropertyGroupDescription('country');
cv.groupDescriptions.push(gd);

您还可以指定一个回调函数，该函数生成该组名。例如，下面的代码会引起致CollectionView 通过它们的‘country’属性的第一个字母来进行分组： 

var cv = new wijmo.collections.CollectionView(items);
var gd = new wijmo.collections.PropertyGroupDescription('country', 
  function(item, propName) {
    return item[propName][0]; // return country's initial
});
cv.groupDescriptions.push(gd);

构造函数

名称 参数 描述
constructor​(property, converter?): PropertyGroupDescription

property (string): The name of the property that specifies which group an item belongs to.
converter? (Function): A callback function that takes an item and a property name and returns the group name. If not specified, the group name is the property value for the item.

 初始化类 PropertyGroupDescription 的一个新实例。

名称 参数 描述
groupNameFromItem​(item, level): any

item (any): 为该项目获取组名。
level (number): 零基组水平索引。

 返回给定项目的组名。

返回值 该项目属于该组的名称。
namesMatch​(groupName, itemName): boolean

groupName (any): 组名称。
itemName (any): 项目名称。

 返回一个值，该值表明组名称和项目名称是否匹配（意味着该项目属于该组）。

返回值 如果名称匹配则为真，否则为假。
