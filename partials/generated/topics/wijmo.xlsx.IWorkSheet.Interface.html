IWorkSheet 类

模块
wijmo.xlsx
文件
wijmo.xlsx.js

表示包含表单属性和数据的 Workbook 对象模型表单定义。

表单的单元格保存在行对象的数组中，通过类似于 sheet.rows[i].cells[j] 的路径进行访问。

属性

名称 类型 描述
F columns? IWorkbookColumn[]

获取或设置表单列定义的数组。

数组中的每一个 IWorkbookColumn 对象描述了位于 xlsx 表单相关位置处的一列， 比如说，位于索引 0 处的列 关联到 xlsx 表单索引 A 处的列，具有索引值为 1 的列定义了表单中具有索引值 B 的列，以此类推。 如果特定的列在 xlsx 文件中没有描述，则相关的数组元素在导入导出操作时均为 undefined。

如果数组中的 IWorkbookColumn 没有指定 width 属性的值，则将应用默认的列宽。
F frozenPane? IWorkbookFrozenPane 获取或设置冻结面板设置。
F name? string 获取或设置表单名称。
F rows? IWorkbookRow[]

获取或设置一个表单行定义的数组。

数组中的每一个 IWorkbookRow 对象描述了位于 xlsx 表单相关位置处的一行， 比如说，位于索引 0 处的列 关联到 xlsx 表单索引 1 处的行，具有索引值为 1 的行定义了表单中具有索引值 2 的行，以此类推。 如果特定的行在 xlsx 文件中没有描述，则相关的数组元素在导入导出操作时均为 undefined。

如果数组中的 IWorkbookRow 没有指定 height 属性的值，则将应用默认的行高。
F style? IWorkbookStyle 获取或设置表单样式。 该属性定义了表单中所有单元格的样式，能被特定单元格样式覆盖。
F summaryBelow? boolean 获取或设置一个值，该值指示是否摘要行出现在内容行的上方还是下方。
F visible? boolean 获取或设置工作表的可见性。
