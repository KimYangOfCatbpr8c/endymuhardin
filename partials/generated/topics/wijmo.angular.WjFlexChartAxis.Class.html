AngularJS 指令应用于 FlexChart Axis 对象.

wj-flex-chart-axis 指令必须包含在一个 WjFlexChart 指令或者 WjFinancialChart 指令中. 支持以下特性:

wj-property
@ 定义 FlexChart 属性名, x轴，y轴 用于初始化指令。
axis-line
@ 轴线是否可见。
format
@ 轴标签的格式化字符串 (参见 Globalize).
labels
@ 轴标签是否可见。
label-angle
@ 轴标签旋转角度。
label-align
@ 轴标签对齐方式。
label-padding
@ 轴标签内边距
major-grid
@ 轴线是否包含主网格线。
major-tick-marks
@ 定义轴上的主刻度线。(参见 TickMark).
major-unit
@ 轴主单位刻度的单位。
max
@ 轴线表示的最大值。
min
@ 轴线表示的最小值。
minor-grid
@ 轴线是否包含小网格线。
minor-tick-marks
@ D定义轴上小刻度线间距。(参见 TickMark).
minor-unit
@ 轴小单位刻度的单位。
origin
@ 轴线原点。
overlappingLabels
@ OverlappingLabels 值，定义怎样处理重叠轴标签。
position
@ Position 值，定义轴的位置。
reversed
@ 是否翻转 (顶转到底，右转到左)。
title
@ 标题。
