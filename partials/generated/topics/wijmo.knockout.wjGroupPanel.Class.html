使用 KnockoutJS 绑定 FlexGrid GroupPanel 控件。

使用 wjGroupPanel 绑定需要使用 grid 属性连接到 FlexGrid 控件，示例如下：

<p>这是一个带有 GroupPanel 的FlexGrid 控件。</p>
 
<div data-bind="wjGroupPanel: { grid: flex(), placeholder: '将列拖动到这里创建组。' }"></div>
 
<div data-bind="wjFlexGrid: { control: flex, itemsSource: data }">
    <div data-bind="wjFlexGridColumn: { 
        header: 'Country', 
        binding: 'country', 
        width: '*' }">
    </div>
    <div data-bind="wjFlexGridColumn: { 
        header: 'Date', 
        binding: 'date' }">
    </div>
    <div data-bind="wjFlexGridColumn: { 
        header: 'Revenue', 
        binding: 'amount', 
        format: 'n0' }">
    </div>
    <div data-bind="wjFlexGridColumn: { 
        header: 'Active', 
        binding: 'active' }">
    </div>
</div>

wjGroupPanel 支持 GroupPanel 类的所有可读写属性及事件。
