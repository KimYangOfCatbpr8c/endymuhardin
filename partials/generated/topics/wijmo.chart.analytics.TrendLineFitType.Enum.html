TrendLineFitType 枚举

模块
wijmo.chart.analytics
文件
wijmo.chart.analytics.js

指定趋势线系列的拟合类型。

值

名称 描述
AverageX X-值的平均值。
AverageY Y-值的平均值。
Exponential 回归拟合到方程 Y(x) = a * exp(b*x)。
Fourier 回归拟合到方程 Y(x) = a + b * cos(x) + c * sin(x) + d * cos(2*x) + e * sin(2*x) + ...。
Linear 最接近所有数据点的一条直线。 Y(x) = a * x + b。
Logarithmic 回归拟合到方程 Y(x) = a * ln(x) + b。
MaxX 最大的 X-值。
MaxY 最大的 Y-值。
MinX 最小的 X-值。
MinY 最小的 Y-值。
Polynomial 回归拟合到方程 Y(x) = a * x^n + b * x^n-1 + c * x^n-2 + ... + z。
Power 回归拟合到方程 Y(x) = a * pow(x, b)。
