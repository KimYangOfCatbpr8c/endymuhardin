AngularJS 指令用于FlexChart 和 FinancialChart Waterfall。 wj-flex-chart-waterfall 指令必须包含在 WjFlexChart or WjFinancialChart 指令中。 它支持下面的属性：

binding
@ 包含系列Y值的属性名。这个值覆盖在图表级别的设置。
binding-x
@ 包含系列X值的属性名。这个值覆盖在图表级别的设置。
items-source
= 包含这个系列数据的数组或者 ICollectionView 对象。
name
@ 展示在图例上的这个系列的名称。
visibility
= SeriesVisibility 决定是否和在哪展示系列。
relative-data
@ 这个值决定是否给定的数据是相对的。
start
@ 开始栏的值。
start-label
@ 开始栏的标签。
show-total
@ 这个值决定是否展示合计栏。
total-label
@ 合计栏的标签。
show-intermediate-total
@ 这个值决定是否展示 决定是否显示中间合计栏。
intermediate-total-positions
@ 包含用于中间合计栏的位置的索引的值。
intermediate-total-labels
@ 这个值包含中间合计栏的标签。
connector-lines
@ 决定是否展示连接线。 The value that determines whether to show connector lines.
styles
@ 瀑布图样式的值。
