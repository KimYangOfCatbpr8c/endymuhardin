使用 KnockoutJS 绑定 FlexGrid 控件。

使用 wjFlexGrid 绑定添加 FlexGrid 控件，示例如下：

<p>Here is a FlexGrid control:</p>
<div data-bind="wjFlexGrid: { itemsSource: data }">
    <div data-bind="wjFlexGridColumn: { 
        header: 'Country', 
        binding: 'country', 
        width: '*' }">
    </div>
    <div data-bind="wjFlexGridColumn: { 
        header: 'Date', 
        binding: 'date' }">
    </div>
    <div data-bind="wjFlexGridColumn: { 
        header: 'Revenue', 
        binding: 'amount', 
        format: 'n0' }">
    </div>
    <div data-bind="wjFlexGridColumn: { 
        header: 'Active', 
        binding: 'active' }">
    </div>
</div>

wjFlexGrid 绑定可以包含 wjFlexGridColumn 子绑定。

lt;b>wjFlexGrid 支持 FlexGrid 控件的所有可读写属性及事件，除了 scrollPosition 和 selection 属性
