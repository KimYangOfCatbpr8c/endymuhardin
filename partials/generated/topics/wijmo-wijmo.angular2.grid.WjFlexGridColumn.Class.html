FlexGrid Column控件对应的 Angular 2 组件。

wj-flex-grid-column 组件必须包含在一个 WjFlexGrid 组件内部。

使用 wj-flex-grid-column 组件添加 Column 对象至您的 Angular 2 应用程序中。关于 Angular 2 标记语法的更多细节，请参见 Angular 2 标记。

WjFlexGridColumn 组件继承自 Column 类，因此继承了其全部的属性，事件和方法。

wj-flex-grid-column 组件可以包含 WjFlexGridCellTemplate 子 Directive 指令。

width any

获取或设置列的宽度。

列宽可以为正数值（以像素数设置列宽）， null 或者 负数值（使用集合上的默认列宽），或者 具有 '{number}*' 格式的字符串（星号尺寸标记）。

星号尺寸标记执行一种 XAML-样式的动态尺寸，列宽和星号之前的数字成比例。 例如，如果一个表单具有三个列，宽度分别设置为 “100”，“*”，以及“3*”， 则第一列宽度将会是 100 像素，第二列将占用余下空间的四分之一，最后一列 将占用余下的四分之三的空间。

星号尺寸标记允许您定义一列自动地拉伸填充可用的宽度。 例如，设置最后一列的宽度为“*”，它将自动的拉伸以填充表单余下的空间，因此，表单上将不存在剩余的空白区。 您也许可以通过设置列上的 minWidth 属性，以防止列宽变得过于狭窄。
