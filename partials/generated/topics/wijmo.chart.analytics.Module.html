模块 wijmo.chart.analytics

文件
wijmo.chart.analytics.js

定义了向图表添加分析功能的一些类，包括 TrendLine， MovingAverage 以及 FunctionSeries。

类

名称 描述
FunctionSeries 表示 FlexChart 的函数系列的一个基类。
MovingAverage 表示 FlexChart 以及 FinancialChart 的一个移动平均趋势线。 它是一种计算分析，通过创建整个数据集的不同子集的平均值系列，以分析数据。 您可以在 MovingAverage 自身上通过设置 MovingAverage 对象上的 type 属性来定义不同的类型。 MovingAverage class 具有一个 period 属性，该属性允许您设置计算平均值的区间值。
ParametricFunctionSeries

表示 FlexChart 的一个参数化函数系列。

ParametricFunctionSeries 允许绘制一个由公式组 x=f(t) 和 y=f(t) 定义的函数图形。 X 和 Y 的值由给定的 xFunc 以及 yFunc 计算。
TrendLine

表示 FlexChart 以及 FinancialChart 上的一条趋势线。

趋势线是一条叠加在图表上的一条线，用来显示数据的整体变化趋势。 您可以通过 fitType 属性为每一个可以添加到 FlexChart 系列集合中的 TrendLine 对象定义一个不同的拟合类型。
TrendLineBase 表示各种趋势线的基类。
Waterfall 代表FlexChart 的一个瀑布系列。 Waterfall 系列主要用于展示开始的位置的增长或下降情况。
YFunctionSeries

表示 FlexChar 的 Y 函数系列。

YFunctionSeries 允许绘制一个由公式 y=f(x) 定义的函数图形。

枚举

名称 描述
MovingAverageType 指定 MovingAverage 系列的类型。
TrendLineFitType 指定趋势线系列的拟合类型。
