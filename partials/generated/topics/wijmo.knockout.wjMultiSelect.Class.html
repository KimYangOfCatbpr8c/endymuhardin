KnockoutJS 绑定用于 MultiSelect 控件。 使用wjMultiSelect 绑定向 KnockoutJS 应用程序添加 MultiSelect 控件。比如：

<p>Here is a MultiSelect control:</p>
<div data-bind="MultiSelect: {
  itemsSource: countries,
  isEditable: false,
  headerFormat: '{count} countries selected' }">
</div>

wjMultiSelect 绑定支持 MultiSelect 所有的读写属性和事件。下面的属性提供双向的数据绑定：

  • isDroppedDown
  • text
  • selectedIndex
  • selectedItem
  • selectedValue
