使用 KnockoutJS 绑定 InputTime 控件。

使用 wjInputTime 绑定添加 InputTime 控件，示例如下：

<p>Here is an InputTime control:</p>
<div data-bind="wjInputTime: {
  min: new Date(2014, 8, 1, 9, 0),
  max: new Date(2014, 8, 1, 17, 0),
  step: 15,
  format: 'h:mm tt',
  value: theDate }">
</div>

wjInputTime 支持 InputTime 控件的所有可读写属性及事件。以下属性支持双向绑定：

  • isDroppedDown
  • text
  • selectedIndex
  • selectedItem
  • selectedValue
  • value
