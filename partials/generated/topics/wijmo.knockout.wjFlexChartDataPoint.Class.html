表示

KnockoutJS 绑定用于 DataPoint 对象。 wjFlexChartDataPoint 必须包含在wjFlexChartAnnotation 对象中。 父对象的 where wjFlexChartDataPoint 属性应当通过wjProperty 属性指定。

使用 wjFlexChartDataPoint 绑定向你的 KnockoutJS 应用程序添加 DataPoint 对象。比如： 

<p>Here is a DataPoint:</p>
  <div data-bind="wjFlexChartDataPoint: { wjProperty: 'point', x: 0.9, y:0.4}" ></div>

wjFlexChartDataPoint 绑定支持所有 DataPoint 的读写属性和事件
