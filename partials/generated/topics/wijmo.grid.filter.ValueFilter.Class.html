定义一个 FlexGrid 控件上某一列的值筛选器。

值筛选器包含一个明确的值列表，由表单显示。

属性

名称 类型 描述
column Column 获取待筛选的 Column。
filterText string 获取或设置一个用做筛选显示的值的字符串。
isActive boolean

获取一个值，该值表示是否该筛选器为活动状态。

如果有任何一个值是选中状态，则该筛选器为活动状态。
maxValues number

获取或设置列表中所显示的值的最大数量。

添加过多数量的项目至该列表将导致搜索的困难程度增加，同时将会大大拖累性能。 该属性限制了任何时刻所显示的项目的数量，但是用户仍然可以通过搜索框筛选他们 所感兴趣的内容。

默认情况下，该属性的值设置为 250。

以下代码将该值设置为 1,000,000，有效地列举该字段全部的不重复的值：

// 修改 ‘id’ 列的 maxItems 属性：
var f = new wijmo.grid.filter.FlexGridFilter(s);
f.getColumnFilter('id').valueFilter.maxValues = 1000000;
showValues any 获取或设置一个对象，该对象包含全部的应当显示在值列表中的格式化的值。
sortValues boolean 获取或设置一个值，指示该列在展示时是否应当排序。 这个属性很有用，当你使用 uniqueValues去提供一个值列表并且希望保留值的顺序。
uniqueValues any[]

获取或设置一个数组，该数组包含待显示在列表中的全部的不重复的值。

如果该属性设置为 null，则该列表将基于表单数据填充。

显式地指定唯一值列表相对于从数据构建该列表来说，前者更加有效。并且如果数据在服务器端 进行筛选时，这是必须的（因为在这种情况下，有些值在客户端不出现，因此会造成不完整的列表）。

例如，以下代码提供了一个国家列表，用做绑定了 'country' 字段的 ValueFilter：

// 为 FlexGrid 创建一个筛选器
var filter = new wijmo.grid.filter.FlexGridFilter(grid);
// 指定一个不重复值得列表至 country 筛选器
var cf = filter.getColumnFilter('country');
cf.valueFilter.uniqueValues = countries;
