模块 wijmo.grid

wijmo.grid.js

定义 FlexGrid 控件及其相关类。

以下示例创建了一个 FlexGrid 控件，并将其绑定至一个‘Data’数组。 该表单具有三列，显示地通过生成 columns 数组指定。

DataMap

表示列的 dataMap 属性的数据映射。

数据列向表单提供了自动查找的能力。比如说，您希望显示一个客户的名称，而不是 ID，或者 一个颜色值，而不是其原始的 RGB 值。

以下代码将一个表单绑定到一组产品的集合，之后指定一个 DataMap 至表单的 ‘CategoryID’ 列，因此表单显示分类的名称而不是其原始的 ID。

// 绑定表单至产品
var flex = new wijmo.grid.FlexGrid();
flex.itemsSource = products;
// 映射 CategoryID 列以显示分类的名称而不是原始的 ID
var col = flex.columns.getColumn('CategoryID');
col.dataMap = new wijmo.grid.DataMap(categories, 'CategoryID', 'CategoryName');

FlexGrid

FlexGrid 控件提供了一种强大而灵活的在一个表格格式下显示和编辑数据的一种方式。

FlexGrid 控件是一个全功能的表格，提供了您所需用到的全部功能，包括集中不同的选择模式，排序， 改变列的顺序，分组，筛选，编辑，自定义单元格，XAML 样式的星号列宽计算，行和列的虚绑定，等等。
FormatItemEventArgs 为 formatItem event 事件提供事件参数。
GridPanel 表示表单的一个逻辑部分，比如说列头，行头，以及可滚动数据区。
GroupRow 表示作为一组行的页眉的行。
HitTestInfo 包含 FlexGrid 控件上，位于指定页面坐标位置的部分的信息的类。
MergeManager

定义 FlexGrid 的单元格合并行为。

该类的一个实例将自动地被创建并添加到表单的 mergeManager 属性，以实现表单的默认合并行为。

如果您希望自定义默认的合并行为，可以创建一个类，继承自 MergeManager 类，并覆盖 getMergedRange 方法。
Row 表示表单中的一行。
RowCol Row 以及 Column 类的一个抽象基类。
RowColCollection 做为行和列集合的基类的抽象类。
RowCollection 表示一个 FlexGrid 控件中的 Row 对象的集合。
